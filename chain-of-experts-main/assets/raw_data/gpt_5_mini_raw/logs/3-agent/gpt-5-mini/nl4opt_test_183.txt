[MODELER]:
Model

- Decision variables
  - L ∈ Z≥0 : number of large-ship trips.
  - S ∈ Z≥0 : number of small-ship trips.

- Objective
  - Minimize total pollution: minimize P = 20L + 15S.

- Constraints
  1. Capacity: 2000L + 800S ≥ 20000.
  2. Small-trip share: S ≥ 0.4(L + S). (Equivalently 0.6S ≥ 0.4L ⇒ S ≥ (2/3)L.)
  3. Upper bound on large trips: L ≤ 7.
  4. Integrality: L, S integers (trip counts must be whole numbers).
  5. Nonnegativity: L ≥ 0, S ≥ 0.

Remarks
- Pollution per passenger: large = 20/2000 = 0.01 per passenger (more efficient), small = 15/800 = 0.01875 per passenger (less efficient). So use as many large trips as allowed, subject to the small-trip share and capacity constraints.

Solution (integer)
- Evaluate feasible integer L = 0..7 and compute minimum S satisfying S ≥ ceil(max((20000 − 2000L)/800, 2L/3)).
- This yields the optimal integer solution: L = 7 large trips and S = 8 small trips.
- Total transported customers = 7·2000 + 8·800 = 14000 + 6400 = 20400 (≥20000).
- Total pollution = 20·7 + 15·8 = 260 units.

(Continuous relaxation, allowing fractional trips, gives L=7, S=7.5 with pollution 252.5, but fractional trips are not realistic here.)

Suggested solvers / Python libraries
- pulp (for small MILPs with CBC solver)
- OR-Tools CP-SAT (good for integer/CP)
- pyomo with CBC/Gurobi/CPLEX (for more advanced use)
- mip (Python-MIP) also works

Minimal example using pulp
- Formulate L and S as LpInteger variables, add constraints above, set objective minimize 20L+15S, and solve with default CBC solver.

[CODER]:
import math

# Data
capacity_per_large = 2000
capacity_per_small = 800
pollution_large = 20
pollution_small = 15
required_capacity = 20000
L_upper = 7  # L <= 7

best_obj = None
best_solution = None

for L in range(L_upper + 1):
    # capacity constraint: 2000L + 800S >= 20000 => S >= (20000 - 2000L)/800
    needed_capacity = max(0, required_capacity - capacity_per_large * L)
    S_cap = (needed_capacity + capacity_per_small - 1) // capacity_per_small  # ceil division

    # small-trip share: S >= (2/3) L => ceil(2L/3)
    S_share = (2 * L + 3 - 1) // 3  # ceil(2L/3)

    S = max(S_cap, S_share, 0)
    obj = pollution_large * L + pollution_small * S

    if best_obj is None or obj < best_obj:
        best_obj = obj
        best_solution = (L, S, obj)

# best_solution should be (7,8,260) per analysis
value = best_obj
print(f"FINAL_OBJ: {value}")
