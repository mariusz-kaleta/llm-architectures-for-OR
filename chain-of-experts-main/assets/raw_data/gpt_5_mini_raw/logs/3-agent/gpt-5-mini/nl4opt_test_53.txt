[MODELER]:
Model summary

Decision variables
- x_C >= 0 : kilograms of fertilizer C to use
- x_Y >= 0 : kilograms of fertilizer Y to use

Objective
- Minimise total cost: minimize Z = 2 x_C + 3 x_Y

Constraints
- Nitrous oxide requirement: 1.5 x_C + 5 x_Y >= 5
- Vitamin mix requirement: 3 x_C + 1 x_Y >= 8
- Nonnegativity: x_C >= 0, x_Y >= 0

Solution (analytic)
Solve the two binding constraints as equalities to find the LP corner:
1.5 x_C + 5 x_Y = 5
3 x_C + 1 x_Y = 8

Multiply the first by 2: 3 x_C + 10 x_Y = 10
Subtract the second: 9 x_Y = 2 => x_Y = 2/9
Then 3 x_C + (2/9) = 8 => 3 x_C = 70/9 => x_C = 70/27

Objective value:
Z = 2*(70/27) + 3*(2/9) = 158/27 ≈ 5.85185

Check other corner options (x_C = 0 or x_Y = 0) give higher cost (24 and 20/3 ≈ 6.6667 respectively), so the intersection above is optimal.

Answer
- Optimal mix: x_C = 70/27 kg (≈ 2.5926 kg), x_Y = 2/9 kg (≈ 0.2222 kg)
- Minimum cost: 158/27 dollars ≈ $5.85

Python libraries you can use
- pulp (easy modelling of LPs with >= constraints)
- scipy.optimize.linprog (requires converting >= to <= form)
- OR-Tools linear solver (Google OR-Tools) for larger/more complex variants

Example (pulp) code
from pulp import LpProblem, LpMinimize, LpVariable, lpSum, PULP_CBC_CMD

prob = LpProblem("Fertilizer_mix", LpMinimize)
xC = LpVariable('xC', lowBound=0)
xY = LpVariable('xY', lowBound=0)

prob += 2*xC + 3*xY  # objective
prob += 1.5*xC + 5*xY >= 5
prob += 3*xC + 1*xY >= 8

prob.solve(PULP_CBC_CMD(msg=False))
print(xC.value(), xY.value(), "Cost =", (2*xC.value() + 3*xY.value()))
