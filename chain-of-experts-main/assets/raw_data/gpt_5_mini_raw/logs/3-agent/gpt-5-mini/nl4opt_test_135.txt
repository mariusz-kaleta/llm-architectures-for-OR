[MODELER]:
Formal model

- Decision variables
  - S ∈ Z_{\ge 0}: number of seasonal snow removers.
  - P ∈ Z_{\ge 0}: number of permanent snow removers.

- Objective
  - Minimise total number of workers: minimize N = S + P.

- Constraints
  1. Labor-hour requirement: 6S + 10P ≥ 300.
  2. Budget limit: 120S + 250P ≤ 6500.
  3. Integrality and nonnegativity: S, P ∈ Z_{\ge 0}.

So the integer linear program is:
  minimize S + P
  subject to
    6S + 10P ≥ 300,
    120S + 250P ≤ 6500,
    S, P ∈ Z_{\ge 0}.

Solution (short analysis)
- P is limited by budget to at most floor(6500/250)=26, so search P ∈ {0,...,26}.
- For each P take S = ceil(max(0,(300−10P)/6)) and check the budget constraint; pick the pair with smallest S+P.
- That yields the optimal integer solution S = 35, P = 9 (total workers = 44).
  - Hours: 6·35 + 10·9 = 210 + 90 = 300 (meets requirement exactly).
  - Cost: 120·35 + 250·9 = 4200 + 2250 = 6450 ≤ 6500.

Suggested solvers/libraries
- pulp (Python) — user-friendly MILP interface using CBC/GLPK.
- OR-Tools (CP-SAT) — very fast integer/CP solver from Google.
- mip (Python-mip) — convenient interface to CBC/Gurobi/CPLEX.
(Do not use scipy.optimize for this because it is for continuous problems; use an MILP solver.)

Example (outline) using pulp
- Define integer variables S and P.
- Add constraints 6*S + 10*P >= 300 and 120*S + 250*P <= 6500.
- Set objective S + P to minimize and solve.

[CODER]:
import math

# Data
labor_hours_required = 300
hours_per_S = 6
hours_per_P = 10

cost_per_S = 120
cost_per_P = 250
budget = 6500

# Search bounds
max_P = budget // cost_per_P  # 0..26

best_obj = None
best_S = None
best_P = None

for P in range(max_P + 1):
    # compute minimum S to meet labor requirement
    remaining_hours = labor_hours_required - hours_per_P * P
    if remaining_hours <= 0:
        S_req = 0
    else:
        S_req = math.ceil(remaining_hours / hours_per_S)
    # Check budget
    total_cost = cost_per_S * S_req + cost_per_P * P
    if total_cost <= budget and S_req >= 0:
        N = S_req + P
        if best_obj is None or N < best_obj:
            best_obj = N
            best_S = S_req
            best_P = P

# At this point best_obj should be found (problem is feasible)
value = int(best_obj) if best_obj is not None else None

print(f"FINAL_OBJ: {value}")
