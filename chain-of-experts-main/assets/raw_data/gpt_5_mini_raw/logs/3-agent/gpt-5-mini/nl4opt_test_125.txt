[MODELER]:
Model

Decision variables
- B ≥ 0 — number of bananas (fruits).  
- M ≥ 0 — number of mangoes (fruits).  
(If the number of fruits must be whole, add integer constraints B, M ∈ Z≥0.)

Objective
- Minimize total sugar S = 10B + 8M  (grams).

Constraints
1. Calories: 80B + 100M ≥ 4000.  
   (equivalently 4B + 5M ≥ 200)
2. Potassium: 20B + 15M ≥ 150.  
   (equivalently 4B + 3M ≥ 30)
3. Mango proportion: at most 33% of fruits are mangoes:
   M ≤ 0.33 (B + M).  
   This can be written as (1 − 0.33) M ≤ 0.33 B, i.e. 0.67 M ≤ 0.33 B, or
   M ≤ (33/67) B ≈ 0.492537 B.
4. Nonnegativity: B ≥ 0, M ≥ 0 (and integer if required).

This is a linear program (LP); if B and M must be integers it is a mixed integer linear program (MILP).

Solving (continuous LP)
- Because mangoes have lower sugar per calorie than bananas (8/100 < 10/80), the LP pushes M to the maximum allowed by the 33% rule. At optimum the mango proportion and the calorie constraint bind.
- Solving M = (33/67) B and 4B + 5M = 200 gives the continuous optimum:
  B* = 13400/433 ≈ 30.962
  M* = 6600/433 ≈ 15.248
  Total sugar S* = 186800/433 ≈ 431.61 grams.

Integer (whole fruits) solution
- Requiring integer fruits, find the smallest sugar among integer pairs (B,M) satisfying the constraints. A simple search or MILP solver gives:
  B = 32 bananas, M = 15 mangoes (feasible because M/(B+M) = 15/47 ≈ 0.319 ≤ 0.33).
  Total sugar = 10·32 + 8·15 = 440 grams.
- (The continuous optimum uses fractional fruits; the nearest feasible integer solution with minimum sugar is B=32, M=15.)

Suggested Python libraries
- For continuous LP: scipy.optimize.linprog, cvxpy.  
- For MILP (integer fruits): pulp (CBC solver), OR-Tools (CP-SAT or linear solver), pyomo (with a solver like CBC/Gurobi).  
- If you have commercial solvers available: Gurobi or CPLEX will solve it very quickly.

Short example (pulp, integer version)
- Build variables B, M as LpInteger ≥ 0, add the constraints as above, set objective 10·B + 8·M, and solve. The solver will return the integer-optimal B = 32, M = 15.

Remarks
- If fractional fruits are allowed, use the LP solution (B ≈ 30.96, M ≈ 15.25). If fruits must be whole, use the MILP solution (B = 32, M = 15).  
- The potassium constraint is not active at the optimum here (the calorie and mango-proportion constraints are the binding ones).
