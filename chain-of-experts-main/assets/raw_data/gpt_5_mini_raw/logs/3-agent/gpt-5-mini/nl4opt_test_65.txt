[MODELER]:
Mathematical model

Decision variables
- x = square feet of engineered laminate planks produced per week
- y = square feet of carpet produced per week

Objective
- Maximize profit: maximize Z = 2.1 x + 3.3 y

Constraints
- Demand minimums: x ≥ 15,000 ; y ≥ 5,000
- Shipping minimum (total product): x + y ≥ 50,000
- Raw-material (upper) limits: x ≤ 40,000 ; y ≤ 20,000
- Nonnegativity: x ≥ 0, y ≥ 0  (redundant given above bounds)

Solution reasoning (short)
- Carpet yields 3.3 per ft² while laminate yields 2.1 per ft², so carpet is more profitable. Produce carpet up to its cap y = 20,000.
- With y = 20,000, shipping constraint requires x ≥ 50,000 − 20,000 = 30,000. This meets the x ≥ 15,000 lower bound and is ≤ x ≤ 40,000 cap, so choose x = 30,000.
- This is optimal because any reduction in y and corresponding increase in x changes profit by (2.1 − 3.3) = −1.2 per ft² (net loss).

Optimal solution
- x* = 30,000 ft² (laminate)
- y* = 20,000 ft² (carpet)
- Maximum profit Z* = 2.1·30,000 + 3.3·20,000 = 63,000 + 66,000 = $129,000 per week

Suggested solvers / libraries
- PuLP (easy Python LP modeling) — recommended for this problem
- OR-Tools (Google) — LP/MIP solver
- scipy.optimize.linprog — for small linear programs (note: is a minimizer)
- CVXOPT / CVXPY — for more advanced convex optimization interfaces

Example PuLP code (Python)
```python
import pulp

# Problem
prob = pulp.LpProblem("FlooringProfit", pulp.LpMaximize)

# Variables
x = pulp.LpVariable('laminate', lowBound=15000, upBound=40000)  # x
y = pulp.LpVariable('carpet',   lowBound=5000,  upBound=20000)  # y

# Objective
prob += 2.1 * x + 3.3 * y

# Constraints
prob += x + y >= 50000

# Solve
prob.solve()
print("Status:", pulp.LpStatus[prob.status])
print("laminate x =", x.value())
print("carpet   y =", y.value())
print("Max profit =", pulp.value(prob.objective))
```

This will return x = 30000, y = 20000, Max profit = 129000.

# Self-contained script to solve the flooring profit LP and print the final objective value.
# No external solvers required; we enumerate LP corner candidates analytically.

# Coefficients
c_x = 2.1  # profit per ft^2 for engineered laminate (x)
c_y = 3.3  # profit per ft^2 for carpet (y)

# Bounds and constraints
x_min, x_max = 15000.0, 40000.0
y_min, y_max = 5000.0, 20000.0
shipping_min = 50000.0  # x + y >= shipping_min

# Generate candidate corner points (bounds and intersections with shipping constraint)
candidates = set()

# Add corners from bounds combinations
for x_bound in (x_min, x_max):
    for y_bound in (y_min, y_max):
        candidates.add((x_bound, y_bound))

# Add intersections of x bounds with shipping line x + y = shipping_min
for x_bound in (x_min, x_max):
    y_val = shipping_min - x_bound
    if y_min <= y_val <= y_max:
        candidates.add((x_bound, y_val))

# Add intersections of y bounds with shipping line x + y = shipping_min
for y_bound in (y_min, y_max):
    x_val = shipping_min - y_bound
    if x_min <= x_val <= x_max:
        candidates.add((x_val, y_bound))

# Also consider the case where both variables are at their upper bounds (if feasible)
candidates.add((x_max, y_max))
candidates.add((x_min, y_min))

# Evaluate feasible candidates and pick the best
best_obj = None
best_xy = None

for x, y in candidates:
    # Check feasibility
    if not (x_min <= x <= x_max and y_min <= y <= y_max):
        continue
    if x + y < shipping_min - 1e-9:
        continue
    obj = c_x * x + c_y * y
    if (best_obj is None) or (obj > best_obj + 1e-9):
        best_obj = obj
        best_xy = (x, y)

# As an extra check, if no candidate found (shouldn't happen), fall back to logical solution:
if best_obj is None:
    # Carpet is more profitable, set y to its upper bound and x to minimal required by shipping
    y_fb = y_max
    x_fb = max(x_min, shipping_min - y_fb)
    x_fb = min(x_fb, x_max)
    best_xy = (x_fb, y_fb)
    best_obj = c_x * x_fb + c_y * y_fb

# Prepare final value variable as requested
# If objective is integer-valued (within tolerance), present as int for cleaner output
if abs(best_obj - round(best_obj)) < 1e-9:
    value = int(round(best_obj))
else:
    value = float(best_obj)

# Final output line (must match required format)
print(f"FINAL_OBJ: {value}")
