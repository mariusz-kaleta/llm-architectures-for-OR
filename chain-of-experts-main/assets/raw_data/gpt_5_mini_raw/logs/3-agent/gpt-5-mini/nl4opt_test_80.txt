Formal model

Decision variables
- x = number of color printers produced per day (≥ 0)
- y = number of black-and-white printers produced per day (≥ 0)

(If you need integer solutions, add x, y ∈ Z. For this problem the LP relaxation yields integer values.)

Objective
- Maximize profit: maximize Z = 200x + 70y

Constraints
1. Color-team capacity: x ≤ 20
2. B&W-team capacity: y ≤ 30
3. Shared paper-tray machine capacity: x + y ≤ 35
4. Nonnegativity: x ≥ 0, y ≥ 0

Mathematical formulation
Maximize Z = 200x + 70y
subject to
x ≤ 20
y ≤ 30
x + y ≤ 35
x, y ≥ 0

Solution reasoning (short)
Profit per machine use is higher for color printers (200) than for black-and-white (70). Use the shared machine capacity first for color printers up to the color-team limit, then use remaining machine capacity for black-and-white. So set x = 20 (color capacity), remaining machine capacity = 35 − 20 = 15 → y = 15. This satisfies y ≤ 30. Objective value = 200·20 + 70·15 = 5050.

Optimal production plan
- Produce 20 color printers per day
- Produce 15 black-and-white printers per day
- Maximum daily profit = $5050
