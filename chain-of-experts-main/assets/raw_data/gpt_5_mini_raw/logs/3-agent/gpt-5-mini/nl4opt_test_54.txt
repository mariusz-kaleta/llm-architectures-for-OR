[MODELER]:
Model

- Decision variables
  - x = number of glasses of milk consumed (>= 0)
  - y = number of plates of vegetables consumed (>= 0)

  (If you require whole items, add integrality constraints x, y integer.)

- Objective
  - Minimize total cost: minimize C = 1*x + 2*y

- Nutrient constraints
  - Calcium: 40 x + 15 y >= 100
  - Iron:    25 x + 30 y >= 50

- Nonnegativity
  - x >= 0, y >= 0

Solution reasoning

Milk is strictly cheaper per unit of both nutrients than vegetables:
- milk: $1 gives 40 Ca and 25 Fe
- vegetables: $2 give 15 Ca and 30 Fe
Thus milk dominates vegetables (better nutrients per dollar), so the cost-minimizing solution uses only milk.

Continuous (fractional) solution
- Set y = 0. Calcium requires 40 x >= 100 -> x >= 2.5.
- Iron with x = 2.5 gives 25*2.5 = 62.5 >= 50, so iron requirement is met.
- Optimal continuous solution: x = 2.5 glasses of milk, y = 0 plates vegetables, cost = $2.50.

Integer (whole items) solution
- If x, y must be integers, the smallest integer x that meets both requirements with y = 0 is x = 3.
- Integer optimal: x = 3 glasses of milk, y = 0 plates vegetables, cost = $3.00.

Recommended solvers / Python libraries
- Continuous LP: scipy.optimize.linprog, PuLP (with CBC), or OR-Tools linear solver.
- Integer (MILP/IP): PuLP (CBC, GLPK), OR-Tools CP-SAT, or commercial solvers (Gurobi, CPLEX) if available.

import math
import numpy as np
from scipy.optimize import linprog

# Data initialization
# Decision variables: x = glasses of milk, y = plates of vegetables
c = np.array([1.0, 2.0])  # objective coefficients (minimize)
# Constraints (converted to <= form for linprog):
# 40 x + 15 y >= 100  ->  -40 x -15 y <= -100
# 25 x + 30 y >= 50   ->  -25 x -30 y <= -50
A_ub = np.array([[-40.0, -15.0],
                 [-25.0, -30.0]])
b_ub = np.array([-100.0, -50.0])
bounds = [(0, None), (0, None)]  # x >=0, y >=0

# Solve continuous LP
res = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')

if res.success:
    final_obj = float(res.fun)
else:
    # Fallback analytical solution: milk dominates, so y = 0, x = max(100/40, 50/25)
    x_fallback = max(100.0 / 40.0, 50.0 / 25.0)
    final_obj = 1.0 * x_fallback + 2.0 * 0.0

# (Optional) compute integer solution for verification (not printed)
int_best_cost = math.inf
int_best_xy = None
# search reasonable bounds
for xi in range(0, 101):
    for yi in range(0, 101):
        if 40*xi + 15*yi >= 100 and 25*xi + 30*yi >= 50:
            cost = xi*1 + yi*2
            if cost < int_best_cost:
                int_best_cost = cost
                int_best_xy = (xi, yi)

# Final required output line
print(f"FINAL_OBJ: {final_obj}")
