Model

- Variables
  - T = amount invested in telecom (dollars), T ≥ 0
  - H = amount invested in healthcare (dollars), H ≥ 0

- Constraints
  - Budget: T + H ≤ 100000
  - Ratio: T ≥ 3 H
  - Telecom cap: T ≤ 70000

- Objective
  - Maximize profit P = 0.03 T + 0.01 H

Solve (analysis)
- Telecom yields higher return (3% vs 1%), so we want T as large as possible subject to constraints.
- Set T to its upper bound T = 70000. Then ratio T ≥ 3H gives H ≤ 70000/3 ≈ 23333.33. The budget allows H up to 30000, but the ratio is tighter, so choose H = 70000/3.
- Therefore optimal solution: T = 70000, H = 70000/3 ≈ 23333.33.

Profit
- P = 0.03·70000 + 0.01·(70000/3) = 2100 + 700/3 ≈ 2333.33 dollars.
