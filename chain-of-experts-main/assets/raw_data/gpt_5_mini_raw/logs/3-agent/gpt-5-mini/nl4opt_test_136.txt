Decision variables
- S ∈ Z≥0: number of seasonal volunteers.
- F ∈ Z≥0: number of full‑time volunteers.

Objective
Maximise total gifts delivered:
max Z = 5S + 8F

Constraints
1. Points budget: 2S + 5F ≤ 200
2. At most 30% of volunteers can be seasonal:
   S ≤ 0.3 (S + F)
   Equivalent algebraically: 0.7 S ≤ 0.3 F  ⇔  S ≤ (3/7) F
3. At least 10 full‑time volunteers: F ≥ 10
4. Integrality and nonnegativity: S, F ∈ Z≥0

Remarks: Constraint (2) was rewritten to eliminate S on the right-hand side; the equivalent linear form S ≤ (3/7) F is used above.

Solution (analysis)
- Seasonal volunteers give 5 gifts for 2 points → 2.5 gifts/point.
- Full‑time volunteers give 8 gifts for 5 points → 1.6 gifts/point.
Because seasonal volunteers are more gift‑efficient per point, we want as many seasonal volunteers as the 30% rule allows, i.e. S as large as S ≤ (3/7)F, subject also to the points limit.

Substitute the ratio limit and points constraint and search integer F ≥ 10:
- Using S = floor((3/7)F) and the points constraint 2S + 5F ≤ 200 gives the best feasible integer solution:
  F = 34, S = 14.
- This yields total gifts Z = 5·14 + 8·34 = 70 + 272 = 342, and uses 2·14 + 5·34 = 198 points (≤ 200).

Therefore: hire 14 seasonal volunteers and 34 full‑time volunteers to maximize gifts (342 gifts).

import math

# Brute-force search over integer decision variables
best_obj = None
best_solution = None

# Bounds from constraints:
# F >= 10, 5F <= 200 -> F <= 40
for F in range(10, 41):
    # From seasonal proportion constraint: S <= floor((3/7) * F)
    s_bound_ratio = math.floor(3 * F / 7)
    # From points constraint: 2S + 5F <= 200 -> S <= floor((200 - 5F)/2)
    s_bound_points = (200 - 5 * F) // 2
    s_ub = min(s_bound_ratio, s_bound_points)
    if s_ub < 0:
        continue
    for S in range(0, s_ub + 1):
        # Check constraints explicitly (redundant but safe)
        if 2 * S + 5 * F > 200:
            continue
        if S > (3 * F) / 7:
            continue
        # Objective: maximize 5S + 8F
        obj = 5 * S + 8 * F
        if best_obj is None or obj > best_obj:
            best_obj = obj
            best_solution = (S, F)

# Print final objective in required format
print(f"FINAL_OBJ: {best_obj}")
