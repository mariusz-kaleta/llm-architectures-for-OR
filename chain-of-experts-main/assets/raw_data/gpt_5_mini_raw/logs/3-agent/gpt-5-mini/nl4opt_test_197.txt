Objective
- Minimize total gelatine: minimize Z = 20x + 60y (mg)

Constraints
1. Antibiotics availability: 30x + 65y ≤ 35,000
2. More first-doses than second-doses: x ≥ y + 1
3. Minimum second-doses: y ≥ 40
4. Integrality and nonnegativity: x, y ∈ Z, x,y ≥ 0

Analytic reasoning / solution
- To reduce gelatine you want both x and y as small as allowed. Constraint (2) ties x to y, so choose x = y + 1.
- Substitute into antibiotics constraint: 30(y+1) + 65y = 95y + 30 ≤ 35,000 → y ≤ floor(34970/95) = 368.
- Because you want y as small as allowed, set y = 40 (minimum). Then x = 41.
- Check feasibility: 30·41 + 65·40 = 1,230 + 2,600 = 3,830 ≤ 35,000, so feasible.

Optimal solution
- First-dose: x = 41
- Second-dose: y = 40
- Minimum gelatine used: 20·41 + 60·40 = 3,220 mg

# Self-contained script to solve the integer LP described and print final objective.
# Decision variables:
#   x: number of first-dose vaccines (integer, x >= 0)
#   y: number of second-dose vaccines (integer, y >= 0)
# Objective:
#   Minimize Z = 20*x + 60*y
# Constraints:
#   30*x + 65*y <= 35000
#   x >= y + 1
#   y >= 40

def solve():
    best_obj = None
    best_x = None
    best_y = None

    # Bounds derived from constraints:
    # y at least 40. Upper bound for y from 65*y <= 35000 -> y <= 538
    y_min = 40
    y_max = 35000 // 65  # 538

    for y in range(y_min, y_max + 1):
        # For given y, x must be at least y+1.
        x_min = y + 1
        # Also x must satisfy antibiotics constraint: 30*x <= 35000 - 65*y -> x <= floor((35000-65y)/30)
        remaining = 35000 - 65 * y
        if remaining < 0:
            continue
        x_max_allowed = remaining // 30
        if x_min > x_max_allowed:
            continue  # no feasible x for this y
        # Since objective coefficient for x (20) is less than for y (60), for fixed y we should pick smallest feasible x (x_min).
        x = x_min
        obj = 20 * x + 60 * y
        if (best_obj is None) or (obj < best_obj):
            best_obj = obj
            best_x = x
            best_y = y

    # As a safety, ensure we found a feasible solution
    if best_obj is None:
        raise RuntimeError("No feasible solution found")

    # Final objective value
    return best_obj, best_x, best_y

if __name__ == "__main__":
    final_obj, x_opt, y_opt = solve()
    # Only the final objective line as required:
    print(f"FINAL_OBJ: {final_obj}")
