Here is a clean mathematical model and a short solution approach for the problem.

Decision variables
- O ∈ Z≥0 : number of otters
- D ∈ Z≥0 : number of dolphins

Objective
- Maximize total tricks: maximize Z = 3·O + 1·D

Constraints
1. Treat availability: 3·O + 5·D ≤ 200
2. Minimum dolphins: D ≥ 10
3. At most 30% of performers are otters:
   O ≤ 0.3·(O + D)  (linearize: 10O ≤ 3O + 3D ⇒ 7O − 3D ≤ 0)
   so use 7·O − 3·D ≤ 0
4. Integrality and nonnegativity: O, D ∈ Z≥0

So the MILP is:
- maximize 3O + D
- subject to
    3O + 5D ≤ 200
    D ≥ 10
    7O − 3D ≤ 0
    O, D ∈ Z≥0

Comments on solving
- This is a small integer linear program (MILP). Suitable solvers/libraries:
  - pulp (Python) with CBC (open-source) or commercial solvers (Gurobi, CPLEX)
  - OR-Tools (CP-SAT) for integer optimization
  - scipy.optimize only for continuous problems (not recommended for integrality)
  - brute-force enumeration is feasible here (O up to floor(200/3)=66)
- Strategy: for each feasible O, set D = floor((200 − 3O)/5) (use as many dolphins as treats allow), check D ≥ max(10, ceil(7O/3)). Pick O that gives highest 3O + D.

Example / computed optimal solution
- By enumeration or MILP solve, an optimal solution is:
  - O = 13 otters
  - D = 32 dolphins
  - Total tricks = 3·13 + 1·32 = 71
  - Treats used = 3·13 + 5·32 = 199 (within 200)
  - Otter share = 13/(13+32) ≈ 0.2889 ≤ 0.3 (constraint satisfied)
  - D ≥ 10 satisfied

Suggested Python approach (pulp sketch)
- Use pulp.LpVariable(..., cat='Integer') for O and D
- Define objective and constraints as above
- Solve with pulp.PULP_CBC_CMD() or another solver
This will directly give the optimal (O, D) and objective value.

import math

# Enumerate feasible integer solutions (O = otters, D = dolphins)
best_obj = None
best_O = None
best_D = None

# Upper bound for O from treats: 3O <= 200 -> O <= 66
for O in range(0, 200 // 3 + 1):
    remaining_treats = 200 - 3 * O
    if remaining_treats < 0:
        continue
    # Max dolphins allowed by treats
    D_max = remaining_treats // 5
    # Minimum dolphins due to constraint D >= 10 and otter-share constraint 7O - 3D <= 0 -> D >= ceil(7O/3)
    D_min_from_share = (7 * O + 2) // 3  # ceil(7O/3)
    D_min = max(10, D_min_from_share)
    if D_min > D_max:
        continue
    # Since objective coefficient for D is positive (1), for fixed O we take largest feasible D (D_max)
    D = D_max
    obj = 3 * O + D
    if (best_obj is None) or (obj > best_obj):
        best_obj = obj
        best_O = O
        best_D = D

# Print final objective value in required format
print(f"FINAL_OBJ: {best_obj}")
