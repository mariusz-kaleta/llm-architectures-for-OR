[MODELER]:
Model (linear program)

- Decision variables
  - x_C = number of chairs produced (≥ 0)
  - x_D = number of dressers produced (≥ 0)

- Objective
  - Maximize profit P = 43 x_C + 52 x_D

- Constraints
  - Stain: 1.4 x_C + 1.1 x_D ≤ 17
  - Oak wood: 2 x_C + 3 x_D ≤ 11
  - Nonnegativity: x_C ≥ 0, x_D ≥ 0

Continuous (LP) solution
- The stain constraint is redundant given the wood constraint (max stain when 2 x_C + 3 x_D ≤ 11 is 7.7 gallons ≤ 17), so the wood constraint controls feasibility.
- Profit per unit of wood: chair = 43/2 = 21.5, dresser = 52/3 ≈ 17.333. Chairs yield higher profit per unit wood, so use all wood for chairs:
  x_C = 11/2 = 5.5, x_D = 0.
- Maximum (continuous) profit = 43 × 5.5 + 52 × 0 = $236.50.

Integer (practical) solution
- If x_C and x_D must be integers, enumerate feasible small integer combinations (wood limits x_C ≤ 5 and x_D ≤ 3). The best integer feasible choice is:
  x_C = 4, x_D = 1 → profit = 4×43 + 1×52 = $224.
- (Other nearby integer candidates: 5 chairs only → $215; 3 dressers + 1 chair → $199, etc.)

Recommended Python libraries
- Continuous LP: scipy.optimize.linprog, PuLP, OR-Tools linear solver, CVXOPT
- Integer/ILP: PuLP (with CBC), OR-Tools CP-SAT or MIP solver, or commercial solvers (Gurobi, CPLEX)
- Example quick solvers: use scipy.optimize.linprog for the continuous LP, and PuLP or OR-Tools to enforce integrality.

```python
import math
import numpy as np

# Problem data
profit = np.array([43.0, 52.0])  # [x_C, x_D]
# Constraints: A x <= b
# Wood: 2 x_C + 3 x_D <= 11
# Stain:1.4 x_C +1.1 x_D <=17
A = np.array([[2.0, 3.0],
              [1.4, 1.1]])
b = np.array([11.0, 17.0])

tol = 1e-9

# Define the candidate constraint lines:
# xC = 0, xD = 0, wood line, stain line
candidates = []

# xC = 0 & xD = 0
candidates.append((0.0, 0.0))

# xC = 0 with wood and stain
# xC = 0 -> xD = b_i / a_i2
candidates.append((0.0, b[0] / A[0,1]))  # with wood
candidates.append((0.0, b[1] / A[1,1]))  # with stain

# xD = 0 with wood and stain
candidates.append((b[0] / A[0,0], 0.0))  # with wood
candidates.append((b[1] / A[1,0], 0.0))  # with stain

# intersection of wood and stain (solve A x = b)
try:
    x_inter = np.linalg.solve(A, b)
    candidates.append((x_inter[0], x_inter[1]))
except np.linalg.LinAlgError:
    pass

# Filter feasible vertices (satisfy both constraints and nonnegativity)
feasible_vertices = []
for xC, xD in candidates:
    if xC >= -tol and xD >= -tol:
        # check constraints A x <= b + tol
        lhs = A.dot(np.array([xC, xD]))
        if np.all(lhs <= b + 1e-7):
            # clamp small negatives to zero
            xC_clamped = max(0.0, xC)
            xD_clamped = max(0.0, xD)
            feasible_vertices.append((xC_clamped, xD_clamped))

# Remove duplicates (within tolerance)
unique_vertices = []
seen = []
for v in feasible_vertices:
    found = False
    for s in seen:
        if math.isclose(v[0], s[0], rel_tol=1e-9, abs_tol=1e-9) and math.isclose(v[1], s[1], rel_tol=1e-9, abs_tol=1e-9):
            found = True
            break
    if not found:
        unique_vertices.append(v)
        seen.append(v)

# Evaluate objective at vertices
best_cont = None
best_cont_val = -math.inf
for xC, xD in unique_vertices:
    val = profit[0]*xC + profit[1]*xD
    if val > best_cont_val + 1e-9:
        best_cont_val = val
        best_cont = (xC, xD)

# If for some reason no vertex found (shouldn't happen), fallback analytic
if best_cont is None:
    # Use wood-only solution (as per analysis)
    xC = b[0] / A[0,0]  # 11/2 = 5.5
    xD = 0.0
    best_cont = (xC, xD)
    best_cont_val = profit[0]*xC + profit[1]*xD

# Integer (ILP) solution by enumeration
# Upper bounds for enumeration from wood constraint
max_xC = int(math.floor(b[0] / A[0,0]))  # floor(11/2) = 5
max_xD = int(math.floor(b[0] / A[0,1]))  # floor(11/3) = 3

best_int = None
best_int_val = -math.inf
for xC in range(0, max_xC + 1):
    for xD in range(0, max_xD + 1):
        # check constraints
        if 2.0*xC + 3.0*xD <= 11.0 + 1e-9 and 1.4*xC + 1.1*xD <= 17.0 + 1e-9:
            val = profit[0]*xC + profit[1]*xD
            if val > best_int_val + 1e-9:
                best_int_val = val
                best_int = (xC, xD)

# Print summary (continuous and integer solutions)
print(f"Continuous LP optimum: x_C = {best_cont[0]:.6f}, x_D = {best_cont[1]:.6f}, Profit = {best_cont_val:.6f}")
print(f"Integer optimum: x_C = {best_int[0]}, x_D = {best_int[1]}, Profit = {best_int_val:.6f}")

# Final required line: print final objective value (choose the integer/practical optimum)
print(f"FINAL_OBJ: {int(round(best_int_val))}")
```
