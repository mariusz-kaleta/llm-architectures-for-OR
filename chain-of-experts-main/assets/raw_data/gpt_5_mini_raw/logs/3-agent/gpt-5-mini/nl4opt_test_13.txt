Model

- Decision variables:
  - t1 >= 0 = minutes of Beam 1
  - t2 >= 0 = minutes of Beam 2

- Objective: minimize total pancreas dose
  - minimize P = 0.3 t1 + 0.2 t2

- Constraints:
  - Tumor must receive at least 3 units:
    0.6 t1 + 0.4 t2 >= 3
  - Skin must receive at most 4 units:
    0.2 t1 + 0.1 t2 <= 4
  - Nonnegativity: t1 >= 0, t2 >= 0

Analysis and solution

Note that the objective equals 0.5 times the tumor dose:
0.3 t1 + 0.2 t2 = 0.5*(0.6 t1 + 0.4 t2).
Thus minimizing pancreas dose is equivalent to minimizing the tumor dose subject to the lower bound; the minimum tumor dose allowed is exactly 3. Hence the minimal pancreas dose is
P_min = 0.5 * 3 = 1.5 units.

All optimal (t1,t2) are those that deliver exactly 3 units to the tumor and respect the skin constraint and nonnegativity:
0.6 t1 + 0.4 t2 = 3,
0.2 t1 + 0.1 t2 <= 4,
t1 >= 0, t2 >= 0.

Solving the equality 0.6 t1 + 0.4 t2 = 3 gives t2 = 7.5 − 1.5 t1. Feasibility (t2 >= 0) requires 0 <= t1 <= 5. The skin constraint does not bind (it allows up to t1 = 65), so the feasible optimal set is:
{(t1,t2) | 0 <= t1 <= 5, t2 = 7.5 − 1.5 t1}.

Two convenient representative optimal solutions:
- Use only Beam 2: t1 = 0, t2 = 7.5 minutes (pancreas dose = 1.5).
- Use only Beam 1: t1 = 5, t2 = 0 minutes (pancreas dose = 1.5).
