Formal model

Variables
- A >= 0 : units of anxiety medication
- D >= 0 : units of anti‑depressant

(If units must be whole numbers, add integrality: A, D ∈ Z≥0. I will give both continuous and integer solutions.)

Objective
Minimize total time T (minutes):
T = 3A + 5D

Constraints
1. Total medication at least 100:
   A + D ≥ 100
2. At least 30 units of anxiety medication:
   A ≥ 30
3. Anxiety at most twice anti‑depressant:
   A ≤ 2D
4. Nonnegativity:
   A ≥ 0, D ≥ 0
(Plus A, D integer if units are discrete.)

Solution (continuous LP)
This is a linear program. Solve analytically by inspecting feasible vertices:
- The optimum occurs where A + D = 100 and A = 2D interchange. Solving A + D = 100 and A = 2D gives:
  A* = 200/3 ≈ 66.6667, D* = 100/3 ≈ 33.3333.
- Minimum total time T* = 3(200/3) + 5(100/3) = 200 + 500/3 = 1100/3 ≈ 366.6667 minutes.

Integer (units must be whole)
If A and D must be integers, test nearby integer feasible points. The best integer solution is:
- A = 66, D = 34 (A + D = 100, A ≤ 2D holds).
- Total time T = 3·66 + 5·34 = 198 + 170 = 368 minutes.

#!/usr/bin/env python3
# Self-contained script to solve the given LP (continuous) and integer variant by enumeration.

# Problem:
# Minimize T = 3A + 5D
# subject to:
#   A + D >= 100
#   A >= 30
#   A <= 2D
#   A, D >= 0
# Optional: A, D integers.

from math import isclose

# Continuous (analytic) solution derived from intersection of A + D = 100 and A = 2D:
A_cont = 200.0 / 3.0
D_cont = 100.0 / 3.0
T_cont = 3.0 * A_cont + 5.0 * D_cont  # should equal 1100/3

# Print continuous solution
print(f"Continuous solution: A = {A_cont}, D = {D_cont}, T = {T_cont}")

# Integer solution by enumeration
min_T_int = None
best_int_solutions = []  # allow multiple optimal integer solutions if ties

# We can bound search reasonably: D from 0..200 (more than enough since A+D >= 100)
for D in range(0, 201):
    # For a given D, feasible A must satisfy:
    # A >= 30
    # A >= 100 - D  (from A + D >= 100)
    # A <= 2D
    A_min = max(30, 100 - D, 0)
    A_max = 2 * D
    if A_min > A_max:
        continue
    # Since cost coefficient for A (3) is less than for D (5),
    # for fixed D it's optimal to take A as large as allowed (A_max).
    # But we'll still iterate to be thorough.
    for A in range(A_min, int(A_max) + 1):
        T = 3 * A + 5 * D
        if (min_T_int is None) or (T < min_T_int):
            min_T_int = T
            best_int_solutions = [(A, D, T)]
        elif T == min_T_int:
            best_int_solutions.append((A, D, T))

# Print integer results
print("Integer optimal solutions (A, D, T):")
for sol in best_int_solutions:
    print(sol)

# As required, the final printed line must be exactly in the format:
# print(f"FINAL_OBJ: {value}")
# We choose to present the integer optimal objective value as the final objective value.
if min_T_int is None:
    # No feasible integer solution found (should not happen here)
    min_T_int = float('inf')

# Final required single-line output (must be the last line)
print(f"FINAL_OBJ: {min_T_int}")
