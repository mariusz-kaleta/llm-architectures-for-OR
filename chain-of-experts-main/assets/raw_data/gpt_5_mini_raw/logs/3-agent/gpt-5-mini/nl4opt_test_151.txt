[MODELER]:
Model

Decision variables
- g = number of gummies (>= 0, integer)
- p = number of pills   (>= 0, integer)

Objective
- Maximize total zinc: maximize Z = 4g + 5p
  (each gummy gives 4 units zinc, each pill gives 5 units zinc)

Constraints
1. Magnesium limit: 3g + 2p ≤ 200   (each gummy 3 Mg, each pill 2 Mg)
2. Minimum pills: p ≥ 10
3. Preference for gummies: g ≥ 3p
4. Integrality and nonnegativity: g, p ∈ Z≥0

Analysis and solution (continuous LP view)
- Because for fixed p the objective increases with g, take g as large as allowed by magnesium: g = (200 − 2p)/3, provided g ≥ 3p.
- The feasibility condition g ≥ 3p leads to 11p ≤ 200, so p ≤ 200/11 ≈ 18.18. Objective as a function of p (with g at its mag-limit) is increasing in p, so the continuous optimum is at p = 200/11 and g = 600/11, giving Z = 3400/11 ≈ 309.09.

Integer solution (required since gummies/pills are whole items)
- p must be integer between 10 and 18. Evaluate p = 18:
  g = floor((200 − 2·18)/3) = floor(164/3) = 54 (and g ≥ 3·18 = 54, OK).
  Z = 4·54 + 5·18 = 216 + 90 = 306.
- Checking neighboring integer p shows p = 18, g = 54 is optimal among integers.

Answer
- Eat 54 gummies and take 18 pills. This uses 3·54 + 2·18 = 198 units of magnesium (≤ 200) and yields 306 units of zinc, which is maximal under the constraints.

Suggested Python libraries to solve this
- For integer linear programming: PuLP, OR-Tools (CP-SAT or linear solver), cvxpy with a MILP backend, Gurobi or CPLEX if available.
- For continuous LP: scipy.optimize.linprog.

Sample solver code (PuLP)
```python
import pulp

# Define problem
prob = pulp.LpProblem("Maximize_Zinc", pulp.LpMaximize)

# Variables (integers)
g = pulp.LpVariable('g', lowBound=0, cat='Integer')
p = pulp.LpVariable('p', lowBound=0, cat='Integer')

# Objective
prob += 4*g + 5*p

# Constraints
prob += 3*g + 2*p <= 200
prob += p >= 10
prob += g >= 3*p

# Solve
prob.solve(pulp.PULP_CBC_CMD(msg=0))

print("Status:", pulp.LpStatus[prob.status])
print("Gummies g =", int(pulp.value(g)))
print("Pills   p =", int(pulp.value(p)))
print("Total zinc =", int(pulp.value(prob.objective)))
```

This will return g = 54, p = 18, total zinc = 306.

import math

# Data initialization
MAG_LIMIT = 200  # magnesium limit
MIN_PILLS = 10

# Search over integer p (pills)
best = {'g': 0, 'p': 0, 'z': -1}
# Upper bound for p from magnesium alone is 100, but preference g >= 3p reduces it.
for p in range(MIN_PILLS, 101):
    # Maximum gummies allowed by magnesium constraint: 3g + 2p <= MAG_LIMIT -> g <= (MAG_LIMIT - 2p)/3
    if 2 * p > MAG_LIMIT:
        break
    g_max = (MAG_LIMIT - 2 * p) // 3
    # Must satisfy preference for gummies: g >= 3p
    if g_max < 3 * p:
        continue
    g = g_max  # since objective increases with g for fixed p, pick max feasible g
    z = 4 * g + 5 * p
    if z > best['z']:
        best = {'g': g, 'p': p, 'z': z}

# Output results
g_opt = best['g']
p_opt = best['p']
z_opt = best['z']

print(f"Gummies (g) = {g_opt}")
print(f"Pills   (p) = {p_opt}")
print(f"Total zinc  = {z_opt}")

# Final required print (must be the last line, exact format)
print(f"FINAL_OBJ: {z_opt}")
