Model

- Decision variables
  - x = number of batches of vitamin shots (integer, x ≥ 0)
  - y = number of batches of vitamin pills (integer, y ≥ 0)

- Objective
  - Maximize the number of people supplied:
    maximize Z = 10 x + 7 y
    (each shots batch supplies 10 people, each pills batch supplies 7)

- Constraints
  1. Vitamin C availability: 30 x + 50 y ≤ 1200
  2. Vitamin D availability: 40 x + 30 y ≤ 1500
  3. Pills must be more numerous than shots (strictly larger). Since x,y are integer batches, enforce y ≥ x + 1.
  4. Shots production capacity: x ≤ 10
  5. Nonnegativity and integrality: x, y ∈ Z, x ≥ 0, y ≥ 0

Remarks: If you treated batches as continuous (not required here), replace integrality by x,y ≥ 0 real and change y ≥ x + ε (or y ≥ x + 1 if integer requirement is important).

Solve and result (analysis & solution)

Solving the integer LP (or its continuous relaxation) gives the optimal solution:
- x = 10 batches of vitamin shots
- y = 18 batches of vitamin pills

This uses all binding vitamin C capacity (30·10 + 50·18 = 300 + 900 = 1200) and satisfies vitamin D and the y ≥ x + 1 constraint. The number of people served is Z = 10·10 + 7·18 = 100 + 126 = 226 people.

Recommended solvers / Python libraries

- For integer linear programming: PuLP (with CBC solver), OR-Tools (CP-SAT), or Gurobi/Cplex if available.
- For continuous LP: scipy.optimize.linprog or PuLP/ORTools as well.

Example (PuLP) Python code

from pulp import LpProblem, LpMaximize, LpVariable, lpSum, PULP_CBC_CMD, LpStatus

# Define problem
prob = LpProblem("Vitamin_batches", LpMaximize)

# Variables (integer)
x = LpVariable('shots_batches', lowBound=0, upBound=10, cat='Integer')
y = LpVariable('pills_batches', lowBound=0, cat='Integer')

# Objective
prob += 10 * x + 7 * y

# Constraints
prob += 30 * x + 50 * y <= 1200   # Vitamin C
prob += 40 * x + 30 * y <= 1500   # Vitamin D
prob += y >= x + 1                # pills more than shots

# Solve
prob.solve(PULP_CBC_CMD(msg=False))
print("Status:", LpStatus[prob.status])
print("Shots batches (x) =", int(x.value()))
print("Pills batches (y)  =", int(y.value()))
print("People served =", int(10 * x.value() + 7 * y.value()))

best_x = None
best_y = None

# Bounds and coefficients initialized directly
C_capacity = 1200
D_capacity = 1500
for x in range(0, 11):  # x <= 10
    # Compute maximum feasible y from the two resource constraints
    rem_C = C_capacity - 30 * x
    rem_D = D_capacity - 40 * x

    if rem_C < 0 or rem_D < 0:

    max_y_C = rem_C // 50
    max_y_D = rem_D // 30

            best_obj = obj
            best_x = x
            best_y = y
