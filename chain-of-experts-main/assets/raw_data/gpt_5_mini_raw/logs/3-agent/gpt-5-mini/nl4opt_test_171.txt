[MODELER]:
Mathematical model (integer linear program)

- Decision variables
  - G ∈ Z≥0 : number of glass jars filled
  - P ∈ Z≥0 : number of plastic jars filled

- Objective
  - Maximize total jars: maximize Z = G + P

- Constraints
  1. Volume constraint: 250 G + 300 P ≤ 20 000 (ml)
  2. Plastic at least twice glass: P ≥ 2 G
  3. At least 20 glass jars: G ≥ 20
  4. Integrality: G, P integers ≥ 0

Analytic simplification
- Use P = 2G + k with k ∈ Z≥0. Volume becomes 850 G + 300 k ≤ 20 000.
- Objective Z = G + P = 3G + k. To maximize Z, note glass adds 3 jars per 850 ml (≈0.003529 jars/ml) while additional plastic adds 1 jar per 300 ml (≈0.003333 jars/ml). So prefer increasing G first subject to constraints.

Compute maximal feasible G:
- 850 G ≤ 20 000 ⇒ G ≤ ⌊20 000 / 850⌋ = 23. Also G ≥ 20, so G ∈ {20,21,22,23}.

For each G compute k = ⌊(20 000 − 850 G) / 300⌋ (fill remaining volume with plastics):
- G = 20 ⇒ k = 10 ⇒ P = 2·20 + 10 = 50 ⇒ Z = 20 + 50 = 70 (uses all 20 000 ml)
- G = 21 ⇒ k = 7  ⇒ P = 49 ⇒ Z = 70 (uses 19 950 ml)
- G = 22 ⇒ k = 4  ⇒ P = 48 ⇒ Z = 70 (uses 19 900 ml)
- G = 23 ⇒ k = 1  ⇒ P = 47 ⇒ Z = 70 (uses 19 850 ml)

Conclusion
- The maximum total number of jars is 70.
- Possible optimal allocations (G, P): (20, 50), (21, 49), (22, 48), or (23, 47). Any of these meets the constraints and attains the maximum 70 jars.

Suggested Python libraries to solve this
- pulp (high-level LP/MIP solver in Python; works with CBC/other solvers)
- ortools.linear_solver (Google OR-Tools MIP solver)
- mip (python-mip) or gurobipy / cplex if commercial solvers are available

Example (pulp) sketch
- Define integer variables G, P
- Add constraints 250*G + 300*P <= 20000, P >= 2*G, G >= 20
- Maximize G + P
- Solve and read G, P (will return one of the optimal tuples above).

[CODER]:
#!/usr/bin/env python3
# Self-contained solver for the integer linear program described.
# Decision variables:
#   G >= 0 integer (glass jars)
#   P >= 0 integer (plastic jars)
# Maximize Z = G + P
# s.t. 250*G + 300*P <= 20000
#      P >= 2*G
#      G >= 20

def solve():
    capacity_ml = 20000
    best_value = None
    best_solutions = []

    # Upper bounds for search (loose)
    G_min = 20
    G_max = capacity_ml // 250  # at most this many glass jars if only glass
    for G in range(G_min, G_max + 1):
        # P must be at least 2G
        P_min = 2 * G
        # From volume constraint: 300*P <= capacity - 250*G
        remaining = capacity_ml - 250 * G
        if remaining < 0:
            continue
        P_max = remaining // 300
        if P_max < P_min:
            continue
        # It's optimal to fill remaining with as many plastics as possible for fixed G,
        # but we'll iterate to be general.
        # However iterating from P_min..P_max small range, so OK.
        for P in range(P_min, P_max + 1):
            if 250 * G + 300 * P <= capacity_ml and P >= 2 * G:
                Z = G + P
                if (best_value is None) or (Z > best_value):
                    best_value = Z
                    best_solutions = [(G, P)]
                elif Z == best_value:
                    best_solutions.append((G, P))

    # Print found optimal allocations (optional informational output)
    # (This is not the final required line; final required line printed below.)
    if best_value is None:
        print("No feasible solution found.")
    else:
        print("Optimal allocations (G, P):")
        for sol in best_solutions:
            G, P = sol
            used = 250 * G + 300 * P
            print(f"  G = {G}, P = {P}, total jars = {G+P}, volume used = {used} ml")

    # Final required output line (must be the last printed line)
    print(f"FINAL_OBJ: {best_value}")

if __name__ == "__main__":
    solve()
