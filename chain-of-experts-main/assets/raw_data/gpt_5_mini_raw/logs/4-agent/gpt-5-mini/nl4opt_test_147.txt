[MODELER]:
Formal ILP model

Decision variables
- c ∈ Z, c ≥ 0 : number of cars used.
- b ∈ Z, 0 ≤ b ≤ 4 : number of buses used (upper bound 4).

Objective
- Minimise total pollution: minimize Z = 10·c + 30·b.

Constraints
- Capacity: 4·c + 20·b ≥ 300 (at least 300 employees transported).
- Integrality and bounds: c, b integers; b ≤ 4; c ≥ 0.

Notes / reasoning
- Pollution per seat: car = 10/4 = 2.5 units/employee; bus = 30/20 = 1.5 units/employee. Because buses are strictly more pollution-efficient per passenger, use as many buses as allowed (b = 4), then satisfy remaining demand with cars.
- With b = 4, capacity from buses = 80 employees. Remaining employees = 300 − 80 = 220. Cars needed: c = ceil(220/4) = 55.
- Optimal solution: b = 4 buses, c = 55 cars. Total pollution Z = 30·4 + 10·55 = 120 + 550 = 670 units.

- gurobi / CPLEX (commercial solvers) if available
- scipy.optimize is not ideal because this is an integer problem (useable only for continuous relaxations)

[CODER]:
import math

# Problem data
demand = 300
car_capacity = 4
bus_capacity = 20
car_cost = 10
bus_cost = 30
max_buses = 4

best_obj = None
best_solution = None

for b in range(0, max_buses + 1):
    remaining = max(0, demand - b * bus_capacity)
    c = math.ceil(remaining / car_capacity)
    if c < 0:
        c = 0
    obj = car_cost * c + bus_cost * b
    if best_obj is None or obj < best_obj:
        best_obj = obj
        best_solution = (c, b)

# best_solution contains (c, b), best_obj is the minimal pollution
# Print final objective in required format
print(f"FINAL_OBJ: {best_obj}")
