A. Assumptions
- There is one bus and one car that can each do multiple sequential trips.
- A bus trip transports 100 chickens and takes 2.0 hours.
- A car trip transports 40 chickens and takes 1.5 hours.
- At most 10 bus trips in total.
- At least 60% of all trips must be car trips.
- Need to transport at least 1200 chickens.
- Decision variables are integer counts of trips (no fractional trips).

B. Variables
- b ∈ Z, 0 ≤ b ≤ 10 : number of bus trips.
- c ∈ Z, c ≥ 0 : number of car trips.
- T ∈ R, T ≥ 0 : (if using makespan objective) the completion time (hours).

C. Constraints
1. Capacity:
   100 b + 40 c ≥ 1200

2. Bus trip limit:
   b ≤ 10

3. Car-share fraction (≥ 60% of total trips by car):
   c / (b + c) ≥ 0.60  (linearized as)  c ≥ 1.5 b
   (Note: this assumes b + c > 0; transportation requirement enforces that.)

4. Integrality and nonnegativity:
   b, c integers, b ≥ 0, c ≥ 0.

5. If using makespan T (primary/recommended objective):
   T ≥ 2 b    (bus total time = 2 hours × b trips)
   T ≥ 1.5 c  (car total time = 1.5 hours × c trips)

D. Objectives (two natural options)
1. Primary (realistic): Minimise makespan T = max(2 b, 1.5 c).
   Formulation as an ILP: minimise T subject to constraints above and T ≥ 2b, T ≥ 1.5c.

2. Alternate (simpler but less realistic): Minimise total vehicle-hours = 2 b + 1.5 c (sum of all trip durations).
   Formulation: minimise 2 b + 1.5 c subject to constraints above (no T variable needed).

E. Small enumeration solution (recommended since b ∈ {0,...,10})
Because b ranges only 0..10, a simple exact enumeration over b and the smallest feasible c is trivial and robust.

For each b in 0..10 compute minimal integer c satisfying:
   c ≥ ceil(1.5 b)   (car-share)
   c ≥ ceil( (1200 - 100 b) / 40 )   (capacity)

Then compute objective values and pick the best.

F. Computed optimal solution (makespan objective)
Using the makespan objective (minimise T = max(2 b, 1.5 c)) the enumeration yields:

- Optimal: b = 8 bus trips, c = 12 car trips.
- Makespan T = max(2·8, 1.5·12) = max(16, 18) = 18.0 hours.
- Capacity used = 8·100 + 12·40 = 1280 chickens (80 seats unused).
- Car-trip fraction = 12 / (8 + 12) = 0.60 (meets exactly 60% requirement).
- This is unique optimal in the feasible set for the makespan objective.

G. If instead you minimise total vehicle-hours (2 b + 1.5 c)
- The best pair is b = 7, c = 13 giving total = 2·7 + 1.5·13 = 14 + 19.5 = 33.5 vehicle-hours.
- Makespan for that pair would be max(14, 19.5) = 19.5 hours (worse than 18 hours).

- Very simple: write a tiny Python enumeration loop (no external solver required).
- ILP solvers (if you prefer formal ILP):
  - pulp (CBC) – simple and pure Python.
  - mip (Coin-or CBC/Gurobi interfaces) – integer programming.
  - OR-Tools CP-SAT (ortools.sat.python) – robust for integer problems.
  - commercial: Gurobi, CPLEX if available.

[CODER]:
import math

# Problem data (initialized directly)
bus_capacity = 100       # chickens per bus trip
car_capacity = 40        # chickens per car trip
bus_time = 2.0           # hours per bus trip
car_time = 1.5           # hours per car trip
min_chickens = 1200
max_bus_trips = 10
min_car_fraction = 0.60  # at least 60% of trips must be car trips

best_T = float('inf')
best_b = None
best_c = None
solutions = []

for b in range(0, max_bus_trips + 1):
    # Car-share constraint: c >= 1.5 * b  (since c/(b+c) >= 0.6 => c >= 0.6(b+c) => 0.4c >= 0.6b => c >= 1.5b)
    c_min1 = math.ceil(1.5 * b)
    # Capacity constraint: 100*b + 40*c >= 1200 -> c >= (1200 - 100*b) / 40
    remaining = max(0, min_chickens - bus_capacity * b)
    c_min2 = math.ceil(remaining / car_capacity) if remaining > 0 else 0
    c = max(c_min1, c_min2)
    # c must be non-negative integer
    if c < 0:
        c = 0

    # Verify feasibility (redundant but safe)
    if bus_capacity * b + car_capacity * c < min_chickens:
        continue
    if c < math.ceil(1.5 * b):
        continue

    T = max(bus_time * b, car_time * c)
    solutions.append((b, c, T))
    if T < best_T - 1e-9:
        best_T = T
        best_b = b
        best_c = c

# Print a brief summary (optional)
if best_b is None:
    print("No feasible solution found.")
else:
    used_capacity = bus_capacity * best_b + car_capacity * best_c
    car_fraction = best_c / (best_b + best_c) if (best_b + best_c) > 0 else 0.0
    print(f"Optimal solution (minimising makespan): b = {best_b}, c = {best_c}")
    print(f"Makespan T = {best_T:.1f} hours")
    print(f"Total capacity used = {used_capacity} chickens")
    print(f"Car trip fraction = {car_fraction:.2f}")

# Final required output line (must be the last line)
print(f"FINAL_OBJ: {best_T}")
