Decision variables

- R ∈ Z≥0 (or R≥0 continuous): number of regular desktops produced and sold per month.

Objective
- Maximise total monthly profit:
  max Z = 500P + 300R

Constraints
1. Sales capacity: P + R ≤ 200
2. Production budget: 2000P + 1000R ≤ 300000
3. Nonnegativity (and optionally integrality): P ≥ 0, R ≥ 0 and optionally P,R integer.

Feasibility check
- Producing 200 regular desktops costs 200·1000 = $200,000 ≤ $300,000, so the feasible region is nonempty.

Solve analytically (corner evaluation)
- Intersection of the two constraints: solve P + R = 200 and 2P + R = 300 → P = 100, R = 100.
- Evaluate corners:
  - (P,R) = (100,100): profit = 500·100 + 300·100 = $80,000
  - (P,R) = (150,0) (budget binding): profit = $75,000
  - (P,R) = (0,200) (sales binding): profit = $60,000
- Optimal solution: P = 100 premium, R = 100 regular, maximum profit = $80,000.

Binding constraints at optimum
- Both constraints bind: P + R = 200 and 2000P + 1000R = 300000.

[CODER]:
import math

# Parameters
profit_P = 500  # profit per premium desktop
profit_R = 300  # profit per regular desktop

max_sales_capacity = 200        # P + R <= 200
budget_dollars = 300000         # 2000*P + 1000*R <= 300000

# For easier arithmetic, convert budget constraint to thousands:
# 2000P + 1000R <= 300000  ->  2P + R <= 300
budget_rhs = budget_dollars // 1000  # 300

# Brute-force integer search (P and R are integers >= 0)
best_profit = -1
best_P = None
best_R = None

for P in range(0, max_sales_capacity + 1):
    # Given P, R must satisfy:
    # R <= max_sales_capacity - P
    # R <= budget_rhs - 2*P
    R_upper1 = max_sales_capacity - P
    R_upper2 = budget_rhs - 2 * P
    R_max = min(R_upper1, R_upper2)
    if R_max < 0:
        continue
    # Since profit increases with R (positive coefficient), choose R = floor(R_max)
    R = int(math.floor(R_max))
    if R < 0:
        continue
    # Check feasibility in original units (safety)
    if P + R > max_sales_capacity:
        continue
    if 2000 * P + 1000 * R > budget_dollars:
        continue
    profit = profit_P * P + profit_R * R
    if profit > best_profit:
        best_profit = profit
        best_P = P
        best_R = R

# Print solution details
print(f"Optimal integer solution: P = {best_P}, R = {best_R}, Profit = {best_profit}")

# Final required output line (must be last)
print(f"FINAL_OBJ: {best_profit}")
