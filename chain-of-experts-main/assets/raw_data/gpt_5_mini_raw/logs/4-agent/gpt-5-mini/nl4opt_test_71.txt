- y ∈ Z, y ≥ 0 : number of doses of medicine B.

Objective
- Maximize total people treated: maximize Z = 12x + 8y.

Constraints

   30x + 40y ≤ 300
2. mRNA capacity:
   50x + 30y ≤ 400
3. Upper bound on A:
   0 ≤ x ≤ 5
4. Strict relationship (B must be strictly greater than A):
   y ≥ x + 1
5. Integrality:
   x, y integer

(Equivalently, the model is a small MILP: maximize 12x + 8y subject to the linear constraints above with integer variables.)

Solution approach
Because x is bounded 0..5 and the problem is tiny, the simplest robust method is to enumerate feasible x values and for each choose the largest feasible integer y ≥ x+1 that satisfies the resource constraints.

Quick enumeration (feasible y upper bounds come from the two resource constraints; we take the tighter bound and the integrality condition):

- x = 0 → y ≤ floor(min((300−30·0)/40, (400−50·0)/30)) = floor(min(7.5, 13.33)) = 7. Max feasible y given y ≥ 1 is 7. Objective = 12·0 + 8·7 = 56.
- x = 1 → y ≤ floor(min((300−30)/40, (400−50)/30)) = floor(min(6.75, 11.67)) = 6. Max feasible y ≥ 2 is 6. Objective = 12·1 + 8·6 = 60.
- x = 2 → y ≤ floor(min((300−60)/40, (400−100)/30)) = floor(min(6.0, 10.0)) = 6. Max feasible y ≥ 3 is 6. Objective = 12·2 + 8·6 = 72.
- x = 3 → y ≤ floor(min((300−90)/40, (400−150)/30)) = floor(min(5.25, 8.33)) = 5. Max feasible y ≥ 4 is 5. Objective = 12·3 + 8·5 = 76.
- x = 4 → y ≤ floor(min((300−120)/40, (400−200)/30)) = floor(min(4.5, 6.67)) = 4. But y must be ≥ 5 → infeasible.
- x = 5 → y ≤ floor(min((300−150)/40, (400−250)/30)) = floor(min(3.75, 5.0)) = 3. But y must be ≥ 6 → infeasible.

Optimal solution from enumeration: x = 3, y = 5 with objective Z = 76 people treated.

- mRNA used = 50·3 + 30·5 = 150 + 150 = 300 ≤ 400.

[CODER]:
import math

# Data (from the problem statement)
imported_capacity = 300
mrna_capacity = 400
# resource coefficients
a_import = 30
b_import = 40
a_mrna = 50
b_mrna = 30
# objective coefficients
a_obj = 12
b_obj = 8
# bounds
x_min, x_max = 0, 5

best_obj = None
best_x = None
best_y = None

for x in range(x_min, x_max + 1):
    # compute upper bounds for y from resource constraints (floor division)
    ub_import = (imported_capacity - a_import * x) // b_import
    ub_mrna = (mrna_capacity - a_mrna * x) // b_mrna
    y_max = min(ub_import, ub_mrna)
    # y must be at least x+1
    y_min = x + 1
    if y_max >= y_min:
        y = y_max
        obj = a_obj * x + b_obj * y
        if (best_obj is None) or (obj > best_obj):
            best_obj = obj
            best_x = x
            best_y = y

# If no feasible solution found, set objective to 0 (though model guarantees feasibility for some x)
if best_obj is None:
    best_obj = 0

print(f"FINAL_OBJ: {best_obj}")
