- T = number of taxi rides (integer, >= 0).

4. Key constraints to include (description, not formal equations)

- Company-car share limit: At most 60% of the rides may be company car rides. Express this as a linear relation between C and total rides (e.g., C <= 0.6 * R). For modelling convenience this can be algebraically rearranged to a linear inequality in C and T (equivalently C <= 1.5 * T).

- Non-negativity and integrality: T and C must be nonnegative integers (rides are whole numbers).

[CODER]:
import math

# Parameters
total_employees = 500
min_company_cars = 30

# Search for minimal T (taxi rides) with feasible C (company-car rides)
best_T = None
best_C = None

# Upper bound for search: 1000 is more than sufficient for this small problem
for T in range(0, 1000):
    # Minimum C required to meet capacity (and minimum company cars)
    C_min_capacity = math.ceil((total_employees - 2 * T) / 3)
    C_min = max(min_company_cars, C_min_capacity)
    # Maximum C allowed by company-car share limit: C <= 1.5 T
    C_max = math.floor(1.5 * T)
    if C_min <= C_max:
        best_T = T
        best_C = C_min
        break

if best_T is None:
    raise RuntimeError("No feasible solution found within search bounds.")

# Final objective value (minimized number of taxi rides)
value = best_T
print(f"FINAL_OBJ: {value}")
