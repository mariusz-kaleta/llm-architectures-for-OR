4) Quick analytic reasoning
From s ≥ 2·l and s + l ≤ 70:
2·l + l ≤ 70 ⇒ 3·l ≤ 70 ⇒ l ≤ floor(70/3) = 23.
Also we need l ≥ 20 and l ≤ 50, so feasible l is 20..23. Flowers per bouquet: large bouquets give more flowers per unit, so choose the maximum feasible l (l = 23). Then s = 70 − l = 47 (this satisfies s ≥ 2·l since 47 ≥ 46, and s ≤ 80). Objective value:
Z = 5·47 + 10·23 = 235 + 230 = 465 flowers.

Thus the optimal plan is s = 47 small bouquets and l = 23 large bouquets, total flowers = 465.

Example using pulp:

from pulp import LpProblem, LpVariable, LpInteger, LpMaximize, lpSum, value

# Define problem
prob = LpProblem("Bouquets", LpMaximize)

# Variables
s = LpVariable('s', lowBound=0, upBound=80, cat=LpInteger)
l = LpVariable('l', lowBound=0, upBound=50, cat=LpInteger)

# Objective
prob += 5*s + 10*l

# Constraints
prob += s + l <= 70
prob += l >= 20
prob += s >= 2*l

# Solve
prob.solve()
print("Status:", prob.status)
print("s =", int(s.value()), "l =", int(l.value()))
print("Total flowers =", value(prob.objective))

Example using OR-Tools CP-SAT:

from ortools.sat.python import cp_model

model = cp_model.CpModel()

model.Add(s + l <= 70)
model.Add(l >= 20)
model.Add(s >= 2*l)

model.Maximize(5*s + 10*l)

solver = cp_model.CpSolver()
solver.Solve(model)
print("s =", solver.Value(s), "l =", solver.Value(l))
print("Total flowers =", 5*solver.Value(s) + 10*solver.Value(l))

# Self-contained solver for the bouquet transportation ILP (enumeration)
# Finds integers s (small bouquets) and l (large bouquets) maximizing 5*s + 10*l
# Subject to:
#   0 <= s <= 80
#   0 <= l <= 50
#   s + l <= 70
#   l >= 20
#   s >= 2*l

best_obj = None
best_s = None
best_l = None

for l in range(0, 51):   # l up to 50
    for s in range(0, 81):  # s up to 80
        if s + l > 70:
            continue
        if l < 20:
            continue
        if s < 2 * l:
            continue
        obj = 5 * s + 10 * l
        if (best_obj is None) or (obj > best_obj):
            best_obj = obj
            best_s = s
            best_l = l
