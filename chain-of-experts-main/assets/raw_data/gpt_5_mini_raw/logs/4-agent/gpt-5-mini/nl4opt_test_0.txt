3) Constraints (linear)
- Capacity: 10·B + 8·C ≥ 300  (at least 300 ducks transported)
- Proportion (at least 60% of trips must be canoe): C / (B + C) ≥ 0.6.
  This is linearised as:
    C ≥ 0.6(B + C)  =>  C − 0.6C ≥ 0.6B  => 0.4C ≥ 0.6B
  Multiply both sides by 5 (to avoid decimals) or directly by 2:
    2·C ≥ 3·B  =>  2C − 3B ≥ 0
- Boat trip upper bound: B ≤ 12
- Integrality and non-negativity: B, C ∈ Z_{\ge 0}.

Full ILP formulation:
min 20B + 40C
s.t.
10B + 8C ≥ 300
2C − 3B ≥ 0
0 ≤ B ≤ 12, C ≥ 0
B, C integers

import math

# Problem data
DUCKS_REQUIRED = 300
BOAT_CAPACITY = 10
CANOE_CAPACITY = 8
BOAT_TIME = 20
CANOE_TIME = 40
B_UPPER = 12  # 0..12 inclusive

best_T = None
best_solutions = []  # list of tuples (B, C, T)

for B in range(0, B_UPPER + 1):
    # minimum C by capacity
    deficit = DUCKS_REQUIRED - BOAT_CAPACITY * B
    C_min_capacity = 0 if deficit <= 0 else math.ceil(deficit / CANOE_CAPACITY)
    # minimum C by proportion: C >= 1.5 * B
    C_min_prop = math.ceil(1.5 * B)
    C = max(C_min_capacity, C_min_prop)
    # Ensure non-negative
    if C < 0:
        C = 0

    # Verify constraints (safety check)
    if (BOAT_CAPACITY * B + CANOE_CAPACITY * C) >= DUCKS_REQUIRED and (2 * C - 3 * B) >= 0:
        T = BOAT_TIME * B + CANOE_TIME * C
        if best_T is None or T < best_T:
            best_T = T
            best_solutions = [(B, C, T)]
        elif T == best_T:
            best_solutions.append((B, C, T))

# Print results
if best_T is None:
    print("No feasible solution found within B <= 12.")
else:
    # Print all optimal solutions found (should be unique per problem statement)
    for sol in best_solutions:
        B, C, T = sol
        ducks = BOAT_CAPACITY * B + CANOE_CAPACITY * C
        proportion = C / (B + C) if (B + C) > 0 else 0.0
        print(f"Optimal solution: B={B}, C={C}, Time={T} minutes, Ducks={ducks}, Proportion={proportion:.6f}")

# Final required single-line output
print(f"FINAL_OBJ: {best_T}")
