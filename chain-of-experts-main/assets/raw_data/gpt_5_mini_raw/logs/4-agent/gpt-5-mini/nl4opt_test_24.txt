1) Decision variables

2) Objective
Minimise total parking spots:
minimise P = 1·S + 2·L

3) Constraints
- Capacity: 2·S + 6·L ≥ 80
- Minimum small units: S ≥ 5
- Proportion (large units ≥ 75% of vehicles): L/(L+S) ≥ 0.75 → L ≥ 3·S
- Integrality and nonnegativity: S, L ∈ Z≥0
- (Optional solver bounds) e.g. S ≤ 40, L ≤ 40

4) Comments on linearity
All constraints and the objective are linear integers → an Integer Linear Program (ILP).

5) Quick analytical solution
Note that if L = 3S (the proportion constraint at equality), capacity becomes
2S + 6(3S) = 2S + 18S = 20S.
To meet capacity 20S ≥ 80 ⇒ S ≥ 4. Since S must be at least 5 anyway, choosing S = 5 and L = 3·5 = 15 satisfies all constraints. For S ≥ 5, L = 3S already yields capacity ≥ 100, so the capacity constraint is not binding given S ≥ 5 and L ≥ 3S.

Objective value at S = 5, L = 15:
P = 1·5 + 2·15 = 5 + 30 = 35 parking spots.
Total people capacity = 2·5 + 6·15 = 10 + 90 = 100 people.

Thus the optimal solution is:
- S* = 5 small units
- L* = 15 large units
- Minimum parking spots = 35
Binding constraints: S ≥ 5 and L ≥ 3S are binding. The capacity constraint is not binding at the optimum (it is slack).

6) How to solve computationally
Because the problem is tiny you can either:
- Do a tiny enumeration: for S from 5 to an upper bound (e.g. 40) compute Lmin = max(3·S, ceil((80 − 2·S)/6), 0), compute P = S + 2·Lmin, pick the minimal P.
- Or formulate the ILP and solve with a standard solver.

import math

# Initialize problem data
required_capacity = 80

# Search bounds (as suggested)
S_min = 5
S_max = 40  # inclusive upper bound for search

best_P = None
best_S = None
best_L = None

for S in range(S_min, S_max + 1):
    # capacity needed after small units
    needed_capacity = max(0, required_capacity - 2 * S)
    # minimal L to satisfy capacity
    L_capacity = math.ceil(needed_capacity / 6)
    # minimal L to satisfy proportion constraint L >= 3*S
    L_prop = 3 * S
    # choose the larger of the two
    L = max(L_capacity, L_prop)
    # compute objective (parking spots)
    P = S + 2 * L

    # track best (minimize P)
    if best_P is None or P < best_P:
        best_P = P
        best_S = S
        best_L = L

# Validate solution
assert best_S is not None
assert best_L is not None
assert (2 * best_S + 6 * best_L) >= required_capacity
assert best_S >= 5
assert best_L >= 3 * best_S

# Optional: compute capacity at optimum
opt_capacity = 2 * best_S + 6 * best_L

# Print a short summary (optional)
print(f"S_opt = {best_S}, L_opt = {best_L}, capacity = {opt_capacity}")

# Final required output line (exact format)
print(f"FINAL_OBJ: {best_P}")
