From C >= 0.25 D and C + D <= B (B = 760000) the minimal total investment for a given D is when C = 0.25 D, giving 1.25 D <= B, so feasible only if D <= B / 1.25. Thus any required minimum D (call it D_min) must satisfy D_min <= B / 1.25. For this problem:
B / 1.25 = 760000 / 1.25 = 608000, and D_min = 20000 <= 608000, so the model is feasible.

5) Analytical solution (since this LP is trivial)
Because both returns are positive and detached houses yield more per dollar than condos, the solver will use the full budget and push as much money as allowed into detached houses subject to the condo-share requirement.

The condo-share constraint binds at optimum: C = 0.25 D, and budget binds: C + D = 760000.
Solve:
1.25 D = 760000 -> D* = 760000 / 1.25 = 608000
C* = 760000 - 608000 = 152000

Objective value:
Z* = 0.5 * 152000 + 1.0 * 608000 = 76000 + 608000 = 684000

Thus optimal investment: C = $152,000 in condos, D = $608,000 in detached houses; total profit = $684,000.
