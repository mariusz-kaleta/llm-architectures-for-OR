max   Z = 3·o + 1·d

3) Constraints

3·o + 5·d ≤ 200

d ≥ 10

a) Using T = o + d:
o ≤ 0.3 · T  (equivalently multiply by 10 to keep integer coefficients)
10·o ≤ 3·T
plus T = o + d

b) Eliminated-T algebraic form:
o ≤ 0.3·(o + d)
0.7·o ≤ 0.3·d
equivalently (multiply by 10):
7·o ≤ 3·d
(or o ≤ (3/7)·d)

o, d (and T if used) are nonnegative integers.

#!/usr/bin/env python3
# Self-contained script to solve the otter/dolphin planning ILP by enumeration.

# Problem data
TREATS_BUDGET = 200
TREAT_OTTER = 3
TREAT_DOLPHIN = 5
MIN_DOLPHINS = 10

# Search bounds
d_min = MIN_DOLPHINS
d_max = TREATS_BUDGET // TREAT_DOLPHIN  # max dolphins if only dolphins purchased

best_Z = None
best_o = None
best_d = None

for d in range(d_min, d_max + 1):
    # From otter proportion constraint: 7*o <= 3*d  => o <= floor((3*d)/7)
    o_cap1 = (3 * d) // 7

    # From treats budget: 3*o + 5*d <= 200 => o <= floor((200 - 5*d)/3)
    remaining_treats = TREATS_BUDGET - TREAT_DOLPHIN * d
    if remaining_treats < 0:
        continue
    o_cap2 = remaining_treats // TREAT_OTTER

    o = min(o_cap1, o_cap2)
    if o < 0:
        continue

    Z = 3 * o + 1 * d  # objective: 3 tricks per otter, 1 per dolphin

    if (best_Z is None) or (Z > best_Z):
        best_Z = Z
        best_o = o
        best_d = d

# Ensure we found a feasible solution
if best_Z is None:
    best_Z = 0
    best_o = 0
    best_d = 0

# Final required output line (only this line must be exactly in this format)
print(f"FINAL_OBJ: {best_Z}")
