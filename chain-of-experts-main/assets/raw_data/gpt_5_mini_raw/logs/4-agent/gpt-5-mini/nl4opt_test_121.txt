- Set x = 500 (upper bound). Then y ≤ floor((1/3)·500) = 166 (integer).
- Check budget: cost = 0.30·500 + 0.50·166 = 150 + 83 = 233 ≤ 500 (budget not binding).
- Profit = 0.75·500 + 0.80·166 = 375 + 132.8 = $507.80.

Therefore the optimal integer solution is:
- Carrots x* = 500
- Cucumbers y* = 166
- Total purchase cost = $233
- Total monthly profit = $507.80

- Carrot upper bound (x ≤ 500) is binding (x* = 500).
- The cucumber count is at the maximum allowable integer under the ratio (y* = floor(x*/3) = 166). In the LP relaxation y would be 166.666..., so integrality prevents that extra 0.666 cucumber.

- Small changes: raising the carrot upper bound above 500 (or raising carrot profit) would increase optimal profit; small changes to the cucumber profit or cost are unlikely to change the recommendation unless they make cucumbers more profitable per dollar than carrots or relax the carrot upper bound.

```python
import math

# Data / parameters
budget = 500.0
cost_carrot = 0.30
cost_cucumber = 0.50
profit_carrot = 0.75
profit_cucumber = 0.80
x_min, x_max = 300, 500

best_obj = -1e9
best_x = None
best_y = None
best_cost = None

for x in range(x_min, x_max + 1):
    # Ratio constraint: y <= floor(x/3)
    y_max_ratio = x // 3

    # Budget constraint: 0.30*x + 0.50*y <= 500 -> y <= floor((500 - 0.30*x)/0.50)
    remaining_budget = budget - cost_carrot * x
    if remaining_budget < 0:
        # No feasible y for this x
        continue
    y_max_budget = int(math.floor(remaining_budget / cost_cucumber + 1e-12))
    if y_max_budget < 0:
        continue

    # Feasible maximal y (profit increases with y, so take the largest feasible y)
    y = min(y_max_ratio, y_max_budget)
    if y < 0:
        continue

    cost = cost_carrot * x + cost_cucumber * y
    if cost - budget > 1e-9:
        # infeasible due to numerical issues
        continue

    obj = profit_carrot * x + profit_cucumber * y
    if obj > best_obj:
        best_obj = obj
        best_x = x
        best_y = y
        best_cost = cost

# Prepare final value formatted to two decimals (as in the problem statement)
value = f"{best_obj:.2f}"

# Final required output line (must be the last line)
print(f"FINAL_OBJ: {value}")
```
