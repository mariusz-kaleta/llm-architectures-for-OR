- Combined with O ≥ 3 gives feasible integer O ∈ {3, 4, ..., 11}.
- For fixed O, A must satisfy 3·O ≤ A ≤ floor((300 − 8·O)/6). For a fixed O the objective is increasing with A (coefficient 12 > 0), so the best A for that O is A = floor((300 − 8·O)/6) provided that value ≥ 3·O.

3) Small enumeration (exact, fast)
- Loop O from 3 to 11, set A_max = floor((300 − 8·O)/6). If A_max ≥ 3·O compute Z = 10·O + 12·A_max. Choose (O,A_max) with largest Z.

```python
#!/usr/bin/env python3
# Self-contained script to solve the small integer optimisation problem by enumeration.

# Problem data (directly initialised)
vitamin_c_limit = 300
coef_c_O = 8
coef_c_A = 6
coef_d_O = 10  # vitamin D from orange boxes
coef_d_A = 12  # vitamin D from apple boxes
pref_multiplier = 3  # A >= 3 * O
O_min = 3  # minimum orange boxes

# Derived bound for O (from 8*O + 6*(3*O) <= 300 -> 26*O <= 300)
O_max = (vitamin_c_limit) // (coef_c_O + coef_c_A * pref_multiplier)  # floor(300/26) = 11
if O_max < O_min:
    O_max = O_min  # still allow loop but will detect infeasibility

best_obj = None
best_O = None
best_A = None

for O in range(O_min, O_max + 1):
    # Maximum A allowed by vitamin C constraint
    remaining_c = vitamin_c_limit - coef_c_O * O
    if remaining_c < 0:
        continue
    A_max = remaining_c // coef_c_A
    # Respect preference constraint A >= pref_multiplier * O
    if A_max < pref_multiplier * O:
        continue
    A = A_max  # objective increases with A, so take the max feasible A
    obj = coef_d_O * O + coef_d_A * A
    if (best_obj is None) or (obj > best_obj):
        best_obj = obj
        best_O = O
        best_A = A

# If no feasible solution found, set best_obj to 0 and variables to None
if best_obj is None:
    best_obj = 0
    best_O = None
    best_A = None

# Optional: print the chosen decision variables (not the final required line)
if best_O is not None:
    print(f"Optimal solution: O = {best_O}, A = {best_A}")
else:
    print("No feasible integer solution found.")

# CRITICAL final line exactly as required
print(f"FINAL_OBJ: {best_obj}")
```
