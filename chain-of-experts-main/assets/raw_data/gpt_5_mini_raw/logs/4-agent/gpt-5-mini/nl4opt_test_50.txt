  Max Z = 12 x1 + 15 x2.

Constraints

- Nonnegativity:
  x1 ≥ 0, x2 ≥ 0.

Remarks about model type
- This is a continuous linear program (LP) with two variables and two resource constraints.

Analytical solution (corner-point method)
- Check corners (axes and intersection of the two binding constraints).
- Intersection of both resource constraints (solve simultaneously):
    0.20 x1 + 0.35 x2 = 20

  Solving gives x1 = 30, x2 = 40.
- Profit at that point: Z = 12(30) + 15(40) = 360 + 600 = $960.
- Axis extremes give smaller profits (x1 = 62.5, x2 = 0 ⇒ Z = 750; x1 = 0, x2 = 57.1429 ⇒ Z ≈ 857.14).
- Therefore optimal solution: x1* = 30 kg, x2* = 40 kg, max profit = $960.

import numpy as np

# Data
# Decision vars: x1 (Mix 1), x2 (Mix 2)
# Objective coefficients (profit per kg)
c = np.array([12.0, 15.0])

# Constraint matrix (rows: constraints, columns: x1, x2)
# 0.20 x1 + 0.35 x2 <= 20  (cat paw)
# 0.80 x1 + 0.65 x2 <= 50  (gold shark)
A = np.array([[0.20, 0.35],
              [0.80, 0.65]])
b = np.array([20.0, 50.0])

# Solve intersection of both binding constraints (A x = b)
x_solution = np.linalg.solve(A, b)
x1_opt, x2_opt = x_solution

# Compute objective at that point
obj_opt = float(c.dot(x_solution))

# Compute axis extremes (for verification)
# x2 = 0 -> find max feasible x1
x1_max_from_cat = b[0] / A[0,0]     # 20 / 0.2 = 100
x1_max_from_gold = b[1] / A[1,0]    # 50 / 0.8 = 62.5
x1_axis = min(x1_max_from_cat, x1_max_from_gold)
obj_x1_axis = c[0] * x1_axis

# x1 = 0 -> find max feasible x2
x2_max_from_cat = b[0] / A[0,1]     # 20 / 0.35
x2_max_from_gold = b[1] / A[1,1]    # 50 / 0.65
x2_axis = min(x2_max_from_cat, x2_max_from_gold)
obj_x2_axis = c[1] * x2_axis

# Compute duals (shadow prices) by solving A^T * y = c
y = np.linalg.solve(A.T, c)
y_cat, y_gold = y  # shadow prices for cat paw and gold shark

# Print results (informational)
print(f"Optimal x1 = {x1_opt:.6f} kg")
print(f"Optimal x2 = {x2_opt:.6f} kg")
print(f"Objective (profit) = {obj_opt:.6f}")
print(f"Axis extreme x1 (x2=0): x1 = {x1_axis:.6f}, profit = {obj_x1_axis:.6f}")
print(f"Axis extreme x2 (x1=0): x2 = {x2_axis:.6f}, profit = {obj_x2_axis:.6f}")
print(f"Shadow price (cat paw) = {y_cat:.6f} $/kg")
print(f"Shadow price (gold shark) = {y_gold:.6f} $/kg")

# Prepare final objective value for printing exactly as required.
final_obj = obj_opt
# If very close to integer, convert to int for neatness
if abs(final_obj - round(final_obj)) < 1e-9:
    final_obj = int(round(final_obj))

print(f"FINAL_OBJ: {final_obj}")
