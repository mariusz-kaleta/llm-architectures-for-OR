4) Objective
Minimise total fuel:
minimize F = 20 T + 15 C.

5) Practical solution approach (exact, simple)
Because T is bounded (0..5), enumerate T = 0..5. For each T compute the minimal integer C that satisfies both constraints:
- C ≥ ceil((500 − 50 T)/30)  (capacity)
- C ≥ ceil(3 T / 7)           (proportion 7C ≥ 3T)
So C_min = max( ceil((500 − 50 T)/30), ceil(3 T / 7), 0 ).
If C_min exists, compute F(T, C_min) = 20 T + 15 C_min. Pick the (T, C) with minimum F.

```python
import math

# Parameters
truck_capacity = 50
car_capacity = 30
truck_fuel = 20
car_fuel = 15
T_max = 5
demand = 500

best_F = None
best_T = None
best_C = None

for T in range(0, T_max + 1):
    # capacity requirement
    cap_remain = demand - truck_capacity * T
    if cap_remain <= 0:
        need_C_cap = 0
    else:
        need_C_cap = math.ceil(cap_remain / car_capacity)
    # car proportion requirement: 7*C >= 3*T  ->  C >= ceil(3*T/7)
    need_C_prop = math.ceil((3 * T) / 7)
    C_min = max(need_C_cap, need_C_prop, 0)
    F = truck_fuel * T + car_fuel * C_min

    if (best_F is None) or (F < best_F):
        best_F = F
        best_T = T
        best_C = C_min

# Optional verification prints (not required)
print(f"Optimal T (trucks): {best_T}")
print(f"Optimal C (cars):   {best_C}")
print(f"Total fuel (L):     {best_F}")
print("Verification:")
print(f"  Capacity satisfied: {truck_capacity*best_T + car_capacity*best_C} >= {demand}")
print(f"  Car proportion:     {best_C} / ({best_T}+{best_C}) = {best_C/(best_T+best_C):.4f} >= 0.30")
print(f"  Truck limit:        {best_T} <= {T_max}")
print(f"  Integrality:        integers OK")

print(f"FINAL_OBJ: {best_F}")
```
