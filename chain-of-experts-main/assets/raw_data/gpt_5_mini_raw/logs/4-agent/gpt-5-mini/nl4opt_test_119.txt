- r ∈ {0, …, 20} (given).

Optional secondary objective (tie-breaker)
- If multiple (r,s) yield the same minimum fuel, minimize total trips T = r + s.

Solution (analysis)
- Regular boats are more fuel-efficient (20 pieces / 10 L = 2 pieces/L) than speed boats (30 pieces / 20 L = 1.5 pieces/L), so we want as many regular trips as allowed while satisfying s ≥ r.
- With r = 20 (the maximum), s must be at least 20 by the proportion constraint. Those values give capacity 20·20 + 30·20 = 1000 (exactly required) and fuel F = 10·20 + 20·20 = 600 liters.
- Any other feasible integer (r,s) yields F ≥ 600, so the unique optimal integer solution is r = 20, s = 20 with total fuel 600 L and total delivered mail 1000 pieces.

# Self-contained script to solve the described ILP by brute-force enumeration.

# Decision variables bounds (directly initialized)
r_min, r_max = 0, 20            # r ∈ {0,...,20}
s_min, s_max = 0, 34            # s ∈ {0,...,34} (ceil(1000/30)=34)

# Parameters
mail_per_r = 20
mail_per_s = 30
fuel_per_r = 10
fuel_per_s = 20
required_mail = 1000

# Best solution trackers
best_obj = None
best_solutions = []  # list of tuples (r,s,obj)

for r in range(r_min, r_max + 1):
    for s in range(s_min, s_max + 1):
        # Constraints:
        # 1) Mail capacity
        if mail_per_r * r + mail_per_s * s < required_mail:
            continue
        # 2) Proportion: s >= r
        if s < r:
            continue
        # Objective: minimize fuel
        obj = fuel_per_r * r + fuel_per_s * s
        if (best_obj is None) or (obj < best_obj):
            best_obj = obj
            best_solutions = [(r, s, obj)]
        elif obj == best_obj:
            best_solutions.append((r, s, obj))

# Tie-breaker: minimize total trips r + s
if not best_solutions:
    raise RuntimeError("No feasible solution found.")
best_solution = min(best_solutions, key=lambda t: (t[0] + t[1], t[0], t[1]))
best_r, best_s, best_obj = best_solution
best_trips = best_r + best_s
best_capacity = mail_per_r * best_r + mail_per_s * best_s

# Optional informational print (can be removed if only FINAL_OBJ is desired)
print(f"Optimal solution found: r={best_r}, s={best_s}, fuel={best_obj}, trips={best_trips}, capacity={best_capacity}")

# Final required print (must be the last line)
print(f"FINAL_OBJ: {best_obj}")
