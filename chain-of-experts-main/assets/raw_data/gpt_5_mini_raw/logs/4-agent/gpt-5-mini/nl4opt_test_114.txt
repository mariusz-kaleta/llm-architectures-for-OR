- Goal: Maximise the total number of patients seen (total swabs = throat swabs + nasal swabs) within the available clinic time.

- Let T = number of throat swabs performed (nonnegative integer).
- Let N = number of nasal swabs performed (nonnegative integer).

4. Key constraints to consider (description form)
- Time budget: The total time consumed by all swabs cannot exceed 20,000 minutes. (A throat swab uses 5 minutes each; a nasal swab uses 3 minutes each.)
- Minimum nasal requirement: At least 30 nasal swabs must be performed (N ≥ 30).
- Ratio requirement: At least four times as many throat swabs as nasal swabs must be done (T ≥ 4 × N).
- Integrality and nonnegativity: T and N must be nonnegative integers.

5. Modelling/solution guidance (how to solve)
- Objective: Maximise T + N subject to the constraints above.
- Observational simplification: Because throat swabs take more time per patient (5 vs 3 minutes) but count equally in the objective, at an optimal solution you should not do more throat swabs than necessary. Therefore expect T = 4 × N (the ratio constraint active).
- Substitute that relation into the time constraint to reduce to a single variable problem: this yields an upper bound on N. Solve for the largest integer N that satisfies the time limit, then set T = 4 × N.
- Check feasibility of the minimum nasal requirement after that (N ≥ 30). If infeasible, report infeasibility.

- Total patients = T + N = 3,476 + 869 = 4,345.
- This satisfies the N ≥ 30 requirement.

# Integer optimization by direct enumeration (self-contained, no external solvers)
# Problem:
# Maximise Z = T + N
# subject to
#   5T + 3N <= 20000
#   N >= 30
#   T >= 4N
#   T, N integers >= 0

best_obj = -1
best_T = None
best_N = None

# N must be at least 30. Upper bound: when T=0, 3N <= 20000 -> N <= 6666, but T>=4N so feasible N will be much smaller.
for N in range(30, 20001):
    # maximum feasible T for given N given time constraint
    if 3 * N > 20000:
        break
    T_max = (20000 - 3 * N) // 5  # floor division
    T_min = 4 * N
    if T_max < T_min:
        continue  # no feasible T for this N
    # For fixed N, objective T+N is increasing in T, so choose T = T_max
    T = T_max
    obj = T + N
    if obj > best_obj:
        best_obj = obj
        best_T = T
        best_N = N

# final objective value printed exactly as required
print(f"FINAL_OBJ: {best_obj}")
