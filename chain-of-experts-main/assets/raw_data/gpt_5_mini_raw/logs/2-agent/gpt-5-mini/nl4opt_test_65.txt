[MODELER]:
Model

- Decision variables
  - L = square feet of engineered laminate planks produced per week (continuous)
  - C = square feet of carpets produced per week (continuous)

- Objective
  Maximize profit: Z = 2.1 L + 3.3 C

- Constraints
  1. Demand minimums:
     - L ≥ 15,000
     - C ≥ 5,000
  2. Shipping contract (total shipped at least 50,000):
     - L + C ≥ 50,000
  3. Raw-material maximums:
     - L ≤ 40,000
     - C ≤ 20,000
  4. Nonnegativity:
     - L, C ≥ 0

Discussion / solution reasoning
- Both profit coefficients (2.1 and 3.3) are positive. With independent upper bounds on L and C, profit is increased by producing more of each product.
- Therefore the optimal choice is to produce each product at its maximum given the raw-material limits:
  - L* = 40,000
  - C* = 20,000
- These values satisfy the lower bounds and L* + C* = 60,000 ≥ 50,000, so all constraints are satisfied.

Optimal profit
- Z* = 2.1(40,000) + 3.3(20,000) = 84,000 + 66,000 = $150,000

Suggested Python solvers / libraries
- pulp (simple linear programming modeling, uses CBC by default)
- scipy.optimize.linprog (for small LPs)
- ortools.linear_solver (Google OR-Tools; fast and robust)
- commercial solvers if available: Gurobi, CPLEX (via their Python APIs)

Example (PuLP)
- Install: pip install pulp
- Code:
  from pulp import LpProblem, LpMaximize, LpVariable, lpSum, value

  prob = LpProblem("Flooring", LpMaximize)
  L = LpVariable("L", lowBound=15000, upBound=40000)   # laminate
  C = LpVariable("C", lowBound=5000, upBound=20000)    # carpets

  # shipping constraint
  prob += L + C >= 50000

  # objective
  prob += 2.1*L + 3.3*C

  prob.solve()
  print("L =", L.varValue)
  print("C =", C.varValue)
  print("Profit = $", value(prob.objective))

This will return L = 40000, C = 20000, Profit = 150000.
