Variables
- x1 = hours run on factory 1 (continuous, >= 0)
- x2 = hours run on factory 2 (continuous, >= 0)

Objective
- Minimize total cost: minimize Z = 300 x1 + 600 x2

Constraints (production must meet or exceed daily demand)
1. Black bears: 5 x1 + 10 x2 >= 20
2. White bears: 6 x1 + 10 x2 >= 5
3. Brown bears: 3 x1 >= 15
4. Nonnegativity: x1 >= 0, x2 >= 0

Notes on constraints
- Constraint (3) implies x1 >= 5 (brown production only from factory 1).
- Given x1 >= 5, constraint (2) is automatically satisfied (6·5 = 30 white bears ≥ 5), so (2) is redundant.
- The problem remains a linear program; if factory run-times must be integer hours, add integer constraints to x1, x2 (becomes an MILP).

Analytic solution
- From brown constraint x1 ≥ 5. At x1 = 5, blacks produced = 5·5 = 25 (≥ 20), whites = 6·5 = 30 (≥ 5). Thus x2 = 0 is feasible.
- Cost at (x1,x2) = (5,0) is Z = 300·5 + 600·0 = $1,500.
- Any positive x2 increases cost (factory 2 is more expensive per hour and not needed to meet any unmet demand), so the optimal solution is x1 = 5, x2 = 0 with minimum cost $1,500.
