Below is a clear LP formulation, optional integer variant, and recommended solvers.

Decision variables
- x1 = number of days the farmer operates the old farm (days, ≥ 0)
- x2 = number of days the farmer operates the new farm (days, ≥ 0)

Objective
Minimize the total operating cost:
minimize Z = 300 x1 + 200 x2

Supply (production) constraints (must meet or exceed contract quantities)
- Raspberries: 2 x1 + 4 x2 ≥ 10
- Blueberries: 2 x1 + 1 x2 ≥ 9
- Strawberries: 4 x1 + 2 x2 ≥ 15

Nonnegativity
- x1 ≥ 0, x2 ≥ 0

(If days must be integer, add integrality constraints x1, x2 ∈ Z≥0; that becomes an MILP.)

Remarks / solution
- This is a linear program in two variables. Solving the continuous LP gives
  x1 = 13/3 ≈ 4.3333 days (old farm),
  x2 = 1/3 ≈ 0.3333 days (new farm),
  with minimum cost Z = 4100/3 ≈ $1,366.67.
- If the farmer needs whole days (integer variables), the best integer-feasible solution near the LP optimum is
  x1 = 4 days, x2 = 1 day (check: 2·4+4·1 = 12 Rasp; 2·4+1·1 = 9 Blue; 4·4+2·1 = 18 Straw),
  cost = 300·4 + 200·1 = $1,400.

- pulp (pure-Python LP/MIP solver interface; uses CBC by default) — good for both LP and MILP.
- OR-Tools (Google) — fast for LP and MILP.
- scipy.optimize.linprog — for continuous LPs (no integer capability).
- cvxpy — convenient for modeling and solving LPs (continuous); can also call MILP solvers via certain backends.

# Ready-to-run Python script to solve the given LP (continuous) and MILP (integer days)
# Uses PuLP (CBC) as the solver backend.
# Initializes data directly in code, solves both models, and prints results.
# Final line must be exactly: print(f"FINAL_OBJ: {value}")

import pulp

def solve_continuous():
    # Define LP problem (minimization)
    prob = pulp.LpProblem("Farm_Operation_LP", pulp.LpMinimize)

    # Decision variables: x1 (old farm days), x2 (new farm days), continuous >= 0
    x1 = pulp.LpVariable("x1", lowBound=0, cat="Continuous")
    x2 = pulp.LpVariable("x2", lowBound=0, cat="Continuous")

    # Objective: minimize 300*x1 + 200*x2
    prob += 300 * x1 + 200 * x2

    # Supply constraints (>=)
    prob += 2 * x1 + 4 * x2 >= 10    # Raspberries
    prob += 2 * x1 + 1 * x2 >= 9     # Blueberries
    prob += 4 * x1 + 2 * x2 >= 15    # Strawberries

    # Solve (suppress solver output)
    solver = pulp.PULP_CBC_CMD(msg=False)
    prob.solve(solver)

    obj = pulp.value(prob.objective)
    x1_val = pulp.value(x1)
    x2_val = pulp.value(x2)
    return obj, x1_val, x2_val

def solve_integer():
    # Define MILP problem (minimization) with integer variables
    prob = pulp.LpProblem("Farm_Operation_MILP", pulp.LpMinimize)

    x1 = pulp.LpVariable("x1", lowBound=0, cat="Integer")
    x2 = pulp.LpVariable("x2", lowBound=0, cat="Integer")

    prob += 300 * x1 + 200 * x2

    prob += 2 * x1 + 4 * x2 >= 10
    prob += 2 * x1 + 1 * x2 >= 9
    prob += 4 * x1 + 2 * x2 >= 15

    solver = pulp.PULP_CBC_CMD(msg=False)
    prob.solve(solver)

    obj = pulp.value(prob.objective)
    x1_val = pulp.value(x1)
    x2_val = pulp.value(x2)
    return obj, x1_val, x2_val

def main():
    cont_obj, cont_x1, cont_x2 = solve_continuous()
    int_obj, int_x1, int_x2 = solve_integer()

    # Print both solutions (informational)
    print("Continuous LP solution:")
    print(f" x1 (old farm days) = {cont_x1}")
    print(f" x2 (new farm days) = {cont_x2}")
    print(f" Objective (cost)   = {cont_obj}")

    print("")  # blank line

    print("Integer MILP solution (whole days):")
    print(f" x1 (old farm days) = {int_x1}")
    print(f" x2 (new farm days) = {int_x2}")
    print(f" Objective (cost)   = {int_obj}")

    # Final required single-line output (continuous LP objective as the primary solution)
    # Must be the very last printed line and match the format exactly.
    print(f"FINAL_OBJ: {cont_obj}")

if __name__ == "__main__":
    main()
