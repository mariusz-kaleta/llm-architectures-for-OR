[MODELER]:
Model

Decision variables
- b = number of electric bikes (integer, b ≥ 0)
- s = number of scooters (integer, s ≥ 0)

Objective
- Maximize total meals delivered: maximize Z = 8 b + 5 s

Constraints
1. Charge availability: 3 b + 2 s ≤ 200
2. At most 30% of vehicles can be bikes: b ≤ 0.3 (b + s)  
   Equivalent linear form: 0.7 b ≤ 0.3 s, or b ≤ (3/7) s
3. Minimum scooters: s ≥ 20
4. Integrality and nonnegativity: b, s ∈ Z≥0

Notes: constraint (2) can be rewritten as b ≤ (3/7) s to keep linearity.

Solution (analysis)
- The continuous relaxation optimum occurs where the charge constraint and the bike-ratio constraint both bind:
  s* = 1400/23 ≈ 60.8696, b* = 600/23 ≈ 26.087, giving Z* ≈ 513.04 (fractional).
- Enforcing integer variables, check nearby integer solutions. The best integer feasible solution is:
  b = 26 bikes, s = 61 scooters.
  This satisfies all constraints:
  - Charge: 3·26 + 2·61 = 78 + 122 = 200 (uses full charge)
  - Ratio: 26 ≤ 0.3·(26+61) since 26/(26+61) ≈ 0.29885 ≤ 0.30
  - s ≥ 20 is satisfied
  Total meals = 8·26 + 5·61 = 208 + 305 = 513 (maximum integer value).

#!/usr/bin/env python3
# Complete, run-ready script to solve the integer optimization by enumeration.

# Problem data (initialized directly)
# Decision variables: b (bikes), s (scooters) - integers >= 0
# Objective: maximize Z = 8*b + 5*s
# Constraints:
# 1) 3*b + 2*s <= 200
# 2) b <= (3/7) * s  <=> 7*b <= 3*s
# 3) s >= 20

best_Z = None
best_b = None
best_s = None

# s minimum is 20, maximum from charge constraint when b=0: 2*s <= 200 -> s <= 100
for s in range(20, 101):
    # Max b from charge constraint: 3*b <= 200 - 2*s -> b <= floor((200-2*s)/3)
    max_b_charge = (200 - 2*s) // 3
    # Max b from ratio constraint: 7*b <= 3*s -> b <= floor( (3*s) / 7 )
    max_b_ratio = (3 * s) // 7
    max_b = min(max_b_charge, max_b_ratio)
    if max_b < 0:
        continue
    for b in range(0, max_b + 1):
        # Check constraints explicitly (redundant but safe)
        if 3*b + 2*s > 200:
            continue
        if 7*b > 3*s:
            continue
        Z = 8*b + 5*s
        if (best_Z is None) or (Z > best_Z):
            best_Z = Z
            best_b = b
            best_s = s

# At the very end, print final objective in the required format
print(f"FINAL_OBJ: {best_Z}")
