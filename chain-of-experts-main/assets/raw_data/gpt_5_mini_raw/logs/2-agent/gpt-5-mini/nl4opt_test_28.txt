Objective
- Maximize total fish transported:
  maximize Z = 100·x + 300·y

Constraints
- Budget: 50·x + 100·y ≤ 1000
- Trip ordering (dog trips must be less than truck trips):
  - If trips must be integer (natural model): x and y are integers and x ≤ y − 1 (enforces x < y).
  - If you allow continuous variables (LP relaxation) you cannot use strict “<”; approximate by x ≤ y − ε for a tiny ε > 0, or enforce a practical lower bound x ≤ y (if equality permitted).
- Nonnegativity: x ≥ 0, y ≥ 0
- (Optional integrality) x, y ∈ Z+ if trips are discrete.

Complete MILP (integer model)
- maximize 100x + 300y
- subject to:
  50x + 100y ≤ 1000
  x ≤ y − 1
  x, y ∈ Z+, x ≥ 0, y ≥ 0

#!/usr/bin/env python3
# Self-contained script to solve the MILP by brute force (integer trips).
# Data initialization
DogCapability = 100
TruckCapability = 300
DogCost = 50
TruckCost = 100
MaxBudget = 1000

# Search integer solutions
max_x = MaxBudget // DogCost  # upper bound for x
max_y = MaxBudget // TruckCost  # upper bound for y

best_obj = None
best_solution = None

for x in range(0, max_x + 1):
    for y in range(0, max_y + 1):
        # Constraints:
        if 50 * x + 100 * y > MaxBudget:
            continue
        if not (x <= y - 1):  # x <= y - 1 enforces x < y for integers
            continue
        # Objective
        z = DogCapability * x + TruckCapability * y  # 100*x + 300*y
        if (best_obj is None) or (z > best_obj):
            best_obj = z
            best_solution = (x, y)

# Print solution details (optional)
if best_solution is None:
    print("No feasible integer solution found.")
    final_value = 0
else:
    x_opt, y_opt = best_solution
    # You may print the chosen variables (not required). Keep final line exact as requested.
    print(f"Optimal integer solution: x={x_opt}, y={y_opt}, Z={best_obj}")
    final_value = best_obj

# Final required output line (must be exactly this format)
print(f"FINAL_OBJ: {final_value}")
