[MODELER]:
Formal model

Variables
- x1 >= 0 : number of doses of pain killer 1 (continuous).  
- x2 >= 0 : number of doses of pain killer 2 (continuous).

(If doses must be integer, add x1,x2 integer — then the problem becomes an MILP.)

Objective
Maximize the amount delivered to the back:
maximize Z = 0.8 x1 + 0.4 x2

Constraints
- Sleep-medicine upper bound: 0.3 x1 + 0.6 x2 ≤ 8
- Legs minimum requirement: 0.5 x1 + 0.7 x2 ≥ 4
- Nonnegativity: x1 ≥ 0, x2 ≥ 0

Solution (continuous doses)
This is a 2-variable linear program. The feasible region is the strip between the two lines in the first quadrant; evaluating the objective at the corner points shows the maximum occurs at the sleep constraint bound on the x1 axis:

Optimal solution:
- x1 = 80/3 ≈ 26.6667 doses of pain killer 1
- x2 = 0 doses of pain killer 2

Objective value (back medicine): Z = 0.8*(80/3) = 64/3 ≈ 21.3333 units.

Remarks
- Intuitively: pain killer 1 gives more back medicine per unit of sleep-medicine consumed, so the sleep-medicine limit is binding and all allowance is used on pain killer 1.
- If doses must be integer, solve the MILP (x1,x2 integer). The integer optimal will be near the continuous solution (e.g., 26 doses of PK1 gives 0.3*26=7.8 ≤8 and back = 20.8; 27 doses would violate sleep since 0.3*27=8.1>8).

Recommended Python solvers / libraries
- pulp (LP and MILP solver, easy API)
- scipy.optimize.linprog (for continuous LPs)
- OR-Tools (linear/MIP solvers)
- cvxpy (for convex optimization; can model LPs and MIPs with appropriate solvers)

Example code (pulp, continuous LP)
from pulp import LpProblem, LpMaximize, LpVariable, lpSum, value

prob = LpProblem('painkiller', LpMaximize)
x1 = LpVariable('x1', lowBound=0)
x2 = LpVariable('x2', lowBound=0)

prob += 0.8*x1 + 0.4*x2  # objective
prob += 0.3*x1 + 0.6*x2 <= 8  # sleep limit
prob += 0.5*x1 + 0.7*x2 >= 4  # legs requirement

prob.solve()
print(value(x1), value(x2), value(prob.objective))

This will return x1 = 80/3, x2 = 0 (for the continuous case).

import math

# Define coefficients and constraints
# Variables: x1 (x), x2 (y)
# Objective: maximize Z = 0.8 x + 0.4 y
# Constraints:
#   0.3 x + 0.6 y <= 8   (sleep)
#   0.5 x + 0.7 y >= 4   (legs)
#   x >= 0, y >= 0

# Represent lines in form a*x + b*y = c
lines = [
    (1.0, 0.0, 0.0),       # x = 0 (y-axis)
    (0.0, 1.0, 0.0),       # y = 0 (x-axis)
    (0.3, 0.6, 8.0),       # sleep constraint (<=)
    (0.5, 0.7, 4.0)        # legs constraint (>=)
]

def intersect(line1, line2, tol=1e-12):
    a1, b1, c1 = line1
    a2, b2, c2 = line2
    det = a1*b2 - a2*b1
    if abs(det) < tol:
        return None
    x = (c1*b2 - c2*b1) / det
    y = (a1*c2 - a2*c1) / det
    return (x, y)

# Generate candidate vertices: intersections of any two lines
candidates = []
for i in range(len(lines)):
    for j in range(i+1, len(lines)):
        pt = intersect(lines[i], lines[j])
        if pt is not None:
            candidates.append(pt)

# Remove duplicates (within tolerance)
unique = []
tol = 1e-9
for (x,y) in candidates:
    found = False
    for (ux,uy) in unique:
        if math.hypot(x-ux, y-uy) <= tol:
            found = True
            break
    if not found:
        unique.append((x,y))

# Feasibility check
def feasible(x, y, tol=1e-9):
    if x < -tol or y < -tol:
        return False
    if 0.3*x + 0.6*y > 8.0 + tol:
        return False
    if 0.5*x + 0.7*y < 4.0 - tol:
        return False
    return True

feasible_points = []
for (x,y) in unique:
    if feasible(x,y):
        # clamp small negatives to zero
        x2 = 0.0 if abs(x) < 1e-12 else x
        y2 = 0.0 if abs(y) < 1e-12 else y
        feasible_points.append((x2, y2))

# It's possible the feasible region is unbounded in some direction, but with these constraints and nonnegativity it's bounded for our objective.
# Also include some axis endpoints (where constraints intersect axes) that might not appear due to numerical issues
axis_points = [
    (8.0, 0.0),                  # legs intersects x-axis
    (0.0, 4.0/0.7),              # legs intersects y-axis
    (8.0/0.3, 0.0),              # sleep intersects x-axis
    (0.0, 8.0/0.6)               # sleep intersects y-axis
]
for (x,y) in axis_points:
    if feasible(x,y):
        # add if not already present
        present = any(math.hypot(x-ux, y-uy) <= tol for (ux,uy) in feasible_points)
        if not present:
            feasible_points.append((x,y))

# Evaluate objective for feasible points
def objective(x,y):
    return 0.8*x + 0.4*y

best_val = -math.inf
best_point = None
for (x,y) in feasible_points:
    val = objective(x,y)
    if val > best_val:
        best_val = val
        best_point = (x,y)

# As a fallback, also consider solving analytically: the sleep constraint bound on x-axis gives x = 8/0.3, y=0
x_sleep_axis = 8.0/0.3
if feasible(x_sleep_axis, 0.0):
    val = objective(x_sleep_axis, 0.0)
    if val > best_val + 1e-12:
        best_val = val
        best_point = (x_sleep_axis, 0.0)

# Print results (final required line must be last)
if best_point is not None:
    x_opt, y_opt = best_point
    z_opt = best_val
    # Optional informative prints (can be removed); keep only the final required print as the last line
    print(f"Optimal x1 = {x_opt}")
    print(f"Optimal x2 = {y_opt}")
    print(f"Objective Z = {z_opt}")
else:
    # No feasible solution found (should not happen for this problem)
    x_opt = 0.0
    y_opt = 0.0
    z_opt = float('nan')
    print("No feasible solution found.")

# Final required output line
print(f"FINAL_OBJ: {z_opt}")
