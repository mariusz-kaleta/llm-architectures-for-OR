[MODELER]:
Formal model

Decision variables
- P = acres of potatoes (nonnegative)
- C = acres of cucumbers (nonnegative)

Objective
- Maximize profit Z = 500 P + 650 C

Constraints
1. Land limit: P + C ≤ 50
2. Minimums: P ≥ 12, C ≥ 15
3. Preference (more cucumbers than potatoes): C ≥ P
4. Resource limit: C ≤ 2 P
5. Nonnegativity: P, C ≥ 0

Solution (analytic)
Because profit per acre is higher for cucumbers (650 > 500) you want as many cucumber acres as feasible. The feasible upper bound on C is given by the tighter of C ≤ 2P and C ≤ 50 − P. For P ≤ 50/3 ≈ 16.6667, C ≤ 2P; for P ≥ 50/3, C ≤ 50 − P. Writing the objective with C at its feasible maximum:

- If P ≤ 50/3, set C = 2P → Z = 500P + 650(2P) = 1800 P (increasing with P).
- If P ≥ 50/3, set C = 50 − P → Z = 500P + 650(50 − P) = 32500 − 150 P (decreasing with P).

Thus Z is maximized at the boundary P = 50/3 = 16 2/3 acres, with C = 2P = 100/3 = 33 1/3 acres. These satisfy all minimums and constraints.

Optimal solution
- Potatoes: P = 50/3 ≈ 16.67 acres
- Cucumbers: C = 100/3 ≈ 33.33 acres
- Maximum profit: Z = $30,000

Recommended Python libraries to solve this LP
- pulp (straightforward LP modeling and CBC solver)
- scipy.optimize.linprog (simple LP solver)
- ortools.linear_solver (Google OR-Tools, fast and robust)
- cvxopt (for convex optimization)

# Objective
Z = 500 * P + 650 * C

# Convert to integer if it's an exact integer value

    value = int(Z)

    # otherwise use float with sufficient precision
    value = float(Z)
