- x_H ≥ 0 : dollars placed on Horse.
- x_S ≥ 0 : dollars placed on Soccer.
- x_C ≥ 0 : dollars kept unbet (cash).

Maximize E = (1 − pB) rB x_B + (1 − pH) rH x_H + (1 − pS) rS x_S

# Self-contained script to solve the described LP and print the final objective value.
# Initializes data directly and computes the optimal allocation via a greedy continuous-knapsack
# approach (works here because the risk constraint and budget make this a continuous LP).

# Data initialization
budget = 100000.0
p = {'B': 0.50, 'H': 0.25, 'S': 0.10}   # probability of losing the stake
r = {'B': 1.2,  'H': 0.5,  'S': 0.1}    # payout per $1 if the bet wins

# Derived quantities
coef = {i: (1 - p[i]) * r[i] for i in p}      # expected payout per $1 invested in bet i
risk_cap = 0.30 * budget                      # allowed weighted loss probability * budget

# Greedy allocation by expected payout per unit of risk (coef / p)
# For continuous allocations this yields the LP optimum when risk constraint binds.
items = []
for i in p:
    # avoid division by zero; none of p are zero here, but include protection
    ratio = coef[i] / p[i] if p[i] > 0 else float('inf')
    items.append((ratio, i))
items.sort(reverse=True)  # descending by ratio

# Allocate
alloc = {i: 0.0 for i in p}
alloc['C'] = 0.0  # cash kept unbet
budget_remaining = budget
risk_remaining = risk_cap

for _, i in items:
    if budget_remaining <= 0:
        break
    if p[i] > 0:
        # maximum investable in this bet given the remaining risk capacity
        max_by_risk = risk_remaining / p[i]
    else:
        # if p[i] == 0, risk does not consume risk budget; can invest up to budget_remaining
        max_by_risk = budget_remaining

    alloc[i] = x
    budget_remaining -= x
