[MODELER]:
Model

Decision variables
- x = number of apple servings (nonnegative integer)
- y = number of carrot servings (nonnegative integer)

Objective
- Maximize total fat: maximize F = 2x + 4y

Constraints
1. Preference ratio: x = 3y  (three times as many apple servings as carrot)
2. Minimum carrot servings: y ≥ 2
3. Folate upper bound: 5x + 3y ≤ 100
4. Integrality: x, y ∈ Z≥0 (servings are whole)

Reduced (substitute x = 3y)
- Objective becomes F = 2(3y) + 4y = 10y (so maximize y)
- Folate: 5(3y) + 3y = 18y ≤ 100 ⇒ y ≤ 100/18 ≈ 5.555
- With integer y and y ≥ 2, feasible y ∈ {2,3,4,5}. The objective increases with y, so choose y = 5.
- Then x = 3y = 15.

Optimal integer solution
- Apple servings x = 15
- Carrot servings y = 5
- Total fat = 2·15 + 4·5 = 30 + 20 = 50 units
- Folate used = 5·15 + 3·5 = 75 + 15 = 90 ≤ 100

If servings were allowed to be fractional (continuous variables), the best is y = 100/18 ≈ 5.555, x = 3y = 50/3 ≈ 16.667, giving fat = 10y = 500/9 ≈ 55.556 and folate exactly 100.

Solvers / libraries to use
- For integer (recommended): PuLP (with CBC), OR-Tools CP-SAT, or any MILP solver (Gurobi, CPLEX).
- For continuous relaxation: scipy.optimize.linprog or PuLP.

Example (PuLP) — integer MILP
- Define integer variables x, y
- Add constraints x == 3*y, y >= 2, 5*x + 3*y <= 100
- Maximize 2*x + 4*y
- Solve with default CBC solver -> returns x=15, y=5.
