[MODELER]:
Formulation

- Decision variables
  - A = dollars invested in apartments (≥ 0)
  - T = dollars invested in townhouses (≥ 0)

- Objective
  - Maximize profit P = 0.10 A + 0.15 T

- Constraints
  - Total money: A + T = 600000
  - Apartment upper bound: A ≤ 200000
  - Apartment at least half of townhouses: A ≥ 0.5 T

Solve analytically
Use A + T = 600000 → T = 600000 − A.
Substitute into A ≥ 0.5 T:
  A ≥ 0.5(600000 − A) → A ≥ 300000 − 0.5 A → 1.5 A ≥ 300000 → A ≥ 200000.
Combining with A ≤ 200000 gives A = 200000. Then T = 600000 − 200000 = 400000.

Profit = 0.10(200000) + 0.15(400000) = 20000 + 60000 = 80000.

Answer
- Invest $200,000 in apartments and $400,000 in townhouses.  
- Maximum annual profit = $80,000.

Suggested solvers / Python libraries
- pulp (LP modeling + CBC solver)
- scipy.optimize.linprog
- ortools.linear_solver or ortools.sat for mixed problems
- cvxopt (convex optimization)

Example (pulp)
```python
import pulp

A = pulp.LpVariable('A', lowBound=0)
T = pulp.LpVariable('T', lowBound=0)

prob = pulp.LpProblem('RealEstate', pulp.LpMaximize)
prob += 0.10*A + 0.15*T
prob += A + T == 600000
prob += A <= 200000
prob += A >= 0.5*T

prob.solve()
print('A =', A.value(), 'T =', T.value(), 'Profit =', pulp.value(prob.objective))
```
This will return A = 200000, T = 400000, Profit = 80000.

[PROGRAMMER-VALIDATOR]:
```python
# Self-contained solver for the given LP (analytic solution)
total_money = 600000
A_upper = 200000  # A <= 200000

# From A >= 0.5 T and T = total - A:
# A >= 0.5*(total - A) -> 1.5 A >= 0.5 total -> A >= (0.5/1.5)*total = (1/3)*total
A_lower_from_ratio = total_money / 3.0

# Feasible A must also be nonnegative and <= total and <= A_upper
A = max(0.0, A_lower_from_ratio)
A = min(A, A_upper, total_money)

T = total_money - A
profit = 0.10 * A + 0.15 * T

# If profit is integral, cast to int for cleaner output
if abs(profit - round(profit)) < 1e-9:
    profit = int(round(profit))

print(f"FINAL_OBJ: {profit}")
```
