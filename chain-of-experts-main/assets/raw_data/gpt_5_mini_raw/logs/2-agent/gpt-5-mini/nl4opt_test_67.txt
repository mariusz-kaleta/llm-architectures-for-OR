[MODELER]:
Formal model

Decision variables
- F = units of fertilizer added (continuous, >= 0)
- S = units of seed added (continuous, >= 0)

Objective
- Minimize total time until lawn is ready:
  minimize T = 0.5·F + 1.5·S

Constraints
- Total units limit: F + S ≤ 300
- Minimum fertilizer: F ≥ 50
- Fertilizer no more than twice seeds: F ≤ 2·S
- Nonnegativity: F ≥ 0, S ≥ 0

Remarks: if units must be integer, add integrality constraints F, S ∈ Z≥0. Otherwise treat as continuous.

Solving / reasoning
- The feasible region is a polygon. Because the objective coefficients are positive, the minimum occurs at the smallest feasible (F,S) in the region.
- From F ≥ 50 and F ≤ 2S we get S ≥ F/2 ≥ 25. The smallest feasible sum F+S is 50+25 = 75, which satisfies F+S ≤ 300.
- Therefore the minimizing point is F = 50, S = 25.
- Minimum total time T = 0.5·50 + 1.5·25 = 25 + 37.5 = 62.5 minutes.

- For linear (continuous) problem: scipy.optimize.linprog, PuLP, OR-Tools linear solver, cvxpy.
- For integer version: PuLP with LpInteger variables, OR-Tools CP-SAT (or linear solver with integer option), or commercial solvers (Gurobi, CPLEX).
