[MODELER]:
Formal model

Decision variables
- x = number of premium desktops produced and sold (units, x ≥ 0, integer)
- y = number of regular desktops produced and sold (units, y ≥ 0, integer)

Objective
Maximize total monthly profit:
Maximize Z = 500 x + 300 y

Constraints
1. Sales capacity: x + y ≤ 200
2. Production cost budget: 2000 x + 1000 y ≤ 300000
3. Nonnegativity (and integrality if required): x, y ≥ 0 (integers)

You can simplify the budget constraint by dividing by 1000:
2 x + y ≤ 300

Solution (analysis)
Feasible corner points are:
- (x,y) = (0,200) → Z = 0·500 + 200·300 = 60,000
- (x,y) = (150,0) (from 2x+y=300 with y=0) → Z = 150·500 = 75,000
- Intersection of x+y=200 and 2x+y=300 → x = 100, y = 100 → Z = 100·500 + 100·300 = 80,000

The best feasible point is x = 100 premium and y = 100 regular, giving maximum profit Z = $80,000.

Recommended libraries to solve
- pulp (simple LP/MIP modeling in Python)
- scipy.optimize.linprog (continuous LP)
- OR-Tools (for integer/linear programming)
- cvxpy (convex optimization modeling)

Example (Python, using pulp)
(from pulp import LpProblem, LpMaximize, LpVariable, PULP_CBC_CMD, lpSum)
prob = LpProblem("Desktops", LpMaximize)
x = LpVariable("premium", lowBound=0, cat="Integer")
y = LpVariable("regular", lowBound=0, cat="Integer")
prob += 500*x + 300*y
prob += x + y <= 200
prob += 2000*x + 1000*y <= 300000
prob.solve(PULP_CBC_CMD(msg=False))
print(x.value(), y.value(), "profit =", 500*x.value() + 300*y.value())

This will return x=100, y=100, profit=80000.

Notes
- If fractional production were allowed, the LP relaxation gives the same integer solution here. If integer constraint is required, use an integer solver (pulp/OR-Tools).

[PROGRAMMER-VALIDATOR]:
# Self-contained solver for the described integer LP (enumeration)
# Decision variables:
#   x: number of premium desktops (integer, >=0)
#   y: number of regular desktops (integer, >=0)
# Objective:
#   Maximize Z = 500*x + 300*y
# Constraints:
#   x + y <= 200
#   2*x + y <= 300

# Coefficients and bounds
profit_coeff_x = 500
profit_coeff_y = 300
max_sales_capacity = 200
# From 2*x + y <= 300 => for y>=0, x <= 150
x_upper_bound = 150

best_profit = None
best_x = None
best_y = None

for x in range(0, x_upper_bound + 1):
    # Determine maximum feasible y for given x
    y_max_by_sales = max_sales_capacity - x
    y_max_by_budget = 300 - 2 * x
    y_max = min(y_max_by_sales, y_max_by_budget)
    if y_max < 0:
        continue
    for y in range(0, int(y_max) + 1):
        profit = profit_coeff_x * x + profit_coeff_y * y
        if (best_profit is None) or (profit > best_profit):
            best_profit = profit
            best_x = x
            best_y = y

# At this point best_x, best_y, best_profit hold the optimal integer solution
# Print final objective value in the required format
print(f"FINAL_OBJ: {int(best_profit)}")
