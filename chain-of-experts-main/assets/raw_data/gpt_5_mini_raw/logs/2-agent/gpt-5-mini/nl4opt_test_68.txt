[MODELER]:
Model

Decision variables
- x1 = hours to run factory 1 (>= 0)
- x2 = hours to run factory 2 (>= 0)

Parameters (per hour)
- Allergy pills: a1 = 20 (factory 1), a2 = 10 (factory 2)
- Fever pills: f1 = 15 (factory 1), f2 = 30 (factory 2)
- Rare compound use: r1 = 20 units/hour (factory 1), r2 = 30 units/hour (factory 2)
- Available rare compound: R = 1000 units
- Required production: A_req = 700 allergy, F_req = 600 fever

Objective
Minimize total running time: minimize z = x1 + x2.

Constraints
1) Allergy demand: 20 x1 + 10 x2 >= 700
2) Fever demand: 15 x1 + 30 x2 >= 600
3) Rare compound capacity: 20 x1 + 30 x2 <= 1000
4) Nonnegativity: x1 >= 0, x2 >= 0

This is a linear program (continuous variables).

Analytic solution (vertex check)
Solve the two demand equalities simultaneously:
2 x1 + x2 = 70  (from allergy)
x1 + 2 x2 = 40  (from fever)

Solving gives
x1 = 100/3 ≈ 33.333 hours
x2 = 10/3  ≈ 3.333 hours

Check rare compound: 20*x1 + 30*x2 = 2300/3 ≈ 766.67 <= 1000, so the compound constraint is not binding.

Objective value: x1 + x2 = 110/3 ≈ 36.6667 hours (minimum).

Remarks
- If hours must be integer, the problem becomes an integer program; use a MILP solver and expect a (possibly slightly larger) optimal integer solution.
- If you want to include setup costs or bounds on maximum runtime per factory, add corresponding constraints.

Suggested Python libraries to solve this
- pulp (simple, pure-Python LP/MILP)
- scipy.optimize.linprog (continuous LP)
- OR-Tools linear solver (supports LP and MILP)
- cvxpy (convex optimization; expressive)

Example (pulp)
from pulp import LpProblem, LpMinimize, LpVariable, lpSum, PULP_CBC_CMD

prob = LpProblem("pills", LpMinimize)
x1 = LpVariable("x1", lowBound=0)
x2 = LpVariable("x2", lowBound=0)

prob += x1 + x2
prob += 20*x1 + 10*x2 >= 700
prob += 15*x1 + 30*x2 >= 600
prob += 20*x1 + 30*x2 <= 1000

prob.solve(PULP_CBC_CMD(msg=False))
print("x1 =", x1.value(), "hours; x2 =", x2.value(), "hours; total =", x1.value()+x2.value())

The above returns x1 = 100/3, x2 = 10/3 (approximately) and total ≈ 36.6667 hours.

[PROGRAMMER-VALIDATOR]:
# Self-contained script to compute the LP solution described and print the final objective value.
# Decision variables: x1, x2 (hours)
# Constraints given; analytic solution from solving equality of allergy and fever constraints.

# Initialize parameters
a1, a2 = 20.0, 10.0    # allergy per hour (factory1, factory2)
f1, f2 = 15.0, 30.0    # fever per hour (factory1, factory2)
r1, r2 = 20.0, 30.0    # rare compound usage per hour (factory1, factory2)
R = 1000.0             # available rare compound
A_req = 700.0          # required allergy pills
F_req = 600.0          # required fever pills

# Solve the two demand equalities:
# 20 x1 + 10 x2 = 700  --> 2 x1 + x2 = 70
# 15 x1 + 30 x2 = 600  --> x1 + 2 x2 = 40
# Solve manually:
# From equations:
# 2*x1 + x2 = 70
# x1 + 2*x2 = 40
# Multiply second by 2: 2*x1 + 4*x2 = 80
# Subtract first: 3*x2 = 10 -> x2 = 10/3
x2 = 10.0 / 3.0
# x1 from first: 2*x1 = 70 - x2 -> x1 = (70 - x2)/2
x1 = (70.0 - x2) / 2.0

# Verify constraints (feasibility check)
allergy = a1 * x1 + a2 * x2
fever = f1 * x1 + f2 * x2
rare_used = r1 * x1 + r2 * x2
feasible = (allergy + 1e-9 >= A_req) and (fever + 1e-9 >= F_req) and (rare_used <= R + 1e-9) and (x1 >= -1e-9) and (x2 >= -1e-9)

if not feasible:
    # As a fallback (should not be needed for this problem), brute-force search over a fine grid to find a feasible minimal x1+x2.
    best_obj = float('inf')
    best_x1 = 0.0
    best_x2 = 0.0
    # search up to some reasonable max hours
    max_hours = 1000.0
    step = 0.01
    h = 0.0
    while h <= max_hours:
        # try x1 = h, compute minimal x2 needed to meet both demands given x1, then check rare constraint
        x1_try = h
        # Compute required x2 from allergy and fever demands:
        # From allergy: x2 >= (A_req - a1*x1)/a2
        # From fever:   x2 >= (F_req - f1*x1)/f2
        req_x2_allergy = max(0.0, (A_req - a1 * x1_try) / a2)
        req_x2_fever = max(0.0, (F_req - f1 * x1_try) / f2)
        x2_try = max(req_x2_allergy, req_x2_fever)
        rare_try = r1 * x1_try + r2 * x2_try
        if rare_try <= R + 1e-9:
            obj = x1_try + x2_try
            if obj < best_obj:
                best_obj = obj
                best_x1 = x1_try
                best_x2 = x2_try
        h += step
    if best_obj == float('inf'):
        # No feasible solution found
        FINAL_VALUE = float('nan')
    else:
        FINAL_VALUE = best_obj
else:
    FINAL_VALUE = x1 + x2

print(f"FINAL_OBJ: {FINAL_VALUE}")
