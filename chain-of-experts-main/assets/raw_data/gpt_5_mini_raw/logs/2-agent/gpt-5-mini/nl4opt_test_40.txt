Constraints

2. At least 5 seniors:
   x ≥ 5
3. Number of seniors at least one third of juniors:

4. Weekly wage bill not to exceed the budget (interpret “below $150000” as ≤ 150000; if it is strictly <150000 the problem is infeasible as shown below):
   3000 x + 1000 y ≤ 150000
5. Nonnegativity (and integrality if required):
   x, y ≥ 0   (and integer if required)

Full LP (standard form)
minimize Z = 3000 x + 1000 y
subject to
  x + y ≥ 100
  x ≥ 5
  3x − y ≥ 0
  3000 x + 1000 y ≤ 150000
  x, y ≥ 0  (and x,y ∈ Z if integrality is required)

Analytic observation / optimal solution
From x + y ≥ 100 and 3x ≥ y we get x ≥ 25 (since 100 − x ≤ y ≤ 3x ⇒ 100 ≤ 4x). The smallest feasible x is 25, which forces y = 75. That yields Z = 3000·25 + 1000·75 = $150,000. Thus the minimum attainable wage bill (with the ≤150000 budget) is $150,000 at x = 25 seniors and y = 75 juniors. If the budget constraint were strict (3000x + 1000y < 150000), the problem would be infeasible.
