Formal model

Decision variables

- y = number of cups of drink B (y >= 0, real)

Objective
- Minimize total Vitamin K: minimize Z = 4x + 12y

Constraints
- Vitamin A: 8x + 15y >= 150
- Vitamin D: 6x + 2y >= 300
- Vitamin E: 10x + 20y <= 400
- Nonnegativity: x >= 0, y >= 0

Comments about feasibility
- The system is infeasible. Combine the D and E constraints to see why:

  From E: 10x + 20y <= 400  => 5x + 10y <= 200
  From D: 6x  +  2y >= 300  => 30x + 10y >= 1500  (multiply D by 5)

  Subtract the first from the second:
  (30x + 10y) - (5*(5x + 10y)) >= 1500 - 1000  ==> 30x + 10y >= 1500 and 25x + 50y <= 1000
  A simpler manipulation yields y <= -6, which contradicts y >= 0.

- Intuition: to meet the Vitamin D requirement (6x + 2y >= 300) while keeping x,y >= 0 you need at least x = 50, y = 0 (or other nonnegative mixes). That already produces Vitamin E = 10*50 + 20*0 = 500 > 400. In fact the minimal possible Vitamin E while satisfying the D requirement (over nonnegative x,y) is 500, so the imposed upper bound 400 on Vitamin E cannot be met along with the D requirement.

Conclusion
- There is no feasible combination of drinks A and B (with nonnegative cups) that satisfies all three vitamin constraints simultaneously. Therefore the minimization problem has no feasible solution.

#   8x + 15y >= 150   (Vitamin A)

D_coeff = (6.0, 2.0)    # Vitamin D coefficients
E_coeff = (10.0, 20.0)  # Vitamin E coefficients
A_rhs = 150.0
D_rhs = 300.0

# If min E > 400 then infeasible.
# Solve: minimize 10x + 20y subject to 6x + 2y >= 300, x>=0,y>=0
# Consider boundary points: intersection with axes
#   y=0 -> 6x >= 300 -> x >= 50 -> minimal x on boundary is x = 50 -> E = 10*50 + 20*0 = 500
#   x=0 -> 2y >= 300 -> y >= 150 -> E = 10*0 + 20*150 = 3000
# The boundary of 6x+2y=300 between those axis intersections yields E linear and minimized at x=50, y=0 giving E=500.
min_E_given_D = 500.0

feas_messages: List[str] = []

if min_E_given_D > E_ub + 1e-9:
    feasible = False
    feas_messages.append(
        f"Minimal Vitamin E achievable while meeting D and nonnegativity is {min_E_given_D},"
        f" which exceeds the upper bound {E_ub}."

# For completeness, attempt to locate any feasible point (should not exist).
# We'll try to solve the system by checking corner candidates derived from combinations of constraints treated as equalities.
# Represent line equations in the form a*x + b*y = c
def intersect_lines(a1: float, b1: float, c1: float, a2: float, b2: float, c2: float) -> Optional[Tuple[float, float]]:
    det = a1 * b2 - a2 * b1
    if abs(det) < 1e-12:
        return None
    x = (c1 * b2 - c2 * b1) / det
    y = (a1 * c2 - a2 * c1) / det
    return (x, y)
