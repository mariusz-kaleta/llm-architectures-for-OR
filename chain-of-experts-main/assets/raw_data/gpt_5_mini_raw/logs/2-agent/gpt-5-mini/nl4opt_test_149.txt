[MODELER]:
Model

- Decision variables:
  - x = number of long desks (integer, x >= 0)
  - y = number of short desks (integer, y >= 0)

- Objective:
  - Maximize seating S = 6x + 2y

- Constraints:
  - Budget: 300x + 100y <= 2000
  - Area: 10x + 4y <= 200
  - Integrality and nonnegativity: x, y ∈ Z_{\ge 0}

Remarks and solution
- Divide the budget constraint by 100: 3x + y <= 20.
- The objective can be written as S = 2(3x + y). Thus maximizing S is equivalent to maximizing (3x + y).
- Since (3x + y) is constrained by the budget to be ≤ 20, the maximum possible is 20, giving S_max = 2·20 = 40 seats.
- Any integer pair (x,y) with 3x + y = 20 and x,y ≥ 0 satisfies the budget equality and attains the maximum 40 seats. The area constraint is not binding for these pairs (it is satisfied automatically).

Integer solutions (examples)
- x = 0, y = 20 → 40 seats (area = 80 ft^2, cost = $2000)
- x = 1, y = 17 → 40 seats
- x = 2, y = 14 → 40 seats
- x = 3, y = 11 → 40 seats
- x = 4, y = 8 → 40 seats
- x = 5, y = 5 → 40 seats
- x = 6, y = 2 → 40 seats

So one convenient solution is to buy 6 long desks and 2 short desks (uses full budget, well under the area limit). There are multiple optimal combinations as listed.

Suggested solvers / Python libraries
- For integer linear programming: PuLP, OR-Tools (CP-SAT / MIP), mip, or Pyomo with a suitable solver (CBC, GLPK, Gurobi, CPLEX).
- For continuous relaxation (not required here): scipy.optimize.linprog.

- Define variables as LpInteger, constraints as above, maximize objective, solve with CBC (default).

[PROGRAMMER-VALIDATOR]:
import math

# Problem data (initialized directly)
# Decision variables: x = number of long desks, y = number of short desks (nonnegative integers)
# Objective: maximize S = 6x + 2y
# Constraints:
#   Budget: 300x + 100y <= 2000  ->  3x + y <= 20
#   Area:   10x + 4y <= 200

best_S = -1
best_solutions = []

# Bound for x from budget: 3x <= 20 -> x <= 6
for x in range(0, 7):  # 0..6
    # Budget-derived max y
    y_max_budget = 20 - 3 * x
    if y_max_budget < 0:
        continue
    # Area-derived max y
    remaining_area = 200 - 10 * x
    if remaining_area < 0:
        continue
    y_max_area = remaining_area // 4
    y_max = min(y_max_budget, y_max_area)
    for y in range(0, y_max + 1):
        S = 6 * x + 2 * y
        if S > best_S:
            best_S = S
            best_solutions = [(x, y)]
        elif S == best_S:
            best_solutions.append((x, y))

# At this point best_S holds the optimal seating capacity
print(f"FINAL_OBJ: {best_S}")
