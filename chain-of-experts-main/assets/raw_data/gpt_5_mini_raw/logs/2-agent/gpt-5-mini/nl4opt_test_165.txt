[MODELER]:
Model (linear program)

- Decision variables:
  - x1 ≥ 0 = number of regular tacos to make
  - x2 ≥ 0 = number of deluxe tacos to make

- Objective:
  - Maximize profit P = 2.50*x1 + 3.55*x2

- Constraints:
  - Demand constraints: x1 ≤ 50, x2 ≤ 40
  - Supply/capacity constraint: x1 + x2 ≤ 70
  - Nonnegativity: x1 ≥ 0, x2 ≥ 0

Solution reasoning
- Profit per unit: deluxe (3.55) > regular (2.50), so make as many deluxe tacos as feasible.
- Max deluxe is min(demand, capacity) = min(40,70) = 40 → set x2 = 40.
- Remaining capacity for regular = 70 - 40 = 30, demand allows up to 50, so set x1 = 30.
- This respects all constraints and is a vertex of the feasible region, hence optimal for this LP.

Optimal solution
- x1 = 30 regular tacos
- x2 = 40 deluxe tacos
- Maximum profit = 2.50*30 + 3.55*40 = $217.00

- OR-Tools linear solver (Google OR-Tools)
- CVXOPT or commercial solvers (Gurobi, CPLEX) if needed

Example (PuLP)

- prob = LpProblem("Tacos", LpMaximize)

- prob += x1 + x2 <= 70
- prob.solve()
- print(x1.value(), x2.value(), value(prob.objective))
