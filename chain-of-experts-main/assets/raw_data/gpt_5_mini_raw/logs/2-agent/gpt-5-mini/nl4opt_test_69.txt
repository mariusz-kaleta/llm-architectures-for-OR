Decision variables
- W ∈ Z, W ≥ 5 : number of wide pipes
- N ∈ Z, N ≥ 0 : number of narrow pipes

Objective
- Minimize the total number of pipes: minimize Z = W + N

Constraints
1. Capacity constraint: 25W + 15N ≥ 900
2. Logistics constraint: W ≤ N/3  (equivalently 3W ≤ N)
3. Integrality and bounds: W, N integers with W ≥ 5, N ≥ 0

Full integer linear program
minimize W + N
subject to
- 25W + 15N ≥ 900
- 3W − N ≤ 0
- W ≥ 5, N ≥ 0
- W, N integer

Solution approach (simple exact enumeration)
For any integer W ≥ 5, the smallest feasible N is
N = max(3W, ceil((900 − 25W)/15)).

import math

# Problem data
min_W = 5
max_W = math.ceil(900 / 25)  # 36
best_total = None
best_solutions = []

for W in range(min_W, max_W + 1):
    # capacity requirement: 25W + 15N >= 900 -> N >= ceil((900 - 25W)/15)
    cap_deficit = 900 - 25 * W
    if cap_deficit > 0:
        N_cap = (cap_deficit + 15 - 1) // 15
    else:
        N_cap = 0
    # logistics requirement: N >= 3W
    N_log = 3 * W
    N = max(N_cap, N_log)
    total = W + N
    if (best_total is None) or (total < best_total):
        best_total = total
        best_solutions = [(W, N)]
    elif total == best_total:
        best_solutions.append((W, N))

# Optionally, best_solutions contains all optimal (W,N) pairs
# Print final objective value in required format
print(f"FINAL_OBJ: {best_total}")
