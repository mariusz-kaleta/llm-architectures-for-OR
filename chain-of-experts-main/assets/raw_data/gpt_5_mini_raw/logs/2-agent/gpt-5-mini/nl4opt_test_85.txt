[MODELER]:
Model type: Linear program (continuous variables, maximize revenue).

Decision variables
- x1 = number of (possibly fractional) tanks of light oil to process (>= 0)
- x2 = number of tanks of non-sticky oil to process (>= 0)
- x3 = number of tanks of heavy oil to process (>= 0)

Objective (maximize net revenue)
Maximize Z = 550 x1 + 750 x2 + 950 x3

Resource constraints
- Compound A: 3 x1 + 6 x2 + 9 x3 <= 250
- Compound B: 3 x1 + 2 x2 + 3 x3 <= 150

Nonnegativity
- x1, x2, x3 >= 0

Solution (analytic)
Solve the two resource constraints as equalities (intersection of the two binding constraints):
3x1 + 6x2 + 9x3 = 250
3x1 + 2x2 + 3x3 = 150

From these equations you get:
x1 = 100/3 ≈ 33.333...
x2 = 25 - 1.5 x3

The objective along this line is Z = 37083.333... − 175 x3, which is decreasing in x3, so x3 should be as small as possible (x3 >= 0). Thus x3* = 0, and
x1* = 100/3 ≈ 33.333, x2* = 25.

Maximum net revenue
Z* = 550*(100/3) + 750*25 + 950*0 = $37,083.33 (approximately).

Interpretation
Process about 33.33 tanks of light oil and 25 tanks of non-sticky oil, and do not process heavy oil. This uses all of both compounds:
- Compound A used: 3*(100/3) + 6*25 = 100 + 150 = 250
- Compound B used: 3*(100/3) + 2*25 = 100 + 50 = 150

- ortools.linear_solver (Google OR-Tools) — industrial-strength
- cvxpy — convenient modeling, many solvers available

Example using PuLP
(install with pip install pulp)
from pulp import LpProblem, LpMaximize, LpVariable, lpSum, value

prob = LpProblem("MapleOil", LpMaximize)
x1 = LpVariable("x1", lowBound=0)
x2 = LpVariable("x2", lowBound=0)
x3 = LpVariable("x3", lowBound=0)

prob += 550*x1 + 750*x2 + 950*x3
prob += 3*x1 + 6*x2 + 9*x3 <= 250
prob += 3*x1 + 2*x2 + 3*x3 <= 150

prob.solve()
print(value(x1), value(x2), value(x3), value(prob.objective))

This will return x1 ≈ 33.3333, x2 = 25, x3 = 0 and objective ≈ 37083.33.
