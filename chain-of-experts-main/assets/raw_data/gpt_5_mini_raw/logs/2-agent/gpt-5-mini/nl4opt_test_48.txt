I will translate this into a formal integer linear program (and then show the model is infeasible under the given numbers).

Decision variables
- x = number of motorcycles (nonnegative integer)
- y = number of sedans (nonnegative integer)

Objective
- Maximize total earnings per shift:
  maximize Z = 100 x + 225 y

Constraints
1. Transport requirement (must carry at least 1200 people):
   30 x + 70 y ≥ 1200

2. Pollution limit (company committed to producing less than 200 units):
   4 x + 15 y < 200
   (In linear programming we normally use a non-strict inequality; with integer variables you can enforce this as 4x + 15y ≤ 199. I will keep the conceptual constraint as < 200.)

3. Motorcycle fraction limit (at most 25% of vehicles may be motorcycles):
   x ≤ 0.25 (x + y)
   Rearranged: 3 x ≤ y

4. Integrality and nonnegativity:
   x, y ∈ Z≥0

Feasible-check and infeasibility proof (short)
- From the motorcycle-fraction constraint: y ≥ 3 x. The smallest y for any x that respects the 25% cap is y = 3x.
- Substitute y = 3x into the capacity constraint:
  30 x + 70 (3x) = 30x + 210x = 240 x ≥ 1200 ⇒ x ≥ 1200 / 240 = 5.
  So any feasible solution must have x ≥ 5.
- Substitute y = 3x into the pollution constraint:
  4x + 15(3x) = 4x + 45x = 49x < 200 ⇒ x < 200 / 49 ≈ 4.08.
  Therefore any feasible solution must satisfy x ≤ 4 (integer).
- These two integer requirements (x ≥ 5 and x ≤ 4) are incompatible. Therefore no (integer) x and y satisfy all constraints simultaneously.
Conclusion: The problem as stated has no feasible solution — there is no combination of motorcycles and sedans that satisfies the transport requirement, the pollution limit, and the at-most-25%-motorcycles vehicle mix simultaneously.

Remarks and remedies
- If you relax the pollution cap to at least 245 units, the mix x = 5, y = 15 (which respects y = 3x) meets the capacity (30*5 + 70*15 = 1200) and gives pollution 49*5 = 245.
- If instead you relax the motorcycle-fraction cap (allowing a higher share of motorcycles), then there exist feasible solutions (e.g., all motorcycles: x = 40, y = 0 gives capacity 1200, pollution 160 < 200, revenue = 40 * 100 = 4000, but violates the “at most 25% motorcycles” rule).
