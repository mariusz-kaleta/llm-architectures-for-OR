Here is a linear programming model of the problem.

### Decision variables
Let  
- \(x\) = number of chairs produced (per week)  
- \(y\) = number of dressers produced (per week)

We’ll assume \(x, y \ge 0\) and can be continuous, unless the problem explicitly requires integer production.

---

### Objective function
Maximize total weekly profit:

\[
\max Z = 43x + 52y
\]

---

### Constraints

**Stain constraint**: Each chair uses 1.4 gallons and each dresser uses 1.1 gallons; only 17 gallons are available:

\[
1.4x + 1.1y \le 17
\]

**Wood constraint**: Each chair uses 2 lengths of oak, each dresser uses 3; only 11 lengths are available:

\[
2x + 3y \le 11
\]

**Non-negativity**:

\[
x \ge 0,\quad y \ge 0
\]

This is the complete LP model.

---

### Solving the model (outline with Python)

For a small LP like this, you can use:

- `PuLP` (simple LP modeling, open-source)
- `scipy.optimize.linprog`
- `ortools.linear_solver` from Google OR-Tools

Example with PuLP:

```python
import pulp as pl

# Define the problem
prob = pl.LpProblem("Elm_Furniture_Profit_Max", pl.LpMaximize)

# Decision variables
x = pl.LpVariable('chairs', lowBound=0)
y = pl.LpVariable('dressers', lowBound=0)

# Objective
prob += 43 * x + 52 * y

# Constraints
prob += 1.4 * x + 1.1 * y <= 17   # stain
prob += 2 * x + 3 * y <= 11       # wood

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Optimal chairs (x):", x.varValue)
print("Optimal dressers (y):", y.varValue)
print("Maximum profit:", pl.value(prob.objective))
```

This will give you the maximum profit and the optimal weekly quantities of chairs and dressers.

import pulp as pl

def main():
    # Define the problem
    prob = pl.LpProblem("Elm_Furniture_Profit_Max", pl.LpMaximize)

    # Decision variables
    x = pl.LpVariable('chairs', lowBound=0)
    y = pl.LpVariable('dressers', lowBound=0)

    # Objective function: Maximize 43x + 52y
    prob += 43 * x + 52 * y

    # Constraints
    prob += 1.4 * x + 1.1 * y <= 17   # Stain constraint
    prob += 2 * x + 3 * y <= 11       # Wood constraint

    # Solve the problem
    prob.solve()

    # Get the objective value
    obj_value = pl.value(prob.objective)

    # Print final objective value in required format
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
