[MODELER]:
Let’s turn this into an optimization model.

### 1. Decision variables

Let  
- \( x \in \mathbb{Z}_{\ge 0} \): number of helicopter trips  
- \( y \in \mathbb{Z}_{\ge 0} \): number of bus trips  

(They must be integers since you can’t make a fraction of a trip.)

### 2. Objective: Minimize total time

- Each helicopter trip takes 1 hour.  
- Each bus trip takes 3 hours.

Total time:
\[
\text{Total time} = 1x + 3y
\]

Objective:
\[
\min\; T = x + 3y
\]

### 3. Constraints

1. **Transport capacity (at least 120 patients):**

- Helicopter: 5 patients per trip  
- Bus: 8 patients per trip  

\[
5x + 8y \;\ge\; 120
\]

2. **At least 30% of trips by helicopter:**

Total trips = \( x + y \).  
Proportion of helicopter trips:

\[
\frac{x}{x + y} \;\ge\; 0.3
\]

Assuming \( x + y > 0 \), multiply both sides:

\[
x \;\ge\; 0.3(x + y) \\
x \;\ge\; 0.3x + 0.3y \\
0.7x \;\ge\; 0.3y \\
7x \;\ge\; 3y
\]

So:
\[
7x - 3y \;\ge\; 0
\]

3. **At most 10 bus trips:**

\[
y \;\le\; 10
\]

4. **Non-negativity and integrality:**

\[
x \in \mathbb{Z}_{\ge 0},\quad y \in \mathbb{Z}_{\ge 0}
\]

### 4. Full model

\[
\begin{aligned}
\min \quad & T = x + 3y \\
\text{s.t.} \quad &
5x + 8y \;\ge\; 120 \\
& 7x - 3y \;\ge\; 0 \\
& y \;\le\; 10 \\
& x, y \in \mathbb{Z}_{\ge 0}
\end{aligned}
\]

This is a **mixed-integer linear program** (MILP), but here both variables are integer, so it’s an integer linear program.

### 5. Suitable Python libraries

You can model and solve this with any MILP-capable library, for example:

- **PuLP** (`pulp`) – simple interface, pure Python, can use CBC (default), GLPK, or commercial solvers.
- **OR-Tools** (`ortools.linear_solver`) – very efficient, good for integer programming.
- **Pyomo** – more general algebraic modeling, supports many solvers.
- **SciPy**’s `scipy.optimize` is *not* ideal here because it does not directly support integer constraints.

A minimal PuLP setup would look like:

```python
import pulp as pl

# Problem
prob = pl.LpProblem("PatientTransport", pl.LpMinimize)

# Decision variables (integer)
x = pl.LpVariable('helicopter_trips', lowBound=0, cat='Integer')
y = pl.LpVariable('bus_trips',        lowBound=0, cat='Integer')

# Objective
prob += x + 3*y

# Constraints
prob += 5*x + 8*y >= 120          # capacity
prob += 7*x - 3*y >= 0            # ≥30% helicopter trips
prob += y <= 10                   # bus trip limit

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Helicopter trips:", x.value())
print("Bus trips:", y.value())
print("Min total time:", pl.value(prob.objective))
```

This implements exactly the mathematical model above.

[CODER]:
import pulp as pl

def main():
    # Create the problem
    prob = pl.LpProblem("PatientTransport", pl.LpMinimize)

    # Decision variables: non-negative integers
    x = pl.LpVariable('helicopter_trips', lowBound=0, cat='Integer')
    y = pl.LpVariable('bus_trips', lowBound=0, cat='Integer')

    # Objective: minimize total time
    prob += x + 3 * y

    # Constraints
    prob += 5 * x + 8 * y >= 120        # capacity
    prob += 7 * x - 3 * y >= 0          # at least 30% helicopter trips
    prob += y <= 10                     # at most 10 bus trips

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get objective value
    obj_value = pl.value(prob.objective)

    # Print final objective value in the required format
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
