Here is a clean optimization model for this problem.

## 1. Decision variables

Let

- \( x \in \mathbb{Z}_{\ge 0} \): number of truck trips  
- \( y \in \mathbb{Z}_{\ge 0} \): number of car trips  

(They must be integers because you can’t make fractional trips.)

## 2. Objective

Minimize total gas consumption:

\[
\text{Minimize } Z = 20x + 15y
\]

where:

- each truck trip uses 20 liters,
- each car trip uses 15 liters.

## 3. Constraints

1. **Package demand (at least 500 packages):**

Each truck carries 50 packages, each car carries 30:

\[
50x + 30y \;\ge\; 500
\]

2. **Truck trip limit (at most 5 trips):**

\[
x \;\le\; 5
\]

3. **At least 30% of all trips by car:**

\[
\frac{y}{x + y} \;\ge\; 0.30 \quad\text{for } x + y > 0
\]

Rearrange:

\[
y \;\ge\; 0.30(x + y)
\]
\[
y \;\ge\; 0.30x + 0.30y
\]
\[
0.70y \;\ge\; 0.30x
\]
\[
7y \;\ge\; 3x
\]

So the linear constraint is:

\[
7y - 3x \;\ge\; 0
\]

4. **Nonnegativity and integrality:**

\[
x, y \;\in\; \mathbb{Z}_{\ge 0}
\]

## 4. Full optimization model

\[
\begin{align}
\text{Minimize} \quad & Z = 20x + 15y \\
\text{subject to} \quad 
& 50x + 30y \ge 500 \\
& x \le 5 \\
& 7y - 3x \ge 0 \\
& x, y \in \mathbb{Z}_{\ge 0}
\end{align}
\]

This is an **integer linear programming (ILP)** problem with two integer variables.

## 5. Suitable Python libraries

You can solve this using any MILP-capable library, for example:

- `PuLP` (pure Python, easy to use; works with CBC by default)
- `OR-Tools` (`ortools.linear_solver` or `ortools.math_opt`)
- `Pyomo` (more general algebraic modeling, uses external solvers)
- `scipy.optimize` is **not ideal** here because it does not natively handle integer constraints; you’d need MILP support.

For a quick implementation, `PuLP` or OR-Tools are the most straightforward choices.

import pulp

def main():
    # Create the problem
    prob = pulp.LpProblem("Minimize_Gas_Consumption", pulp.LpMinimize)

    # Decision variables: x = truck trips, y = car trips
    x = pulp.LpVariable('x', lowBound=0, cat=pulp.LpInteger)
    y = pulp.LpVariable('y', lowBound=0, cat=pulp.LpInteger)

    # Objective function: Minimize 20x + 15y
    prob += 20 * x + 15 * y

    # Constraints
    # 1) Package demand: 50x + 30y >= 500
    prob += 50 * x + 30 * y >= 500

    # 2) Truck trip limit: x <= 5
    prob += x <= 5

    # 3) At least 30% of trips by car: 7y - 3x >= 0
    prob += 7 * y - 3 * x >= 0

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the objective value
    final_obj = pulp.value(prob.objective)

    # Print the final objective in the required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
