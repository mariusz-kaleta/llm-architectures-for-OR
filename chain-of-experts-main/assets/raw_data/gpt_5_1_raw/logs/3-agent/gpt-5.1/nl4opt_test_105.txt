Here is a formal optimization model of the problem.

## 1. Decision variables

Let  
- \(x\) = number of batches of hams produced (per year)  
- \(y\) = number of batches of pork ribs produced (per year)  

Both must be nonnegative:
\[
x \ge 0,\quad y \ge 0
\]

(If batches must be whole numbers, you can later enforce \(x, y \in \mathbb{Z}_{\ge 0}\).)

---

## 2. Objective function

Profit per batch:
- Hams: \$150 per batch  
- Pork ribs: \$300 per batch  

Total annual profit:
\[
\text{Maximize } Z = 150x + 300y
\]

---

## 3. Constraints

### Machine time constraints

Each machine can run at most 4000 hours per year.

**Meat slicer:**
- Hams use 4 hours per batch  
- Pork ribs use 2 hours per batch  

Total slicer hours used:
\[
4x + 2y \le 4000
\]

**Meat packer:**
- Hams use 2.5 hours per batch  
- Pork ribs use 3.5 hours per batch  

Total packer hours used:
\[
2.5x + 3.5y \le 4000
\]

---

## 4. Complete linear programming model

\[
\begin{aligned}
\text{Maximize} \quad & Z = 150x + 300y \\
\text{subject to} \quad
& 4x + 2y \le 4000 \quad \text{(slicer time)} \\
& 2.5x + 3.5y \le 4000 \quad \text{(packer time)} \\
& x \ge 0,\ y \ge 0.
\end{aligned}
\]

---

## 5. Solving the model (sketch)

You can solve this using various Python libraries for linear programming:

- `PuLP` (easy to use, pure Python)
- `scipy.optimize.linprog` (SciPy)
- `ortools.linear_solver` (Google OR-Tools)

### Example using PuLP

```python
import pulp as pl

# Define problem
prob = pl.LpProblem("Meat_Production", pl.LpMaximize)

# Decision variables (continuous; use cat="Integer" if needed)
x = pl.LpVariable("hams", lowBound=0)
y = pl.LpVariable("ribs", lowBound=0)

# Objective function
prob += 150 * x + 300 * y

# Constraints
prob += 4 * x + 2 * y <= 4000    # slicer
prob += 2.5 * x + 3.5 * y <= 4000  # packer

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Optimal hams (x):", x.value())
print("Optimal ribs (y):", y.value())
print("Maximum profit:", pl.value(prob.objective))
```

`scipy.optimize.linprog` and OR-Tools can model the same LP; PuLP is typically the most straightforward for this size and type of problem.

import pulp as pl

def main():
    # Define the LP problem: maximize profit
    prob = pl.LpProblem("Meat_Production", pl.LpMaximize)

    # Decision variables (continuous, non-negative)
    x = pl.LpVariable("hams", lowBound=0)
    y = pl.LpVariable("ribs", lowBound=0)

    # Objective function: Maximize 150x + 300y
    prob += 150 * x + 300 * y

    # Constraints
    prob += 4 * x + 2 * y <= 4000      # slicer time
    prob += 2.5 * x + 3.5 * y <= 4000  # packer time

    # Solve the LP
    prob.solve()

    # Get objective value
    final_obj = pl.value(prob.objective)

    # Print required final objective value
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
