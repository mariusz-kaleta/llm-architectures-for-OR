Here is a clean mathematical optimization model for this advertising problem.

---

## Decision variables

Let

- \(x_Z \ge 0\): number of z-tube ads  
- \(x_S \ge 0\): number of soorchle ads  
- \(x_W \ge 0\): number of wassa ads  

We will treat these as **integer** variables (you can’t buy a fraction of an ad).

---

## Objective: maximize total audience

Viewers per ad:

- z-tube: 400,000 viewers/ad  
- soorchle: 5,000 viewers/ad  
- wassa: 3,000 viewers/ad  

Total audience:
\[
\text{Audience} = 400{,}000\,x_Z + 5{,}000\,x_S + 3{,}000\,x_W
\]

**Objective function**:
\[
\max \quad 400{,}000\,x_Z + 5{,}000\,x_S + 3{,}000\,x_W
\]

---

## Constraints

### 1. Budget constraint

Costs:

- z-tube: \$1000/ad  
- soorchle: \$200/ad  
- wassa: \$100/ad  

Budget is \$10,000:

\[
1000\,x_Z + 200\,x_S + 100\,x_W \le 10{,}000
\]

---

### 2. Soorchle ad limit

Soorchle allows at most 15 ads:

\[
x_S \le 15
\]

---

### 3. Wassa at most one third of total ads

“At most a third of the total number of advertisements should occur on wassa”:

\[
x_W \le \frac{1}{3}(x_Z + x_S + x_W)
\]

Rewriting:

\[
3x_W \le x_Z + x_S + x_W \quad\Rightarrow\quad 2x_W \le x_Z + x_S
\]

So constraint becomes:
\[
2x_W \le x_Z + x_S
\]

---

### 4. z-tube at least 5% of total ads

“At least 5% should occur on z-tube”:

\[
x_Z \ge 0.05(x_Z + x_S + x_W)
\]

Rewriting:

\[
x_Z - 0.05x_Z \ge 0.05x_S + 0.05x_W
\]
\[
0.95x_Z \ge 0.05x_S + 0.05x_W
\]

Multiply both sides by 20 to clear decimals:

\[
19x_Z \ge x_S + x_W
\]

So constraint becomes:
\[
19x_Z \ge x_S + x_W
\]

---

### 5. Integrality and non-negativity

\[
x_Z, x_S, x_W \in \mathbb{Z}_{\ge 0}
\]

---

## Final model summary

**Variables**

- \(x_Z, x_S, x_W \in \mathbb{Z}_{\ge 0}\)

**Objective**

\[
\max \quad 400{,}000\,x_Z + 5{,}000\,x_S + 3{,}000\,x_W
\]

**Subject to**

\[
\begin{aligned}
1000\,x_Z + 200\,x_S + 100\,x_W &\le 10{,}000 &\text{(budget)}\\
x_S &\le 15 &\text{(soorchle cap)}\\
2x_W &\le x_Z + x_S &\text{(wassa ≤ one third)}\\
19x_Z &\ge x_S + x_W &\text{(z-tube ≥ 5%)}\\
x_Z, x_S, x_W &\in \mathbb{Z}_{\ge 0} &\text{(integrality)}
\end{aligned}
\]

## Suitable Python libraries

This is a small **integer linear programming (ILP)** problem. Good choices:

- **PuLP** (`pulp`): very straightforward modeling in pure Python; can use CBC (default solver).
- **OR-Tools** (`ortools.linear_solver`): efficient and well-maintained, good for ILP.
- **Pyomo**: more general modeling framework, suitable if you plan more complex models.

For a quick solution with minimal setup, I’d recommend **PuLP** or **OR-Tools**.

import pulp

def main():
    # Create the problem: maximize total audience
    prob = pulp.LpProblem("Advertising_Allocation", pulp.LpMaximize)

    # Decision variables (integers, >= 0)
    x_Z = pulp.LpVariable('x_Z', lowBound=0, cat='Integer')  # z-tube ads
    x_S = pulp.LpVariable('x_S', lowBound=0, cat='Integer')  # soorchle ads
    x_W = pulp.LpVariable('x_W', lowBound=0, cat='Integer')  # wassa ads

    # Objective function: maximize total audience
    prob += 400000 * x_Z + 5000 * x_S + 3000 * x_W

    # Constraints
    # 1. Budget: 1000 x_Z + 200 x_S + 100 x_W <= 10000
    prob += 1000 * x_Z + 200 * x_S + 100 * x_W <= 10000

    # 2. Soorchle cap: x_S <= 15
    prob += x_S <= 15

    # 3. Wassa at most one third of total ads: 2 x_W <= x_Z + x_S
    prob += 2 * x_W <= x_Z + x_S

    # 4. z-tube at least 5% of total ads: 19 x_Z >= x_S + x_W
    prob += 19 * x_Z >= x_S + x_W

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Extract final objective value
    final_obj = pulp.value(prob.objective)

    # Print final objective in required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
