Here is a clean mathematical model of the problem.

## 1. Decision variables

Let

- \(x_P \in \mathbb{Z}_{\ge 0}\): number of commercials on **Pi TV**  
- \(x_B \in \mathbb{Z}_{\ge 0}\): number of commercials on **Beta Video**  
- \(x_G \in \mathbb{Z}_{\ge 0}\): number of commercials on **Gamma Live**

All three are nonnegative integers.

Let
\[
T = x_P + x_B + x_G
\]
denote the total number of commercials.

## 2. Objective function

Maximize total audience reach:

- Pi TV: 2000 viewers per commercial  
- Beta Video: 5000 viewers per commercial  
- Gamma Live: 9000 viewers per commercial  

So the objective is:
\[
\max\; Z = 2000 x_P + 5000 x_B + 9000 x_G
\]

## 3. Constraints

### 3.1 Budget constraint

Weekly budget is \$20,000.

- Pi TV costs \$1200 per commercial  
- Beta Video costs \$2000 per commercial  
- Gamma Live costs \$4000 per commercial  

So:
\[
1200 x_P + 2000 x_B + 4000 x_G \le 20000
\]

### 3.2 Platform-specific constraints

1. **Beta Video cap**: at most 8 commercials:
   \[
   x_B \le 8
   \]

2. **Gamma Live at most one-third of all commercials**:
   \[
   x_G \le \frac{1}{3} T = \frac{1}{3}(x_P + x_B + x_G)
   \]
   Multiply both sides by 3:
   \[
   3x_G \le x_P + x_B + x_G
   \Rightarrow 2x_G \le x_P + x_B
   \]

3. **Pi TV at least 20% of all commercials**:
   \[
   x_P \ge 0.2 T = 0.2(x_P + x_B + x_G)
   \]
   Multiply both sides by 5:
   \[
   5x_P \ge x_P + x_B + x_G
   \Rightarrow 4x_P \ge x_B + x_G
   \]

### 3.3 Integrality and nonnegativity

\[
x_P, x_B, x_G \in \mathbb{Z}_{\ge 0}
\]

---

## 4. Summary model

Maximize
\[
Z = 2000 x_P + 5000 x_B + 9000 x_G
\]

subject to
\[
\begin{aligned}
1200 x_P + 2000 x_B + 4000 x_G &\le 20000 \\
x_B &\le 8 \\
2x_G &\le x_P + x_B \\
4x_P &\ge x_B + x_G \\
x_P, x_B, x_G &\in \mathbb{Z}_{\ge 0}
\end{aligned}
\]

## 5. Suitable Python libraries

This is a small **integer linear programming (ILP)** problem. Any standard MILP solver will handle it easily.

Recommended Python libraries:

- **PuLP**:  
  - Easy modeling of LP/MILP  
  - Integrates with CBC (open-source) and commercial solvers (Gurobi, CPLEX).
- **OR-Tools (Google OR-Tools)**:
  - Has a dedicated MIP solver (`pywraplp`) suitable here.
- **Pyomo**:
  - More general modeling framework; also works well with open-source and commercial solvers.

For a quick implementation, **PuLP** or **OR-Tools** would be the most straightforward.

import pulp

def main():
    # Create the problem: maximization
    prob = pulp.LpProblem("Ad_Allocation", pulp.LpMaximize)

    # Decision variables (nonnegative integers)
    x_P = pulp.LpVariable('x_P', lowBound=0, cat='Integer')  # Pi TV
    x_B = pulp.LpVariable('x_B', lowBound=0, cat='Integer')  # Beta Video
    x_G = pulp.LpVariable('x_G', lowBound=0, cat='Integer')  # Gamma Live

    # Objective function: Maximize total audience reach
    prob += 2000 * x_P + 5000 * x_B + 9000 * x_G

    # Constraints
    # Budget constraint
    prob += 1200 * x_P + 2000 * x_B + 4000 * x_G <= 20000

    # Beta Video cap
    prob += x_B <= 8

    # Gamma Live at most one-third of all commercials: 2x_G <= x_P + x_B
    prob += 2 * x_G <= x_P + x_B

    # Pi TV at least 20% of all commercials: 4x_P >= x_B + x_G
    prob += 4 * x_P >= x_B + x_G

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the optimal objective value
    final_obj = pulp.value(prob.objective)

    # Print the final objective in the required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
