[MODELER]:
Decision variables  
Let  
- \(x\) = acres of blueberries to plant  
- \(y\) = acres of raspberries to plant  

Both must be nonnegative:
\[
x \ge 0,\quad y \ge 0
\]

Objective function (maximize profit)  
Profit per acre:
- Blueberries: \$56  
- Raspberries: \$75  

Total profit:
\[
\text{Maximize } Z = 56x + 75y
\]

Constraints  

1. Land constraint (300 acres total):
\[
x + y \le 300
\]

2. Labor constraint (575 days available):  
Each acre of blueberries uses 6 days; each acre of raspberries uses 3 days:
\[
6x + 3y \le 575
\]

3. Watering budget constraint (\$10{,}000 available):  
Each acre of blueberries costs \$22; each acre of raspberries costs \$25:
\[
22x + 25y \le 10000
\]

LP model summary  
\[
\begin{aligned}
\text{Maximize} \quad & Z = 56x + 75y \\
\text{subject to} \quad 
& x + y \le 300 \\
& 6x + 3y \le 575 \\
& 22x + 25y \le 10000 \\
& x \ge 0,\; y \ge 0
\end{aligned}
\]

Suitable Python libraries to solve this LP  
- `PuLP` (very straightforward for linear programs)  
- `ortools.linear_solver` from Google OR-Tools  
- `scipy.optimize.linprog` (for continuous LPs like this)

[CODER]:
import pulp

# Create the LP problem: maximize profit
prob = pulp.LpProblem("Blueberry_Raspberry_Profit_Maximization", pulp.LpMaximize)

# Decision variables: acres of blueberries (x) and raspberries (y), both nonnegative
x = pulp.LpVariable('x', lowBound=0)
y = pulp.LpVariable('y', lowBound=0)

# Objective function: Maximize Z = 56x + 75y
prob += 56 * x + 75 * y, "Total_Profit"

# Constraints
# 1. Land constraint: x + y <= 300
prob += x + y <= 300, "Land_Constraint"

# 2. Labor constraint: 6x + 3y <= 575
prob += 6 * x + 3 * y <= 575, "Labor_Constraint"

# 3. Watering budget constraint: 22x + 25y <= 10000
prob += 22 * x + 25 * y <= 10000, "Water_Constraint"

# Solve the problem
prob.solve(pulp.PULP_CBC_CMD(msg=False))

# Get objective value
final_obj = pulp.value(prob.objective)

# Print final objective value in required format
print(f"FINAL_OBJ: {final_obj}")
