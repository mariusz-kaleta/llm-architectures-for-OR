Here is a clear mathematical optimization model for this problem.

### 1. Decision variables

Let  
- \(x \ge 0\): number of cups of drink A  
- \(y \ge 0\): number of cups of drink B  

(Both are continuous variables; the problem does not state they must be integers.)

### 2. Objective: Minimize total Vitamin K

Each cup of A has 4 units of K, each cup of B has 12 units of K.  
Total Vitamin K in the mix:

\[
\text{Vitamin K} = 4x + 12y
\]

**Objective function:**

\[
\min \; Z = 4x + 12y
\]

### 3. Constraints

#### Vitamin A requirement (at least 150 units)

A cup of A has 8 units, B has 15 units:

\[
8x + 15y \;\ge\; 150
\]

#### Vitamin D requirement (at least 300 units)

A cup of A has 6 units, B has 2 units:

\[
6x + 2y \;\ge\; 300
\]

#### Vitamin E limit (at most 400 units)

A cup of A has 10 units, B has 20 units:

\[
10x + 20y \;\le\; 400
\]

#### Nonnegativity

\[
x \;\ge\; 0,\quad y \;\ge\; 0
\]

### 4. Complete linear programming model

\[
\begin{aligned}
\min\ & Z = 4x + 12y \\
\text{s.t. } 
& 8x + 15y \ge 150 \quad (\text{Vitamin A})\\
& 6x + 2y \ge 300 \quad (\text{Vitamin D})\\
& 10x + 20y \le 400 \quad (\text{Vitamin E})\\
& x \ge 0,\ y \ge 0
\end{aligned}
\]

This is a standard linear programming (LP) problem.

---

### 5. Suitable Python libraries

To implement and solve this LP in Python, any of the following are appropriate:

- **PuLP** (`pulp`) – simple modeling interface, works well for small/medium LPs.
- **SciPy** (`scipy.optimize.linprog`) – good for basic LP without advanced modeling features.
- **OR-Tools Linear Solver** (`ortools.linear_solver`) – efficient and industrial-grade.

For a problem this size, **PuLP** or **SciPy’s `linprog`** is very convenient.

[CODER]:
import pulp

def main():
    # Create the LP problem: minimize Vitamin K
    prob = pulp.LpProblem("VitaminK_Minimization", pulp.LpMinimize)

    # Decision variables: x, y >= 0
    x = pulp.LpVariable('x', lowBound=0, cat='Continuous')
    y = pulp.LpVariable('y', lowBound=0, cat='Continuous')

    # Objective function: Minimize 4x + 12y
    prob += 4 * x + 12 * y, "Total_Vitamin_K"

    # Constraints
    # Vitamin A: 8x + 15y >= 150
    prob += 8 * x + 15 * y >= 150, "Vitamin_A"

    # Vitamin D: 6x + 2y >= 300
    prob += 6 * x + 2 * y >= 300, "Vitamin_D"

    # Vitamin E: 10x + 20y <= 400
    prob += 10 * x + 20 * y <= 400, "Vitamin_E"

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the objective value
    final_obj = pulp.value(prob.objective)

    # Print the final objective value in the required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
