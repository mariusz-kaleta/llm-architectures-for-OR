[MODELER]:
Define decision variables:

- \(x_1 \ge 0\): hours to run Factory 1  
- \(x_2 \ge 0\): hours to run Factory 2  

Factory production rates:

- Factory 1:  
  - Allergy pills: 20 per hour  
  - Fever-reducing pills: 15 per hour  
  - Rare compound: 20 units per hour  

- Factory 2:  
  - Allergy pills: 10 per hour  
  - Fever-reducing pills: 30 per hour  
  - Rare compound: 30 units per hour  

Requirements:

- At least 700 allergy pills  
- At least 600 fever-reducing pills  
- At most 1000 units of the rare compound  

---

### Constraints

1. Allergy pill requirement:
\[
20x_1 + 10x_2 \;\ge\; 700
\]

2. Fever-reducing pill requirement:
\[
15x_1 + 30x_2 \;\ge\; 600
\]

3. Rare compound availability:
\[
20x_1 + 30x_2 \;\le\; 1000
\]

4. Non-negativity:
\[
x_1 \ge 0,\quad x_2 \ge 0
\]

---

### Objective

Minimize total operating time of both factories:
\[
\min \; Z = x_1 + x_2
\]

This is a linear programming (LP) problem.

---

### Suitable Python libraries

You can model and solve this LP with any of:

- `PuLP` (simple LP modeling in Python)
- `scipy.optimize.linprog` (from SciPy)
- `ortools.linear_solver` (Google OR-Tools for LP/MIP)

For a clear algebraic model and straightforward syntax, `PuLP` or OR-Tools are good choices.

[PROGRAMMER-VALIDATOR]:
import pulp

def main():
    # Create the LP minimization problem
    prob = pulp.LpProblem("Factory_Operation_Time_Minimization", pulp.LpMinimize)

    # Decision variables: hours to run Factory 1 and Factory 2
    x1 = pulp.LpVariable('x1', lowBound=0)  # hours for Factory 1
    x2 = pulp.LpVariable('x2', lowBound=0)  # hours for Factory 2

    # Objective function: Minimize total operating time
    prob += x1 + x2, "Total_Operating_Time"

    # Constraints
    # Allergy pill requirement: 20x1 + 10x2 >= 700
    prob += 20 * x1 + 10 * x2 >= 700, "Allergy_Requirement"

    # Fever-reducing pill requirement: 15x1 + 30x2 >= 600
    prob += 15 * x1 + 30 * x2 >= 600, "Fever_Reducing_Requirement"

    # Rare compound availability: 20x1 + 30x2 <= 1000
    prob += 20 * x1 + 30 * x2 <= 1000, "Rare_Compound_Limit"

    # Solve the LP
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the objective value
    final_obj = pulp.value(prob.objective)

    # Print final objective value as required
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
