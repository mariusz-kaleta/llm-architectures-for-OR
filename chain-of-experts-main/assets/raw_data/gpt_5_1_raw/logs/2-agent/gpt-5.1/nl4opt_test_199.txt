[MODELER]:
Formulate this as an **integer linear programming (ILP)** problem.

### 1. Decision variables

Let  

- \(x_P\) = number of commercials on **Pi TV**  
- \(x_B\) = number of commercials on **Beta Video**  
- \(x_G\) = number of commercials on **Gamma Live**

All are nonnegative integers:  
\[
x_P, x_B, x_G \in \mathbb{Z}_{\ge 0}
\]

Let total number of commercials be:
\[
T = x_P + x_B + x_G
\]

### 2. Objective function (maximize total audience)

Each commercial yields:

- Pi TV: 2000 viewers  
- Beta Video: 5000 viewers  
- Gamma Live: 9000 viewers  

Maximize total viewers:
\[
\max\ Z = 2000x_P + 5000x_B + 9000x_G
\]

### 3. Constraints

#### a) Budget constraint

Weekly budget is \$20,000. Costs:

- Pi TV: \$1200 per commercial  
- Beta Video: \$2000 per commercial  
- Gamma Live: \$4000 per commercial  

So:
\[
1200x_P + 2000x_B + 4000x_G \le 20000
\]

#### b) Beta Video limit

At most 8 commercials on Beta Video:
\[
x_B \le 8
\]

#### c) Gamma Live at most one-third of all commercials

“at most a third of all commercials” on Gamma Live:
\[
x_G \le \frac{1}{3} T = \frac{1}{3}(x_P + x_B + x_G)
\]

Multiply both sides by 3:
\[
3x_G \le x_P + x_B + x_G
\]
\[
2x_G \le x_P + x_B
\]

This is a linear constraint:
\[
2x_G \le x_P + x_B
\]

#### d) Pi TV at least 20% of all commercials

“a minimum of 20%” on Pi TV:
\[
x_P \ge 0.2 T = 0.2(x_P + x_B + x_G)
\]

Multiply by 10 to clear decimals:
\[
10x_P \ge 2x_P + 2x_B + 2x_G
\]
\[
8x_P \ge 2x_B + 2x_G
\]
\[
4x_P \ge x_B + x_G
\]

So:
\[
4x_P \ge x_B + x_G
\]

#### e) Nonnegativity and integrality

\[
x_P, x_B, x_G \ge 0,\quad x_P, x_B, x_G \in \mathbb{Z}
\]

---

### Complete optimization model

**Maximize**
\[
Z = 2000x_P + 5000x_B + 9000x_G
\]

**Subject to**
\[
1200x_P + 2000x_B + 4000x_G \le 20000
\]
\[
x_B \le 8
\]
\[
2x_G \le x_P + x_B
\]
\[
4x_P \ge x_B + x_G
\]
\[
x_P, x_B, x_G \in \mathbb{Z}_{\ge 0}
\]

### Suitable Python libraries

To solve this integer linear program, you can use:

- **PuLP** (`pulp`): simple modeling interface, works with CBC (default), GLPK, etc.
- **OR-Tools** (`ortools.linear_solver` or `ortools.sat.python.cp_model`): very efficient for integer programs.
- **Pyomo**: more general modeling framework, connects to CBC, GLPK, Gurobi, CPLEX, etc.

For this small problem, **PuLP** or **OR-Tools** is ideal.

### Example solver skeleton in Python (PuLP)

```python
import pulp as pl

# Problem
prob = pl.LpProblem("Ad_Allocation", pl.LpMaximize)

# Variables
x_P = pl.LpVariable('x_P', lowBound=0, cat='Integer')
x_B = pl.LpVariable('x_B', lowBound=0, cat='Integer')
x_G = pl.LpVariable('x_G', lowBound=0, cat='Integer')

# Objective
prob += 2000*x_P + 5000*x_B + 9000*x_G

# Constraints
prob += 1200*x_P + 2000*x_B + 4000*x_G <= 20000     # Budget
prob += x_B <= 8                                     # Beta limit
prob += 2*x_G <= x_P + x_B                           # Gamma ≤ 1/3 of total
prob += 4*x_P >= x_B + x_G                           # Pi ≥ 20% of total

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Pi TV:", pl.value(x_P))
print("Beta Video:", pl.value(x_B))
print("Gamma Live:", pl.value(x_G))
print("Max viewers:", pl.value(prob.objective))
```

You can run this to get the exact optimal number of commercials on each platform.

import pulp as pl

def main():
    # Define the ILP problem
    prob = pl.LpProblem("Ad_Allocation", pl.LpMaximize)

    # Decision variables
    x_P = pl.LpVariable('x_P', lowBound=0, cat='Integer')  # Pi TV
    x_B = pl.LpVariable('x_B', lowBound=0, cat='Integer')  # Beta Video
    x_G = pl.LpVariable('x_G', lowBound=0, cat='Integer')  # Gamma Live

    # Objective function: maximize total viewers
    prob += 2000 * x_P + 5000 * x_B + 9000 * x_G

    # Constraints
    prob += 1200 * x_P + 2000 * x_B + 4000 * x_G <= 20000   # Budget constraint
    prob += x_B <= 8                                        # Beta Video limit
    prob += 2 * x_G <= x_P + x_B                            # Gamma ≤ 1/3 of total
    prob += 4 * x_P >= x_B + x_G                            # Pi ≥ 20% of total

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get objective value
    final_obj = pl.value(prob.objective)

    # Print solution details (not required, but useful for checking)
    print(f"x_P (Pi TV) = {pl.value(x_P)}")
    print(f"x_B (Beta Video) = {pl.value(x_B)}")
    print(f"x_G (Gamma Live) = {pl.value(x_G)}")
    print(f"Total viewers = {final_obj}")

    # Final required output
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
