[MODELER]:
We can model this as a linear programming (LP) problem.

### Decision variables
Let  
- \(x_1 \ge 0\): number of servings of syrup 1  
- \(x_2 \ge 0\): number of servings of syrup 2  

### Objective (minimize sugar)
Sugar per serving:
- Syrup 1: 0.5 units
- Syrup 2: 0.3 units  

Total sugar:
\[
\text{Sugar} = 0.5x_1 + 0.3x_2
\]

Objective:
\[
\min\; 0.5x_1 + 0.3x_2
\]

### Constraints

1. **Throat medicine at most 5 units**

Per serving:
- Syrup 1: 0.5 units to throat
- Syrup 2: 0.2 units to throat  

Total throat medicine:
\[
0.5x_1 + 0.2x_2 \le 5
\]

2. **Lung medicine at least 4 units**

Per serving:
- Syrup 1: 0.4 units to lungs
- Syrup 2: 0.5 units to lungs  

Total lung medicine:
\[
0.4x_1 + 0.5x_2 \ge 4
\]

3. **Non-negativity**
\[
x_1 \ge 0,\quad x_2 \ge 0
\]

### Full mathematical model

\[
\begin{aligned}
\min\quad & 0.5x_1 + 0.3x_2 \\
\text{s.t.}\quad
& 0.5x_1 + 0.2x_2 \le 5 \\
& 0.4x_1 + 0.5x_2 \ge 4 \\
& x_1, x_2 \ge 0
\end{aligned}
\]

---

### Solving strategy

This is a small linear program; you can solve it with:

- `PuLP` (very straightforward for LPs in Python)
- `scipy.optimize.linprog`
- Google OR-Tools’ linear solver (`pywraplp`)

#### Example with PuLP

```python
import pulp as pl

# Define problem
prob = pl.LpProblem("SoreThroatSyrup", pl.LpMinimize)

# Decision variables
x1 = pl.LpVariable('x1', lowBound=0)  # servings of syrup 1
x2 = pl.LpVariable('x2', lowBound=0)  # servings of syrup 2

# Objective: minimize sugar
prob += 0.5 * x1 + 0.3 * x2

# Constraints
prob += 0.5 * x1 + 0.2 * x2 <= 5      # throat constraint
prob += 0.4 * x1 + 0.5 * x2 >= 4      # lung constraint

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Status:", pl.LpStatus[prob.status])
print("Servings of syrup 1 (x1):", x1.value())
print("Servings of syrup 2 (x2):", x2.value())
print("Minimum sugar:", pl.value(prob.objective))
```

This will give you the optimal number of servings \(x_1\) and \(x_2\) that minimize sugar while meeting the throat and lung medicine constraints.

[PROGRAMMER-VALIDATOR]:
import pulp as pl

def main():
    # Define problem
    prob = pl.LpProblem("SoreThroatSyrup", pl.LpMinimize)

    # Decision variables
    x1 = pl.LpVariable('x1', lowBound=0)  # servings of syrup 1
    x2 = pl.LpVariable('x2', lowBound=0)  # servings of syrup 2

    # Objective: minimize sugar
    prob += 0.5 * x1 + 0.3 * x2

    # Constraints
    prob += 0.5 * x1 + 0.2 * x2 <= 5      # throat constraint
    prob += 0.4 * x1 + 0.5 * x2 >= 4      # lung constraint

    # Solve
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Extract objective value
    obj_value = pl.value(prob.objective)

    # Print final objective as required
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
