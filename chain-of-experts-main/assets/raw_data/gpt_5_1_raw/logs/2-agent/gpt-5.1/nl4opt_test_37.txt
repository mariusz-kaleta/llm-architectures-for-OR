[MODELER]:
We can model this as a linear programming (LP) problem.

## Decision variables

Let  
- \(x\) = number of thin jars produced per week  
- \(y\) = number of stubby jars produced per week  

We assume \(x, y \ge 0\) and can be real numbers (if integer jars are required, we can later impose integrality).

## Objective function

Maximize total weekly profit:

\[
\text{Maximize } Z = 5x + 9y
\]

where:
- $5 is profit per thin jar
- $9 is profit per stubby jar

## Constraints

1. **Shaping time constraint**

Each thin jar uses 50 minutes; each stubby jar uses 30 minutes. There are at most 3000 minutes per week:

\[
50x + 30y \le 3000
\]

2. **Baking time constraint**

Each thin jar uses 90 minutes; each stubby jar uses 150 minutes. There are at most 4000 minutes per week:

\[
90x + 150y \le 4000
\]

3. **Non-negativity**

\[
x \ge 0, \quad y \ge 0
\]

## Solving the model (outline)

You can solve this LP using several Python libraries, for example:

- `pulp` (simple and educational, great for small LPs)
- `scipy.optimize.linprog` (scientific computing context)
- `ortools.linear_solver` from Google OR-Tools (industrial-strength)

### Example using PuLP

```python
import pulp as pl

# Define problem
prob = pl.LpProblem("Jar_Production", pl.LpMaximize)

# Decision variables (continuous, >= 0)
x = pl.LpVariable('thin_jars', lowBound=0)
y = pl.LpVariable('stubby_jars', lowBound=0)

# Objective function
prob += 5 * x + 9 * y

# Constraints
prob += 50 * x + 30 * y <= 3000      # Shaping time
prob += 90 * x + 150 * y <= 4000     # Baking time

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Thin jars (x):", pl.value(x))
print("Stubby jars (y):", pl.value(y))
print("Maximum profit:", pl.value(prob.objective))
```

This solves the mathematical model and returns the optimal numbers of thin and stubby jars.

[PROGRAMMER-VALIDATOR]:
import pulp as pl

def main():
    # Define problem
    prob = pl.LpProblem("Jar_Production", pl.LpMaximize)

    # Decision variables (continuous, >= 0)
    x = pl.LpVariable('thin_jars', lowBound=0)
    y = pl.LpVariable('stubby_jars', lowBound=0)

    # Objective function: Maximize 5x + 9y
    prob += 5 * x + 9 * y

    # Constraints
    prob += 50 * x + 30 * y <= 3000      # Shaping time
    prob += 90 * x + 150 * y <= 4000     # Baking time

    # Solve
    prob.solve()

    # Get objective value
    max_profit = pl.value(prob.objective)

    # Required final print
    print(f"FINAL_OBJ: {max_profit}")

if __name__ == "__main__":
    main()
