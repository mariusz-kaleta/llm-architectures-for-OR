[MODELER]:
Here is a clear linear programming (LP) formulation of the problem.

## Decision variables

Let  
- \(x_1 \ge 0\): number of hours per day to run Factory 1  
- \(x_2 \ge 0\): number of hours per day to run Factory 2  

## Objective

Minimize the total daily production cost:

\[
\min \; Z = 300x_1 + 600x_2
\]

## Constraints

Bears produced by each factory per hour:

- Factory 1:  
  - 5 black, 6 white, 3 brown
- Factory 2:  
  - 10 black, 10 white, 0 brown

Demand requirements:

1. Black bears (at least 20):

\[
5x_1 + 10x_2 \;\ge\; 20
\]

2. White bears (at least 5):

\[
6x_1 + 10x_2 \;\ge\; 5
\]

3. Brown bears (at least 15):

\[
3x_1 \;\ge\; 15
\]

Non-negativity:

\[
x_1 \ge 0,\quad x_2 \ge 0
\]

That fully defines the linear programming model.

## Suitable Python libraries

You can model and solve this LP using any of these:

- **PuLP** (`pulp`): very straightforward for educational LPs.
- **SciPy**: `scipy.optimize.linprog` for basic linear programs.
- **OR-Tools** (`ortools.linear_solver.pywraplp`): industrial-strength solver, also simple for LPs.

For this problem, **PuLP** or **OR-Tools** are particularly convenient.

[PROGRAMMER-VALIDATOR]:
import pulp

# Create LP problem: minimize cost
prob = pulp.LpProblem("Bear_Production", pulp.LpMinimize)

# Decision variables: hours of operation for each factory (non-negative)
x1 = pulp.LpVariable('x1', lowBound=0)
x2 = pulp.LpVariable('x2', lowBound=0)

# Objective function: Minimize total daily cost
prob += 300 * x1 + 600 * x2

# Constraints
# Black bears: 5x1 + 10x2 >= 20
prob += 5 * x1 + 10 * x2 >= 20

# White bears: 6x1 + 10x2 >= 5
prob += 6 * x1 + 10 * x2 >= 5

# Brown bears: 3x1 >= 15
prob += 3 * x1 >= 15

# Solve the problem
prob.solve(pulp.PULP_CBC_CMD(msg=False))

# Extract objective value
final_obj = pulp.value(prob.objective)

# Print final objective value in required format
print(f"FINAL_OBJ: {final_obj}")
