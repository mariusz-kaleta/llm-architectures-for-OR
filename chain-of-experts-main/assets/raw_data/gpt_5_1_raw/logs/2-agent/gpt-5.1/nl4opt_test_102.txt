[MODELER]:
**1. Decision variables**

Let  
- \(x \ge 0\): number of servings of Rice per day  
- \(y \ge 0\): number of servings of Kebab per day  

(both can be continuous if fractional servings are allowed).

---

**2. Objective: minimize daily cost**

Rice costs \$3 per serving, Kebab costs \$2 per serving:

\[
\min Z = 3x + 2y
\]

---

**3. Constraints**

- **Calorie requirement** (at least 2200 calories):

  Each serving of Rice has 300 calories, each Kebab has 200 calories:

  \[
  300x + 200y \;\ge\; 2200
  \]

- **Protein requirement** (at least 30 grams):

  Each serving of Rice has 4.5 g protein, each Kebab has 4 g protein:

  \[
  4.5x + 4y \;\ge\; 30
  \]

- **Non-negativity**:

  \[
  x \ge 0,\quad y \ge 0
  \]

---

**4. Complete linear programming formulation**

\[
\begin{aligned}
\min\;& Z = 3x + 2y \\
\text{s.t. } 
& 300x + 200y \ge 2200 \\
& 4.5x + 4y \ge 30 \\
& x \ge 0,\; y \ge 0
\end{aligned}
\]

---

**5. Suitable Python libraries**

To solve this LP numerically in Python, you can use:

- **PuLP** (`pulp`) – very straightforward for educational LP models.
- **OR-Tools Linear Solver** (`ortools.linear_solver.pywraplp`) – efficient and well maintained.
- **SciPy** (`scipy.optimize.linprog`) – good for continuous LPs, including this one.

All three can solve this model; for clarity and simplicity, PuLP is often the easiest to start with.

---

[PROGRAMMER-VALIDATOR]:
import pulp

# Create LP problem: minimize cost
prob = pulp.LpProblem("DietProblem", pulp.LpMinimize)

# Decision variables: servings of Rice (x) and Kebab (y)
x = pulp.LpVariable('x', lowBound=0, cat='Continuous')
y = pulp.LpVariable('y', lowBound=0, cat='Continuous')

# Objective function: Minimize 3x + 2y
prob += 3 * x + 2 * y

# Constraints
# Calorie requirement: 300x + 200y >= 2200
prob += 300 * x + 200 * y >= 2200

# Protein requirement: 4.5x + 4y >= 30
prob += 4.5 * x + 4 * y >= 30

# Solve the LP
prob.solve(pulp.PULP_CBC_CMD(msg=False))

# Get the objective value
final_obj = pulp.value(prob.objective)

print(f"FINAL_OBJ: {final_obj}")
