Let’s convert the description into a math optimization model.

## 1. Decision variables

Let  
- \(x\) = number of wraps to make  
- \(y\) = number of platters to make  

We assume \(x, y \ge 0\) and typically integer, since you can’t make fractional wraps or platters.

## 2. Objective function

Each wrap takes 10 minutes; each platter takes 8 minutes.

Minimize total production time:
\[
\min T = 10x + 8y
\]

## 3. Constraints

### Meat usage

- Wrap: 5 units of meat  
- Platter: 7 units of meat  
- Must use at least 3000 units of meat:

\[
5x + 7y \;\ge\; 3000
\]

### Rice usage

- Wrap: 3 units of rice  
- Platter: 5 units of rice  
- Must use at least 2500 units of rice:

\[
3x + 5y \;\ge\; 2500
\]

### Wraps vs platters

“At least 3 times as many wraps as platters”:

\[
x \;\ge\; 3y
\]

### Non-negativity (and integrality if enforced)

\[
x \ge 0,\quad y \ge 0
\]
Optionally: \(x, y \in \mathbb{Z}_{\ge 0}\).

---

## 4. Full model

\[
\begin{aligned}
\min\;&10x + 8y \\
\text{s.t. }& 5x + 7y \ge 3000 \\
            & 3x + 5y \ge 2500 \\
            & x - 3y \ge 0 \\
            & x, y \ge 0\;(\text{and usually integer})
\end{aligned}
\]

This is a linear programming / integer linear programming problem.

---

## 5. Suitable Python libraries

You can solve this with standard LP/MIP libraries such as:

- **PuLP** (`pulp`): simple interface to various LP/MIP solvers.
- **OR-Tools** (`ortools.linear_solver`): from Google, good for linear and mixed-integer problems.
- **SciPy** (`scipy.optimize.linprog`): for continuous LP (if you relax integrality).
- **Pyomo**: more general algebraic modeling language for optimization.

For a small problem like this, **PuLP** or **OR-Tools** is ideal.

---

import pulp

def main():
    # Create the optimization problem: minimize total production time
    prob = pulp.LpProblem("Wraps_and_Platters_Time_Minimization", pulp.LpMinimize)

    # Decision variables (non-negative integers)
    x = pulp.LpVariable('x', lowBound=0, cat='Integer')  # number of wraps
    y = pulp.LpVariable('y', lowBound=0, cat='Integer')  # number of platters

    # Objective function: Minimize total time 10x + 8y
    prob += 10 * x + 8 * y, "Total_Production_Time"

    # Constraints
    # Meat usage: 5x + 7y >= 3000
    prob += 5 * x + 7 * y >= 3000, "Meat_Constraint"

    # Rice usage: 3x + 5y >= 2500
    prob += 3 * x + 5 * y >= 2500, "Rice_Constraint"

    # Wraps vs platters: x >= 3y
    prob += x - 3 * y >= 0, "Wraps_vs_Platters_Constraint"

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the objective value
    final_obj = pulp.value(prob.objective)

    # Print the final objective value as required
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
