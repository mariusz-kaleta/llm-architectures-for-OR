[MODELER]:
Define decision variables:

- \(x\): number of servings of noodles per day  
- \(y\): number of servings of protein bars per day  

Both must be nonnegative:
\[
x \ge 0,\quad y \ge 0
\]

Objective: minimize total daily cost

- Noodles cost: \$5 per serving → total cost \(5x\)  
- Protein bars cost: \$2.5 per serving → total cost \(2.5y\)

So the objective function is:
\[
\min \; Z = 5x + 2.5y
\]

Constraints:

1. Calorie requirement (at least 2000 calories):

- Noodles: 600 calories per serving → \(600x\) calories  
- Protein bars: 250 calories per serving → \(250y\) calories  

Total calories ≥ 2000:
\[
600x + 250y \ge 2000
\]

2. Protein requirement (at least 16 grams):

- Noodles: 1.5 g protein per serving → \(1.5x\) grams  
- Protein bars: 5 g protein per serving → \(5y\) grams  

Total protein ≥ 16 grams:
\[
1.5x + 5y \ge 16
\]

Complete linear programming model:

\[
\begin{align*}
\min \quad & Z = 5x + 2.5y \\
\text{s.t.} \quad 
& 600x + 250y \ge 2000 \\
& 1.5x + 5y \ge 16 \\
& x \ge 0,\; y \ge 0
\end{align*}
\]

Suitable Python libraries to solve this LP:

- `PuLP` (simple and user-friendly for teaching and small LPs)
- `scipy.optimize.linprog` (part of SciPy, good for continuous LPs)
- `ortools.linear_solver` from Google OR-Tools (powerful and scalable)

[PROGRAMMER-VALIDATOR]:
import pulp

# Create the LP problem: minimize cost
prob = pulp.LpProblem("DietProblem", pulp.LpMinimize)

# Decision variables (continuous, nonnegative)
x = pulp.LpVariable('x', lowBound=0, cat='Continuous')  # servings of noodles
y = pulp.LpVariable('y', lowBound=0, cat='Continuous')  # servings of protein bars

# Objective function: Minimize total daily cost
prob += 5 * x + 2.5 * y, "TotalCost"

# Constraints

# 1. Calorie requirement: 600x + 250y >= 2000
prob += 600 * x + 250 * y >= 2000, "CalorieRequirement"

# 2. Protein requirement: 1.5x + 5y >= 16
prob += 1.5 * x + 5 * y >= 16, "ProteinRequirement"

# Solve the problem
prob.solve(pulp.PULP_CBC_CMD(msg=False))

# Get the objective value
final_obj = pulp.value(prob.objective)

# Print the final objective value in the required format
print(f"FINAL_OBJ: {final_obj}")
