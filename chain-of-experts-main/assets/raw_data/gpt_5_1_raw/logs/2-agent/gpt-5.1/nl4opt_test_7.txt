[MODELER]:
Here is a clean mathematical model of the problem as a linear program.

### Decision variables
Let  
- \(x\) = number of chairs produced per week  
- \(y\) = number of dressers produced per week  

We assume \(x, y \ge 0\) and can be real-valued (if only whole items make sense, we add integer constraints later).

### Objective
Maximize total weekly profit:
\[
\max \; Z = 43x + 52y
\]

### Constraints

1. **Stain availability (gallons):**  
   Each chair uses 1.4 gallons and each dresser uses 1.1 gallons. At most 17 gallons available:
   \[
   1.4x + 1.1y \le 17
   \]

2. **Oak wood availability (lengths):**  
   Each chair uses 2 lengths and each dresser uses 3 lengths. At most 11 lengths available:
   \[
   2x + 3y \le 11
   \]

3. **Non-negativity:**
   \[
   x \ge 0,\quad y \ge 0
   \]

If we require whole numbers of items:
\[
x, y \in \mathbb{Z}_{\ge 0}
\]

---

### Solving outline and library suggestions

This is a small linear program (and potentially an integer linear program). Good Python libraries:

- **PuLP** (`pulp`) – simple, educational, great for small LP/ILP models.  
- **OR-Tools Linear Solver** (`ortools.linear_solver`) – efficient, supports LP and MIP.  
- **SciPy** (`scipy.optimize.linprog`) – good for continuous LP (without integrality requirements).

#### Example in PuLP (continuous version)

```python
import pulp as pl

# Create the problem
prob = pl.LpProblem("Elm_Furniture_Profit_Max", pl.LpMaximize)

# Decision variables
x = pl.LpVariable('Chairs', lowBound=0)
y = pl.LpVariable('Dressers', lowBound=0)

# Objective
prob += 43*x + 52*y

# Constraints
prob += 1.4*x + 1.1*y <= 17    # stain constraint
prob += 2*x + 3*y <= 11        # wood constraint

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Chairs:", x.value())
print("Dressers:", y.value())
print("Max profit:", pl.value(prob.objective))
```

To require integer numbers of items, change the variable definitions:

```python
x = pl.LpVariable('Chairs', lowBound=0, cat='Integer')
y = pl.LpVariable('Dressers', lowBound=0, cat='Integer')
```

import pulp as pl

# Create the problem (maximize profit)
prob = pl.LpProblem("Elm_Furniture_Profit_Max", pl.LpMaximize)

# Decision variables (integer, non-negative)
x = pl.LpVariable('Chairs', lowBound=0, cat='Integer')
y = pl.LpVariable('Dressers', lowBound=0, cat='Integer')

# Objective function
prob += 43 * x + 52 * y

# Constraints
prob += 1.4 * x + 1.1 * y <= 17    # Stain availability
prob += 2 * x + 3 * y <= 11        # Oak wood availability

# Solve the problem
prob.solve()

# Get the objective value
max_profit = pl.value(prob.objective)

# Print the required final objective value
print(f"FINAL_OBJ: {max_profit}")
