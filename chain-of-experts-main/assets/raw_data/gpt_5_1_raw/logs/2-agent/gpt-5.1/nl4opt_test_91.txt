[MODELER]:
Here is a formal optimization model of the problem.

## 1. Decision variables

Let  
- \(x\) = number of turkey dinners  
- \(y\) = number of tuna salad sandwiches  

Assume \(x, y \ge 0\) and can be real numbers (if they must be whole meals, add integer constraints).

## 2. Objective function (minimize total fat)

Total fat (in grams):

- Turkey dinner: 12 g fat/meal → \(12x\)
- Tuna sandwich: 8 g fat/meal → \(8y\)

Minimize:
\[
\min\; F = 12x + 8y
\]

## 3. Constraints

### Protein requirement (at least 150 g)

- Turkey: 20 g protein/meal
- Tuna: 18 g protein/meal

\[
20x + 18y \;\ge\; 150
\]

### Carbohydrate requirement (at least 200 g)

- Turkey: 30 g carbs/meal
- Tuna: 25 g carbs/meal

\[
30x + 25y \;\ge\; 200
\]

### At most 40% of meals are turkey dinners

Let total meals be \(x + y\). At most 40% turkey:

\[
\frac{x}{x+y} \;\le\; 0.4 \quad\text{with } x + y > 0
\]

This can be converted to a linear constraint:

\[
x \le 0.4(x + y)
\]

\[
x \le 0.4x + 0.4y
\]

\[
0.6x \le 0.4y
\]

\[
3x \le 2y
\]

Thus:

\[
3x - 2y \le 0
\]

### Nonnegativity

\[
x \ge 0,\quad y \ge 0
\]

(Optionally: \(x, y \in \mathbb{Z}\) if only whole meals are allowed.)

---

## 4. Complete linear program

\[
\begin{aligned}
\min\quad & 12x + 8y \\
\text{s.t.}\quad 
& 20x + 18y \ge 150 \\
& 30x + 25y \ge 200 \\
& 3x - 2y \le 0 \\
& x \ge 0,\; y \ge 0
\end{aligned}
\]

If integer meals are required, add \(x, y \in \mathbb{Z}\).

---

## 5. Suitable Python libraries

This is a linear programming (LP) problem, or a mixed-integer LP if you require whole meals. Appropriate Python libraries:

- **PuLP** (`pulp`) – simple, pure Python LP/MIP modeling.
- **OR-Tools** (`ortools.linear_solver`) – efficient LP/MIP solver by Google.
- **SciPy** (`scipy.optimize.linprog`) – good for continuous LP (no integer constraints).

Example tool selection:

- If *fractional meals are allowed*:  
  Use `scipy.optimize.linprog` or PuLP with a continuous LP.
- If *only whole meals are allowed*:  
  Use PuLP or OR-Tools with integer variables.

---

[PROGRAMMER-VALIDATOR]:
import numpy as np
from scipy.optimize import linprog

# Coefficients for objective: minimize 12x + 8y
c = np.array([12, 8])

# Inequality constraints in form A_ub * x <= b_ub
# Given:
# 20x + 18y >= 150  -> -20x -18y <= -150
# 30x + 25y >= 200  -> -30x -25y <= -200
# 3x - 2y <= 0      ->  3x -2y <= 0
A_ub = np.array([
    [-20, -18],
    [-30, -25],
    [3, -2]
])
b_ub = np.array([
    -150,
    -200,
    0
])

# Bounds: x >= 0, y >= 0
bounds = [(0, None), (0, None)]

# Solve LP
res = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method="highs")

if not res.success:
    raise RuntimeError("Optimization failed: " + res.message)

final_obj = res.fun

print(f"FINAL_OBJ: {final_obj}")
