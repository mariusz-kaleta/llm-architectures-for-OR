[MODELER]:
Here is a linear programming (LP) model of the problem.

### Decision variables
Let  
- \(x \ge 0\): kilograms of **regular mix** to produce  
- \(y \ge 0\): kilograms of **sour surprise mix** to produce  

### Objective
Maximize total profit:

\[
\max Z = 3x + 5y
\]

where  
- \(3\) = profit per kg of regular mix  
- \(5\) = profit per kg of sour surprise mix  

### Constraints

**1. Regular candy availability**

Each kg of:
- regular mix uses \(0.8\) kg of regular candy  
- sour surprise mix uses \(0.1\) kg of regular candy  

Total regular candy available: 80 kg.

\[
0.8x + 0.1y \le 80
\]

**2. Sour candy availability**

Each kg of:
- regular mix uses \(0.2\) kg of sour candy  
- sour surprise mix uses \(0.9\) kg of sour candy  

Total sour candy available: 60 kg.

\[
0.2x + 0.9y \le 60
\]

**3. Non-negativity**

\[
x \ge 0,\quad y \ge 0
\]

---

### Solving the model (outline)

This is a small linear program; it can be solved with:

- `PuLP` (pure Python LP modeling)
- `scipy.optimize.linprog`
- `ortools.linear_solver` from Google OR-Tools

Example with PuLP:

```python
import pulp as pl

# Define problem
prob = pl.LpProblem("CandyMix", pl.LpMaximize)

# Decision variables
x = pl.LpVariable('x', lowBound=0)  # regular mix (kg)
y = pl.LpVariable('y', lowBound=0)  # sour surprise mix (kg)

# Objective
prob += 3*x + 5*y

# Constraints
prob += 0.8*x + 0.1*y <= 80   # regular candy
prob += 0.2*x + 0.9*y <= 60   # sour candy

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Regular mix (kg):", x.value())
print("Sour surprise mix (kg):", y.value())
print("Max profit:", pl.value(prob.objective))
```

[PROGRAMMER-VALIDATOR]:
import pulp as pl

def main():
    # Define problem
    prob = pl.LpProblem("CandyMix", pl.LpMaximize)

    # Decision variables
    x = pl.LpVariable('x', lowBound=0)  # regular mix (kg)
    y = pl.LpVariable('y', lowBound=0)  # sour surprise mix (kg)

    # Objective: Maximize 3x + 5y
    prob += 3 * x + 5 * y

    # Constraints
    prob += 0.8 * x + 0.1 * y <= 80   # regular candy availability
    prob += 0.2 * x + 0.9 * y <= 60   # sour candy availability

    # Solve
    prob.solve()

    max_profit = pl.value(prob.objective)

    # Final required output
    print(f"FINAL_OBJ: {max_profit}")

if __name__ == "__main__":
    main()
