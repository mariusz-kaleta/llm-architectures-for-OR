[MODELER]:
Here’s a clean optimization model of the problem.

---

## 1. Decision variables

Let

- \(v \in \mathbb{Z}_{\ge 0}\): number of van trips  
- \(t \in \mathbb{Z}_{\ge 0}\): number of truck trips  

Both are nonnegative integers.

---

## 2. Objective

Minimize the total number of trips:

\[
\min\; Z = v + t
\]

---

## 3. Constraints

### a) Capacity constraint (at least 1500 boxes)

- Each van trip carries 50 boxes.
- Each truck trip carries 80 boxes.
- At least 1500 boxes must be transported:

\[
50v + 80t \;\ge\; 1500
\]

---

### b) Budget constraint (at most \$1000)

- Each van trip costs \$30.
- Each truck trip costs \$50.
- Total cost must be within budget:

\[
30v + 50t \;\le\; 1000
\]

---

### c) Advertising constraint (more van trips than truck trips)

\[
v \;>\; t
\]

Because we are dealing with integers, this can be implemented as:

\[
v \;\ge\; t + 1
\]

---

### d) Integrality and nonnegativity

\[
v, t \in \mathbb{Z}, \quad v \ge 0,\; t \ge 0
\]

---

## 4. Full model summary

\[
\begin{aligned}
\min\quad & v + t \\
\text{s.t.}\quad 
& 50v + 80t \ge 1500 \\
& 30v + 50t \le 1000 \\
& v \ge t + 1 \\
& v, t \in \mathbb{Z}_{\ge 0}
\end{aligned}
\]

This is a small **integer linear program (ILP)**.

---

import itertools

def is_feasible(v, t):
    if v < 0 or t < 0:
        return False
    if 50 * v + 80 * t < 1500:
        return False
    if 30 * v + 50 * t > 1000:
        return False
    if v < t + 1:
        return False
    return True

best_val = None
best_solution = None

# Bounds: from constraints
# From cost: 30v + 50t <= 1000 ⇒ v <= 1000/30, t <= 1000/50
max_v = 1000 // 30 + 1
max_t = 1000 // 50 + 1

for v in range(max_v + 1):
    for t in range(max_t + 1):
        if is_feasible(v, t):
            obj = v + t
            if best_val is None or obj < best_val:
                best_val = obj
                best_solution = (v, t)

# If no feasible solution found, set objective to None or similar
if best_val is None:
    best_val = float('inf')

print(f"FINAL_OBJ: {best_val}")
