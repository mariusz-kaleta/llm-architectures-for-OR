## 1. Decision variables

Let

- \( S \in \mathbb{Z}_{\ge 0} \): number of small suitcases to send  
- \( L \in \mathbb{Z}_{\ge 0} \): number of large suitcases to send  

---

## 2. Objective function

Each small suitcase carries 50 snacks and each large suitcase carries 80 snacks.  
We want to maximise the total number of snacks shipped:

\[
\max \; Z = 50S + 80L
\]

---

## 3. Constraints

1. **Preference for small suitcases**  
   At least twice as many small as large:

   \[
   S \;\ge\; 2L
   \]

2. **Availability of small suitcases**  

   \[
   S \;\le\; 70
   \]

3. **Availability of large suitcases**  

   \[
   L \;\le\; 50
   \]

4. **Minimum number of large suitcases**

   \[
   L \;\ge\; 15
   \]

5. **Total number of suitcases limit**  

   \[
   S + L \;\le\; 70
   \]

6. **Integrality and non‑negativity**

   \[
   S, L \in \mathbb{Z}, \quad S \ge 0, \; L \ge 0
   \]

---

## 4. Model summary (standard ILP form)

\[
\begin{aligned}
\max\quad & 50S + 80L \\
\text{s.t.}\quad
& S - 2L \;\ge\; 0 \\
& S \;\le\; 70 \\
& L \;\le\; 50 \\
& L \;\ge\; 15 \\
& S + L \;\le\; 70 \\
& S, L \in \mathbb{Z}_{\ge 0}
\end{aligned}

## 5. Suitable Python libraries and a sketch of usage

For this 2-variable ILP, many libraries will work; appropriate choices include:

- `PuLP` (pure Python linear/integer programming modeling)
- `OR-Tools` (Google’s optimisation suite, `pywraplp` for integer programming)
- `Pyomo` (more general algebraic modeling, uses external solvers)
- `scipy.optimize.linprog` (but this handles continuous, not integer, so not ideal here unless you relax integrality)

### Example with PuLP

```python
import pulp as pl

# Create problem
prob = pl.LpProblem("Snack_Suitcases", pl.LpMaximize)

# Decision variables (integer, non-negative)
S = pl.LpVariable('SmallSuitcases', lowBound=0, cat='Integer')
L = pl.LpVariable('LargeSuitcases', lowBound=0, cat='Integer')

# Objective
prob += 50 * S + 80 * L

# Constraints
prob += S >= 2 * L           # Preference for small
prob += S <= 70              # Small availability
prob += L <= 50              # Large availability
prob += L >= 15              # Minimum large
prob += S + L <= 70          # Total suitcase limit

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=0))

print("Status:", pl.LpStatus[prob.status])
print("Optimal S (small):", pl.value(S))
print("Optimal L (large):", pl.value(L))
print("Max snacks:", pl.value(50 * S + 80 * L))
```

### Example with OR-Tools

```python
from ortools.linear_solver import pywraplp

solver = pywraplp.Solver.CreateSolver('CBC')

S = solver.IntVar(0, solver.infinity(), 'SmallSuitcases')
L = solver.IntVar(0, solver.infinity(), 'LargeSuitcases')

# Objective
solver.Maximize(50 * S + 80 * L)

# Constraints
solver.Add(S >= 2 * L)
solver.Add(S <= 70)
solver.Add(L <= 50)
solver.Add(L >= 15)
solver.Add(S + L <= 70)

status = solver.Solve()

if status == pywraplp.Solver.OPTIMAL:
    print("Optimal S:", S.solution_value())
    print("Optimal L:", L.solution_value())
    print("Max snacks:", 50 * S.solution_value() + 80 * L.solution_value())
```

import pulp as pl

def main():
    # Create the problem
    prob = pl.LpProblem("Snack_Suitcases", pl.LpMaximize)

    # Decision variables: integers, non-negative
    S = pl.LpVariable('SmallSuitcases', lowBound=0, cat='Integer')
    L = pl.LpVariable('LargeSuitcases', lowBound=0, cat='Integer')

    # Objective function: maximize total snacks
    prob += 50 * S + 80 * L

    # Constraints
    prob += S - 2 * L >= 0        # Preference for small: S >= 2L
    prob += S <= 70               # Small availability
    prob += L <= 50               # Large availability
    prob += L >= 15               # Minimum large
    prob += S + L <= 70           # Total suitcase limit

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=0))

    # Get the objective value
    value = pl.value(50 * S + 80 * L)

    # Required final print
    print(f"FINAL_OBJ: {value}")

if __name__ == "__main__":
    main()
