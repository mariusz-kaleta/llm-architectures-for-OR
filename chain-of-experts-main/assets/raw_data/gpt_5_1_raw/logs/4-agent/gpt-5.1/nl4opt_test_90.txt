   - Premium models: daily production cannot exceed maximum daily demand for premiums (≤ 6).

   b) Total production capacity  
   - Combined production of regular and premium models is limited to at most 12 cars per day.  
   - This links x1 and x2 jointly.

   c) Non-negativity  
   - x1 ≥ 0 (cannot produce a negative number of regular cars).  
   - x2 ≥ 0 (cannot produce a negative number of premium cars).

6. Modelling / solution strategy for the Modeller

     - Individual upper bounds: x1 ≤ 8, x2 ≤ 6.

     - Either solve graphically by plotting the constraints and identifying the feasible polygon,  
     - Or solve analytically by evaluating the objective at each vertex (corner point) of the feasible region.

   - Candidate corner points to test will be formed from intersections of:
     - x1 = 0, x2 = 0, x1 = 8, x2 = 6, and x1 + x2 = 12,

   - Compute profit at each feasible corner point and choose the combination (x1, x2) that yields the highest profit.

[MODELER]:
Here is a clean mathematical model of the problem and how to solve it.

---

## 1. Decision variables

Let

- \(x_1 \ge 0\): number of regular cars produced per day  
- \(x_2 \ge 0\): number of premium cars produced per day  

Both variables are continuous and nonnegative (if the problem requires whole cars only, we additionally impose integrality, \(x_1, x_2 \in \mathbb{Z}_{\ge 0}\), but for this LP the fractional optimum is already integral).

---

## 2. Objective function

Profit per unit:

- Regular: \$5,000 per car  
- Premium: \$8,500 per car  

Maximize total daily profit:

\[
\max \; Z = 5000 x_1 + 8500 x_2
\]

---

## 3. Constraints

1. **Demand constraints (upper bounds):**

   - Regular demand:  
     \[
     x_1 \le 8
     \]
   - Premium demand:  
     \[
     x_2 \le 6
     \]

2. **Total production capacity:**

   At most 12 cars of either type per day:

   \[
   x_1 + x_2 \le 12
   \]

3. **Nonnegativity:**

   \[
   x_1 \ge 0, \quad x_2 \ge 0
   \]

This is a standard linear programming (LP) model.

---

## 4. Model summary

\[
\begin{align}
\text{Maximize} \quad & Z = 5000 x_1 + 8500 x_2 \\
\text{subject to} \quad
& x_1 \le 8 \\
& x_2 \le 6 \\
& x_1 + x_2 \le 12 \\
& x_1 \ge 0, \; x_2 \ge 0
\end{align}
\]

---

## 5. Solving analytically via corner points

Feasible corner points are from intersecting the constraints:

- \((0, 0)\)
- \((8, 0)\) from \(x_1 = 8, x_2 = 0\)
- \((0, 6)\) from \(x_2 = 6, x_1 = 0\)
- \((8, 4)\) from \(x_1 = 8, x_1 + x_2 = 12 \Rightarrow x_2 = 4\)
- \((6, 6)\) from \(x_2 = 6, x_1 + x_2 = 12 \Rightarrow x_1 = 6\)

Check profit at each:

- \(Z(0, 0) = 5000\cdot 0 + 8500\cdot 0 = 0\)
- \(Z(8, 0) = 5000\cdot 8 + 8500\cdot 0 = 40{,}000\)
- \(Z(0, 6) = 5000\cdot 0 + 8500\cdot 6 = 51{,}000\)
- \(Z(8, 4) = 5000\cdot 8 + 8500\cdot 4 = 40{,}000 + 34{,}000 = 74{,}000\)
- \(Z(6, 6) = 5000\cdot 6 + 8500\cdot 6 = 30{,}000 + 51{,}000 = 81{,}000\)

Maximum profit is \$81,000 at \((x_1, x_2) = (6, 6)\).

**Optimal production plan**:  
- Regular cars: \(x_1^* = 6\) per day  
- Premium cars: \(x_2^* = 6\) per day  
- Total profit: \$81,000 per day  

All constraints are satisfied:
- \(x_1 = 6 \le 8\)
- \(x_2 = 6 \le 6\)
- \(x_1 + x_2 = 12 \le 12\)

## 6. Suitable Python libraries and example formulations

Any LP solver can handle this small model efficiently. Some good choices:

- `PuLP` (pure Python interface, uses CBC or other solvers)
- `scipy.optimize` (e.g., `linprog`)
- `ortools.linear_solver` from Google OR-Tools

### Example with PuLP

```python
import pulp as pl

# Create LP problem (maximize profit)
prob = pl.LpProblem("Car_Production", pl.LpMaximize)

# Decision variables (continuous, non-negative)
x1 = pl.LpVariable("regular", lowBound=0)
x2 = pl.LpVariable("premium", lowBound=0)

# Objective
prob += 5000 * x1 + 8500 * x2

# Constraints
prob += x1 <= 8         # regular demand
prob += x2 <= 6         # premium demand
prob += x1 + x2 <= 12   # total capacity

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Optimal x1 (regular):", x1.value())
print("Optimal x2 (premium):", x2.value())
print("Max profit:", pl.value(prob.objective))
```

### Example with SciPy `linprog`

```python
from scipy.optimize import linprog

# Maximize 5000 x1 + 8500 x2 = minimize -5000 x1 - 8500 x2
c = [-5000, -8500]

# Inequality constraints A_ub x <= b_ub
A_ub = [
    [1, 0],   # x1 <= 8
    [0, 1],   # x2 <= 6
    [1, 1],   # x1 + x2 <= 12
]
b_ub = [8, 6, 12]

# Bounds for x1, x2
bounds = [(0, None), (0, None)]

res = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method="highs")

print("Optimal x1 (regular):", res.x[0])
print("Optimal x2 (premium):", res.x[1])
print("Max profit:", -res.fun)
```

### Example with OR-Tools

```python
from ortools.linear_solver import pywraplp

solver = pywraplp.Solver.CreateSolver('GLOP')  # LP solver

x1 = solver.NumVar(0, solver.infinity(), 'regular')
x2 = solver.NumVar(0, solver.infinity(), 'premium')

# Constraints
solver.Add(x1 <= 8)
solver.Add(x2 <= 6)
solver.Add(x1 + x2 <= 12)

# Objective
objective = solver.Objective()
objective.SetCoefficient(x1, 5000)
objective.SetCoefficient(x2, 8500)
objective.SetMaximization()

status = solver.Solve()

if status == pywraplp.Solver.OPTIMAL:
    print("Optimal x1 (regular):", x1.solution_value())
    print("Optimal x2 (premium):", x2.solution_value())
    print("Max profit:", objective.Value())
```

Any of these will reproduce the analytical solution: produce 6 regular and 6 premium cars per day.

import pulp as pl

def main():
    # Create LP problem (maximize profit)
    prob = pl.LpProblem("Car_Production", pl.LpMaximize)

    # Decision variables (continuous, non-negative)
    x1 = pl.LpVariable("regular", lowBound=0)
    x2 = pl.LpVariable("premium", lowBound=0)

    # Objective: Maximize 5000 * x1 + 8500 * x2
    prob += 5000 * x1 + 8500 * x2

    # Constraints
    prob += x1 <= 8         # regular demand
    prob += x2 <= 6         # premium demand
    prob += x1 + x2 <= 12   # total capacity

    # Solve
    prob.solve()

    # Extract objective value
    obj_value = pl.value(prob.objective)

    # Print results (optional)
    print("Status:", pl.LpStatus[prob.status])
    print("Optimal x1 (regular):", x1.value())
    print("Optimal x2 (premium):", x2.value())
    print("Max profit:", obj_value)

    # Final required output
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
