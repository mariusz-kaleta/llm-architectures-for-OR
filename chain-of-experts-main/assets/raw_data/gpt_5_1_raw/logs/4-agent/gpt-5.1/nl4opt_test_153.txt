## 1. Decision variables

Let

- \(x_1 \ge 0\): number of strawberry cookies produced per day  
- \(x_2 \ge 0\): number of sugar cookies produced per day  

These are continuous in the model; in reality they must be integers, but the LP will naturally give integer values here.

---

## 2. Objective function

Profit per cookie:

- Strawberry: \$5.50 per cookie  
- Sugar: \$12.00 per cookie  

Total daily profit:

\[
\text{Maximise} \quad Z = 5.5 x_1 + 12 x_2
\]

---

## 3. Constraints

### a) Demand constraints (upper bounds on sales)

Daily demand limits:

- At most 100 strawberry cookies
- At most 80 sugar cookies

\[
\begin{aligned}
x_1 &\le 100 \\
x_2 &\le 80
\end{aligned}
\]

### b) Production capacity constraint

Due to staffing, they can make at most 100 cookies total (of any mix):

\[
x_1 + x_2 \le 100
\]

### c) Non-negativity constraints

\[
x_1 \ge 0, \quad x_2 \ge 0
\]

---

## 4. Full linear program

\[
\begin{aligned}
\max_{x_1, x_2} \quad & 5.5 x_1 + 12 x_2 \\
\text{s.t.} \quad 
& x_1 \le 100 \\
& x_2 \le 80 \\
& x_1 + x_2 \le 100 \\
& x_1 \ge 0 \\
& x_2 \ge 0
\end{aligned}
\]

---

## 5. Suitable Python libraries

Any standard LP solver can handle this tiny model:

- `PuLP` (pure Python, easy to use)
- `ortools.linear_solver` (Google OR-Tools)
- `scipy.optimize.linprog` (SciPy)

Example using PuLP:

```python
import pulp as pl

# Problem
prob = pl.LpProblem("Zeta_Bakery", pl.LpMaximize)

# Decision variables
x1 = pl.LpVariable("strawberry", lowBound=0)
x2 = pl.LpVariable("sugar", lowBound=0)

# Objective
prob += 5.5 * x1 + 12 * x2

# Constraints
prob += x1 <= 100          # Strawberry demand
prob += x2 <= 80           # Sugar demand
prob += x1 + x2 <= 100     # Production capacity

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("x1 (strawberry) =", x1.value())
print("x2 (sugar) =", x2.value())
print("Max profit = $", pl.value(prob.objective))
```

You can plug this directly into Python to compute the optimal numbers of each cookie and the corresponding maximum profit.

[CODER]:
import pulp as pl

def main():
    # Define the problem
    prob = pl.LpProblem("Cookie_Production", pl.LpMaximize)

    # Decision variables
    x1 = pl.LpVariable("strawberry", lowBound=0)
    x2 = pl.LpVariable("sugar", lowBound=0)

    # Objective function: Maximize profit
    prob += 5.5 * x1 + 12 * x2

    # Constraints
    prob += x1 <= 100          # Strawberry demand
    prob += x2 <= 80           # Sugar demand
    prob += x1 + x2 <= 100     # Production capacity

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get the final objective value
    final_obj = pl.value(prob.objective)

    # Print the final objective value in required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
