[PLANNER]:
1. Key optimisation goal  
Goal: Minimise total cholesterol intake from burgers and pizza while still meeting minimum fat and calorie requirements and respecting the “pizza at least twice burgers” rule.

2. Suggested model type  
Model: Linear Programming (LP) with integer decision variables (an Integer Linear Program, ILP), since the number of burgers and slices must be whole numbers.

3. Main decision variables  
Define two primary decision variables:
- Number of burgers the man eats.
- Number of pizza slices the man eats.

These should be:
- Nonnegative.
- Integer-valued.

4. Key constraints to consider  

a) Fat requirement (minimum total fat)  
- Each burger contributes 10 units of fat.  
- Each pizza slice contributes 8 units of fat.  
- Total fat must be at least 130 units.

b) Calorie requirement (minimum total calories)  
- Each burger contributes 300 calories.  
- Each pizza slice contributes 250 calories.  
- Total calories must be at least 3000.

c) Cholesterol objective components  
- Each burger contributes 12 units of cholesterol.  
- Each pizza slice contributes 10 units of cholesterol.  
- The objective function is the sum of cholesterol from burgers and pizza; these coefficients go into the objective to be minimised.

d) Pizza-to-burger ratio constraint  
- The man decides to eat at least twice as many slices of pizza as burgers.  
- This gives a linear inequality linking the two decision variables (pizza ≥ 2 × burgers).

e) Nonnegativity and integrality  
- Both decision variables must be ≥ 0.  
- Both must be integers (no fractional burgers/slices).

5. Modelling strategy for the Modeller  

- Formulate the objective as a linear function of the number of burgers and slices using their cholesterol contributions.  
- Encode fat and calorie requirements as “≥” linear constraints.  
- Encode the pizza ≥ 2 × burgers condition as a linear inequality.  
- Enforce integrality on both variables.  
- Solve the ILP; given there are only two variables, you can:
  - Either use a standard ILP solver directly, or  
  - Solve the LP relaxation graphically/analytically, then search nearby integer points to find the integer optimum.

6. Additional checks  

- Verify that the optimal solution satisfies all constraints and that no other nearby integer solution gives a lower cholesterol total.  
- Optionally, report not only the optimal counts but also the resulting fat, calories, and cholesterol to confirm feasibility and optimality.

## 1. Decision variables

Let

- \(x\) = number of burgers eaten  
- \(y\) = number of pizza slices eaten  

Domain:

- \(x, y \in \mathbb{Z}_{\ge 0}\) (nonnegative integers)

---

## 2. Objective function (minimise cholesterol)

Each burger has 12 units of cholesterol.  
Each pizza slice has 10 units of cholesterol.

Total cholesterol:
\[
Z = 12x + 10y
\]

Objective:

\[
\min\; Z = 12x + 10y
\]

---

## 3. Constraints

### a) Fat requirement

Each burger has 10 units of fat.  
Each pizza slice has 8 units of fat.  
Total fat must be at least 130.

\[
10x + 8y \;\ge\; 130
\]

### b) Calorie requirement

Each burger has 300 calories.  
Each pizza slice has 250 calories.  
Total calories must be at least 3000.

\[
300x + 250y \;\ge\; 3000
\]

### c) Pizza-to-burger ratio

The man decides to eat at least twice as many slices of pizza as burgers:

\[
y \;\ge\; 2x
\]

### d) Nonnegativity and integrality

\[
x \;\ge\; 0,\quad y \;\ge\; 0,\quad x,y \in \mathbb{Z}
\]

---

## 4. Complete ILP model

\[
\begin{aligned}
\min\;\; & 12x + 10y \\
\text{s.t.}\;\; 
& 10x + 8y \;\ge\; 130 \quad\text{(fat)}\\
& 300x + 250y \;\ge\; 3000 \quad\text{(calories)}\\
& y - 2x \;\ge\; 0 \quad\text{(pizza at least twice burgers)}\\
& x, y \in \mathbb{Z}_{\ge 0}
\end{aligned}
\]

This is an Integer Linear Program (ILP) with 2 variables and 3 main constraints.

---

## 5. Suitable Python libraries

Any mixed-integer linear programming (MILP) solver can handle this:

- **PuLP**  
  - User-friendly, pure Python interface.  
  - Can use open-source solvers like CBC (default) or GLPK.
  - Good for teaching and small/medium problems.

- **OR-Tools (Google OR-Tools)**  
  - Powerful MILP solver (`ortools.linear_solver.pywraplp`) with CBC or SCIP.  
  - Very efficient and feature-rich.

- **Pyomo**  
  - General algebraic modeling language in Python.  
  - Interfaces to many solvers (CBC, GLPK, CPLEX, Gurobi, etc).

For this simple 2-variable problem, **PuLP** or **OR-Tools** is more than sufficient.

## 6. Example PuLP implementation sketch

```python
import pulp as pl

# Create problem
prob = pl.LpProblem("Cholesterol_Minimization", pl.LpMinimize)

# Decision variables
x = pl.LpVariable("burgers", lowBound=0, cat="Integer")
y = pl.LpVariable("pizza_slices", lowBound=0, cat="Integer")

# Objective function
prob += 12*x + 10*y, "Total_Cholesterol"

# Constraints
prob += 10*x + 8*y >= 130, "Fat_Constraint"
prob += 300*x + 250*y >= 3000, "Calorie_Constraint"
prob += y >= 2*x, "Pizza_Twice_Burgers"

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Status:", pl.LpStatus[prob.status])
print("Burgers:", x.value())
print("Pizza slices:", y.value())
print("Total cholesterol:", pl.value(12*x + 10*y))
print("Total fat:", pl.value(10*x + 8*y))
print("Total calories:", pl.value(300*x + 250*y))
```

This code directly implements the mathematical model above and returns the optimal number of burgers and pizza slices.

import pulp as pl

def main():
    # Create problem
    prob = pl.LpProblem("Cholesterol_Minimization", pl.LpMinimize)

    # Decision variables
    x = pl.LpVariable("burgers", lowBound=0, cat="Integer")
    y = pl.LpVariable("pizza_slices", lowBound=0, cat="Integer")

    # Objective function: Minimize cholesterol
    prob += 12 * x + 10 * y, "Total_Cholesterol"

    # Constraints
    prob += 10 * x + 8 * y >= 130, "Fat_Constraint"
    prob += 300 * x + 250 * y >= 3000, "Calorie_Constraint"
    prob += y >= 2 * x, "Pizza_Twice_Burgers"

    # Solve
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Compute final objective value
    final_obj = pl.value(12 * x + 10 * y)

    # Print detailed solution (optional; not restricted by the instructions)
    print(f"Status: {pl.LpStatus[prob.status]}")

    print(f"Total cholesterol: {final_obj}")
    print(f"Total fat: {pl.value(10 * x + 8 * y)}")
    print(f"Total calories: {pl.value(300 * x + 250 * y)}")

    # Required final line with objective value
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
