Goal: Minimise total pollution generated while transporting at least 80 cows, given a limit on the number of truck trips.

2. Suggested model type  
Model: Integer Linear Programming (ILP) or simple discrete optimisation (small enough for enumeration, but formulate as ILP).

3. Main decision variables  
Define integer decision variables representing the number of trips by each transport mode:
- x_H: number of helicopter trips (non‑negative integer).
- x_T: number of truck trips (non‑negative integer, with an upper bound).

4. Objective function (conceptual)  
Minimise total pollution:
- Each helicopter trip generates 5 units.
- Each truck trip generates 10 units.  
So objective is to minimise:  
Total pollution = 5 * x_H + 10 * x_T.

5. Key constraints to consider  

a) Demand (transport capacity) constraint  
- Each helicopter trip transports 3 cows.  
- Each truck trip transports 7 cows.  
- Need to transport at least 80 cows in total.  
Constraint structure:  
3 * x_H + 7 * x_T ≥ 80.

b) Truck trip budget constraint  
- At most 8 truck trips are affordable.  
Constraint structure:  
x_T ≤ 8.

c) Non-negativity and integrality  
- Both x_H and x_T must be non‑negative.  
- Both must be integers (number of trips cannot be fractional).  

6. Modelling strategy / notes for the Modeller  

- Build an ILP with two integer variables and the above linear objective and constraints.  
- Ensure the “at least” condition on cows is respected; the model can oversatisfy demand if that yields less pollution.  
- Because the problem is tiny, you can also double-check the optimal solution by enumerating integer values of x_T from 0 to 8 and computing the corresponding minimum feasible x_H for each, then comparing total pollution.  
- Confirm that the chosen solution satisfies:
  - All cows transported (no fewer than 80).
  - Truck trips do not exceed 8.
  - Variables are integers.
  - Achieves the minimal pollution value over all feasible combinations.

## 1. Decision variables

Let

- \(x_H \in \mathbb{Z}_{\ge 0}\): number of helicopter trips  
- \(x_T \in \mathbb{Z}_{\ge 0}\): number of truck trips  

Both must be integers and non‑negative.

---

## 2. Objective

Minimise the total pollution generated by all trips.

- Each helicopter trip: 5 units of pollution  
- Each truck trip: 10 units of pollution  

Objective function:
\[
\min\; Z = 5 x_H + 10 x_T
\]

---

## 3. Constraints

1. **Transport capacity (cows demand) constraint**  
   Each helicopter carries 3 cows per trip, each truck carries 7 cows per trip.  
   At least 80 cows must be transported:
   \[
   3 x_H + 7 x_T \;\ge\; 80
   \]

2. **Truck trip budget constraint**  
   At most 8 truck trips:
   \[
   x_T \;\le\; 8
   \]

3. **Non-negativity and integrality**
   \[
   x_H \ge 0,\quad x_T \ge 0,\quad x_H, x_T \in \mathbb{Z}
   \]

This is a small **Integer Linear Programming (ILP)** problem.

---

## 4. Suitable Python libraries

Any standard MILP or LP library can handle this easily. Good choices:

- **PuLP** (`pulp`): Very simple to model ILPs in pure Python; uses CBC solver by default.
- **OR-Tools** (`ortools.linear_solver`): Powerful and fast, backed by CBC or other solvers.
- **SciPy** (`scipy.optimize.linprog`): For continuous LP only; *not* directly integer, so less suitable unless you handle integrality manually for such a small case.

For this problem, **PuLP** or **OR-Tools** is ideal.

## 5. Example PuLP model (sketch)

```python
import pulp as pl

# Define problem
prob = pl.LpProblem("CowTransport", pl.LpMinimize)

# Decision variables (integer, >= 0)
x_H = pl.LpVariable("x_H", lowBound=0, cat="Integer")
x_T = pl.LpVariable("x_T", lowBound=0, upBound=8, cat="Integer")  # upBound enforces x_T ≤ 8

# Objective: minimize pollution
prob += 5 * x_H + 10 * x_T

# Constraints
prob += 3 * x_H + 7 * x_T >= 80  # cows demand

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Helicopter trips (x_H):", pl.value(x_H))
print("Truck trips (x_T):", pl.value(x_T))
print("Minimum pollution:", pl.value(prob.objective))
```

[CODER]:
import pulp as pl

def main():
    # Define the problem
    prob = pl.LpProblem("CowTransport", pl.LpMinimize)

    # Decision variables
    x_H = pl.LpVariable("x_H", lowBound=0, cat="Integer")
    x_T = pl.LpVariable("x_T", lowBound=0, upBound=8, cat="Integer")

    # Objective: minimize pollution
    prob += 5 * x_H + 10 * x_T

    # Constraints
    prob += 3 * x_H + 7 * x_T >= 80  # cow demand

    # Solve
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get objective value
    obj_value = pl.value(prob.objective)

    # Required final print
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
