## 1. Decision variables

Let

- \(x_1 \ge 0\): kilograms of **regular mix** to produce  
- \(x_2 \ge 0\): kilograms of **sour surprise mix** to produce  

Both are continuous variables (you can produce fractional kilograms).

---

## 2. Objective function

Profit per kilogram:

- Regular mix: \$3/kg  
- Sour surprise mix: \$5/kg  

Maximize total profit:

\[
\max\; Z = 3x_1 + 5x_2
\]

---

## 3. Constraints

### a) Regular candy availability

Usage per kg of mix:

- Regular mix uses 0.8 kg regular candy per kg  
- Sour surprise mix uses 0.1 kg regular candy per kg  

Total regular candy available: 80 kg.

\[
0.8x_1 + 0.1x_2 \le 80
\]

### b) Sour candy availability

Usage per kg of mix:

- Regular mix uses 0.2 kg sour candy per kg  
- Sour surprise mix uses 0.9 kg sour candy per kg  

Total sour candy available: 60 kg.

\[
0.2x_1 + 0.9x_2 \le 60
\]

### c) Non-negativity

\[
x_1 \ge 0,\quad x_2 \ge 0
\]

---

## 4. Complete LP model (compact form)

\[
\begin{aligned}
\max\quad & Z = 3x_1 + 5x_2 \\
\text{s.t.}\quad
& 0.8x_1 + 0.1x_2 \le 80 \quad &\text{(regular candy)} \\
& 0.2x_1 + 0.9x_2 \le 60 \quad &\text{(sour candy)} \\
& x_1 \ge 0, \; x_2 \ge 0 &\text{(non-negativity)}
\end{aligned}
\]

## 5. Suitable Python libraries

To solve this linear program in Python, any of the following is suitable:

- **PuLP** (very straightforward for small LPs, pure Python)
- **OR-Tools Linear Solver** (`ortools.linear_solver.pywraplp`) by Google  
- **SciPy** (`scipy.optimize.linprog`) for generic LP solving

For teaching / clarity, **PuLP** is an excellent choice. For larger or more complex models and integration with other combinatorial problems, **OR-Tools** is strong.

## 6. Example: PuLP implementation (sketch)

```python
import pulp as pl

# Create problem: maximize
prob = pl.LpProblem("Candy_Mix_Profit_Max", pl.LpMaximize)

# Decision variables
x1 = pl.LpVariable('regular_mix', lowBound=0)       # x1
x2 = pl.LpVariable('sour_surprise_mix', lowBound=0) # x2

# Objective
prob += 3 * x1 + 5 * x2

# Constraints
prob += 0.8 * x1 + 0.1 * x2 <= 80, "Regular_candy"
prob += 0.2 * x1 + 0.9 * x2 <= 60, "Sour_candy"

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Regular mix (kg):", x1.value())
print("Sour surprise mix (kg):", x2.value())
print("Max profit:", pl.value(prob.objective))
```

[CODER]:
import pulp as pl

def main():
    # Create the LP problem: maximize profit
    prob = pl.LpProblem("Candy_Mix_Profit_Max", pl.LpMaximize)

    # Decision variables
    x1 = pl.LpVariable('regular_mix', lowBound=0)       # x1: kg of regular mix
    x2 = pl.LpVariable('sour_surprise_mix', lowBound=0) # x2: kg of sour surprise mix

    # Objective function: Maximize 3*x1 + 5*x2
    prob += 3 * x1 + 5 * x2

    # Constraints
    # Regular candy: 0.8x1 + 0.1x2 <= 80
    prob += 0.8 * x1 + 0.1 * x2 <= 80, "Regular_candy"

    # Sour candy: 0.2x1 + 0.9x2 <= 60
    prob += 0.2 * x1 + 0.9 * x2 <= 60, "Sour_candy"

    # Solve the problem
    prob.solve()

    # Get the objective value
    final_obj = pl.value(prob.objective)

    # Print final objective in required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
