## Decision variables

Let

- \(x_1\) = number of servings of Rice per day  
- \(x_2\) = number of servings of Kebab per day  

Both are continuous and nonnegative:

\[
x_1 \ge 0,\quad x_2 \ge 0
\]

---

## Objective function (minimise daily cost)

Cost per serving: Rice = \$3, Kebab = \$2.  
Minimise total daily cost:

\[
\min\; Z = 3x_1 + 2x_2
\]

---

## Constraints

### 1. Calorie requirement

Each serving:  
- Rice: 300 calories  
- Kebab: 200 calories  

Total calories must be at least 2200:

\[
300x_1 + 200x_2 \;\ge\; 2200
\]

### 2. Protein requirement

Each serving:  
- Rice: 4.5 grams of protein  
- Kebab: 4 grams of protein  

Total protein must be at least 30 grams:

\[
4.5x_1 + 4x_2 \;\ge\; 30
\]

### 3. Non-negativity

\[
x_1 \ge 0,\quad x_2 \ge 0
\]

---

## Complete LP model

\[
\begin{aligned}
\min\quad & Z = 3x_1 + 2x_2 \\
\text{s.t.}\quad
& 300x_1 + 200x_2 \ge 2200 \quad &\text{(calorie requirement)} \\
& 4.5x_1 + 4x_2 \ge 30 \quad &\text{(protein requirement)} \\
& x_1 \ge 0,\; x_2 \ge 0 \quad &\text{(non-negativity)}
\end{aligned}
\]

[CODER]:
import pulp

def main():
    # Create the LP problem: minimization
    prob = pulp.LpProblem("Jordans_Diet_Problem", pulp.LpMinimize)

    # Decision variables: number of servings of Rice (x1) and Kebab (x2)
    x1 = pulp.LpVariable('x1', lowBound=0, cat='Continuous')  # Rice
    x2 = pulp.LpVariable('x2', lowBound=0, cat='Continuous')  # Kebab

    # Objective function: Minimize cost = 3*x1 + 2*x2
    prob += 3 * x1 + 2 * x2, "Total_Cost"

    # Constraints
    # Calorie requirement: 300x1 + 200x2 >= 2200
    prob += 300 * x1 + 200 * x2 >= 2200, "Calorie_Requirement"

    # Protein requirement: 4.5x1 + 4x2 >= 30
    prob += 4.5 * x1 + 4 * x2 >= 30, "Protein_Requirement"

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the objective value
    final_obj = pulp.value(prob.objective)

    # Print the final objective value in the required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
