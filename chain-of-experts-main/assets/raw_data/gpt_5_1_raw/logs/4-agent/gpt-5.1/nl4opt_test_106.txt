## 1. Decision variables

Let

- \(P \ge 0\): acres of potatoes to plant  
- \(C \ge 0\): acres of cucumbers to plant

---

## 2. Objective function

Profit per acre:
- Potatoes: \$500/acre  
- Cucumbers: \$650/acre  

Maximise total profit:
\[
\max \; Z = 500P + 650C
\]

---

## 3. Constraints

1. **Total land availability (50 acres):**
\[
P + C \le 50
\]

2. **Minimum contract requirements:**
   - Potatoes at least 12 acres:
     \[
     P \ge 12
     \]
   - Cucumbers at least 15 acres:
     \[
     C \ge 15
     \]

3. **Preference: more cucumbers than potatoes**

   The text is “prefers to grow more cucumbers than potatoes.” In LP we use non-strict inequalities, so we model:

\[
C \ge P
\]

4. **Resource limitation: cucumbers at most twice potatoes**
\[
C \le 2P
\]

5. **Nonnegativity** (already implied above, but explicitly):
\[
P \ge 0, \quad C \ge 0
\]

This defines a linear programming model with two continuous variables.

---

## 4. Solving the model analytically

Given cucumbers are more profitable per acre (650 > 500), Mark wants as many cucumber acres as possible within the constraints.

From the land limit \(P + C \le 50\), binding at optimum we expect:
\[
P + C = 50 \quad \Rightarrow \quad C = 50 - P
\]

Now apply the relational constraints to \(C = 50 - P\):

1. **More cucumbers than potatoes**:  
   \(C \ge P\)  
   \[
   50 - P \ge P \Rightarrow 50 \ge 2P \Rightarrow P \le 25
   \]

2. **Cucumbers at most twice potatoes**:  
   \(C \le 2P\)  
   \[
   50 - P \le 2P \Rightarrow 50 \le 3P \Rightarrow P \ge \frac{50}{3} \approx 16.67
   \]

3. **Minimums**:  
   \(P \ge 12\) (we already have stronger \(P \ge 16.67\))  
   \(C \ge 15\): with \(C = 50 - P\),  
   \[
   50 - P \ge 15 \Rightarrow P \le 35
   \]
   which is weaker than \(P \le 25\).

So the feasible range for \(P\) on the land boundary is:
\[
\frac{50}{3} \le P \le 25
\]

Profit as a function of \(P\) on that line:
\[
Z(P) = 500P + 650(50 - P)
      = 500P + 32500 - 650P
      = 32500 - 150P
\]

This is **decreasing** in \(P\) (coefficient \(-150\)), so we want the smallest feasible \(P\):

\[
P^* = \frac{50}{3}, \quad C^* = 50 - \frac{50}{3} = \frac{100}{3}
\]

Compute optimal profit:
\[
Z^* = 500\left(\frac{50}{3}\right) + 650\left(\frac{100}{3}\right)
    = \frac{500 \cdot 50}{3} + \frac{650 \cdot 100}{3}
    = \frac{25{,}000 + 65{,}000}{3}
    = \frac{90{,}000}{3}
    = \$30{,}000
\]

So the LP solution is:
- Potatoes: \(P^* = \frac{50}{3} \approx 16.67\) acres  
- Cucumbers: \(C^* = \frac{100}{3} \approx 33.33\) acres  
- Maximum profit: \$30,000

(If acres must be integer, this becomes an integer programming problem and the optimal discrete solution might differ slightly.)

## 5. Suitable Python libraries and sketch of code

Given this is a small linear program:

- **PuLP**: very easy for simple LPs.
- **OR-Tools** (`ortools.linear_solver.pywraplp`): also good and industrial-strength.
- **SciPy** (`scipy.optimize.linprog`): good for continuous LPs.

### Example with PuLP

```python
import pulp as pl

# Problem
prob = pl.LpProblem("Farm_Profit_Maximization", pl.LpMaximize)

# Decision variables (continuous, >= 0)
P = pl.LpVariable('P', lowBound=0)
C = pl.LpVariable('C', lowBound=0)

# Objective function
prob += 500 * P + 650 * C

# Constraints
prob += P + C <= 50             # Land
prob += P >= 12                 # Min potatoes
prob += C >= 15                 # Min cucumbers
prob += C >= P                  # More cucumbers than potatoes
prob += C <= 2 * P              # At most twice as many cucumbers

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Status:", pl.LpStatus[prob.status])
print("Potatoes (acres) =", P.value())
print("Cucumbers (acres) =", C.value())
print("Maximum profit = $", pl.value(prob.objective))
```

This will return approximately:
- Potatoes ≈ 16.6667  
- Cucumbers ≈ 33.3333  
- Maximum profit = 30000.

[CODER]:
import pulp as pl

def main():
    # Create the LP problem: maximize profit
    prob = pl.LpProblem("Farm_Profit_Maximization", pl.LpMaximize)

    # Decision variables (continuous, non-negative)
    P = pl.LpVariable('P', lowBound=0)  # acres of potatoes
    C = pl.LpVariable('C', lowBound=0)  # acres of cucumbers

    # Objective function: maximize 500P + 650C
    prob += 500 * P + 650 * C

    # Constraints
    prob += P + C <= 50        # Total land
    prob += P >= 12            # Minimum potatoes
    prob += C >= 15            # Minimum cucumbers
    prob += C >= P             # More cucumbers than potatoes
    prob += C <= 2 * P         # Cucumbers at most twice potatoes

    # Solve the LP
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Retrieve objective value
    final_obj = pl.value(prob.objective)

    # Required final output
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
