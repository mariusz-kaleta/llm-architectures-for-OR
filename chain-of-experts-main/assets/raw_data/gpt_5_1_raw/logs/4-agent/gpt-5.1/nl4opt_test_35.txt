[PLANNER]:
1. Key optimisation goal  
Goal: Minimise the total fat intake (in grams) from almonds and cashews over the week, while meeting weekly calorie and protein requirements and the preference on relative quantities.

2. Suggested model type  
Model: Linear Programming (LP) – continuous decision variables for the number of servings.

3. Main decision variables  
Define:
- Decision variable A: number of servings of almonds the woman eats in the week.  
- Decision variable C: number of servings of cashews the woman eats in the week.

Both A and C are nonnegative real numbers (assuming servings can be fractional; if only whole servings are allowed, then Integer LP could be used instead).

4. Objective function structure  
Minimise total fat:
- Each almond serving: 15 g fat  
- Each cashew serving: 12 g fat  

Total fat to minimise:  
- 15·A + 12·C

5. Key constraints to consider  

a) Calorie requirement (minimum weekly calories)  
- Almonds: 200 calories/serving  
- Cashews: 300 calories/serving  
- She needs at least 10,000 calories this week.

Constraint structure:  
- 200·A + 300·C ≥ 10,000

b) Protein requirement (minimum weekly protein)  
- Almonds: 20 g protein/serving  
- Cashews: 25 g protein/serving  
- She needs at least 800 g protein this week.

Constraint structure:  
- 20·A + 25·C ≥ 800

c) Preference ratio between almonds and cashews  
“She decides to eat at least twice as many servings of almonds as cashews.”

Constraint structure:  
- A ≥ 2·C

d) Nonnegativity  
- A ≥ 0  
- C ≥ 0

6. Notes for the modeller  
- Primary solution: Solve the LP with the above objective and constraints.  
- Check corner points defined by combinations of binding constraints:
  - Calorie & protein constraints active simultaneously;
  - Calorie & ratio constraints active;
  - Protein & ratio constraints active;
  - Plus intersections with axes as needed (A = 0 or C = 0) to identify the feasible region.  
- Verify the chosen solution satisfies all three constraints and yields the minimum fat value among candidate extreme points.  
- If integer servings are required, round or explicitly impose integer constraints and resolve as an Integer Linear Program, then compare nearby integer points if needed.

## 1. Decision variables

Let  
- \(A \ge 0\): number of servings of almonds eaten in the week

## 2. Objective function

Minimize total fat intake (grams) from almonds and cashews.

- Almonds: 15 g fat/serving  
- Cashews: 12 g fat/serving  

Objective:
\[
\min \; Z = 15A + 12C

## 3. Constraints

### a) Calorie requirement (at least 10,000 kcal per week)

- Almonds: 200 kcal/serving  
- Cashews: 300 kcal/serving  

\[
200A + 300C \;\ge\; 10{,}000

### b) Protein requirement (at least 800 g protein per week)

- Almonds: 20 g protein/serving  
- Cashews: 25 g protein/serving  

\[
20A + 25C \;\ge\; 800

### c) Preference ratio (at least twice as many almonds as cashews)

\[
A \;\ge\; 2C

### d) Nonnegativity

A \;\ge\; 0,\quad C \;\ge\; 0

## 4. Full LP model

\[
\begin{aligned}
\min_{A, C} \quad & 15A + 12C \\
\text{s.t.} \quad & 200A + 300C \ge 10{,}000, \\
& 20A + 25C \ge 800, \\
& A - 2C \ge 0, \\
& A \ge 0,\; C \ge 0.
\end{aligned}

If integer servings are required:

\[
A, C \in \mathbb{Z}_{\ge 0}.

## 5. Suitable Python libraries

Any standard LP/MIP solver library works:

For a straightforward LP with 2 variables and a few constraints, **PuLP** or **SciPy** are easiest.

## 6. Example PuLP implementation (continuous LP)

```python
import pulp as pl

# Create problem: minimize
prob = pl.LpProblem("Almonds_Cashews_Fat_Minimization", pl.LpMinimize)

# Decision variables
A = pl.LpVariable('A', lowBound=0)  # servings of almonds
C = pl.LpVariable('C', lowBound=0)  # servings of cashews

# Objective: Minimize total fat
prob += 15 * A + 12 * C, "Total_Fat"

# Constraints
prob += 200 * A + 300 * C >= 10000, "Calorie_Requirement"
prob += 20 * A + 25 * C >= 800, "Protein_Requirement"
prob += A - 2 * C >= 0, "Preference_Ratio"

# Solve
prob.solve()

print("Status:", pl.LpStatus[prob.status])
print("Almond servings A =", pl.value(A))
print("Cashew servings C =", pl.value(C))
print("Minimum total fat (g) =", pl.value(prob.objective))
```

To force integer servings, define the variables as:

```python
A = pl.LpVariable('A', lowBound=0, cat='Integer')
C = pl.LpVariable('C', lowBound=0, cat='Integer')
```

## 7. Example SciPy implementation (continuous LP)

SciPy’s `linprog` solves:

- Minimize: \(c^T x\)  
- Subject to: \(A_{ub} x \le b_{ub}\), \(A_{eq} x = b_{eq}\), bounds

We convert “≥” to “≤” by multiplying by −1.

```python
import numpy as np
from scipy.optimize import linprog

# Variables order: x = [A, C]
c = np.array([15, 12])  # objective coefficients

# Constraints:
# 200A + 300C >= 10000  -> -200A - 300C <= -10000
# 20A + 25C   >= 800    -> -20A  - 25C <= -800
# A - 2C      >= 0      -> -A + 2C <= 0

A_ub = np.array([
    [-200, -300],
    [-20,  -25],
    [-1,    2]
])
b_ub = np.array([-10000, -800, 0])

bounds = [(0, None), (0, None)]  # A >= 0, C >= 0

res = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')

print("Success:", res.success)
print("Almond servings A =", res.x[0])
print("Cashew servings C =", res.x[1])
print("Minimum total fat (g) =", res.fun)
```
