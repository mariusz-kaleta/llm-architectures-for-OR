     - Variable 1: Number of regular glass panes produced per day.  
     - Variable 2: Number of tempered glass panes produced per day.  

4. Objective function structure  
   - Total profit = (profit per regular pane × number of regular panes) + (profit per tempered pane × number of tempered panes).

   - Each tempered pane uses 8 minutes on the cooling machine.

   - Structure: (5 × regular panes) + (8 × tempered panes) ≤ 300.

   c) Nonnegativity / feasibility

---

## 1. Decision variables

Let  
- \(x \ge 0\): number of regular glass panes produced per day.  
- \(y \ge 0\): number of tempered glass panes produced per day.

(These are naturally integer, but the model is linear and can be solved as a continuous LP; in this setting, integer solutions will appear automatically at the optimum because of the structure.)

---

## 2. Objective function

Profit per pane:  
- Regular pane: \$8  
- Tempered pane: \$10  

Total daily profit:
\[
\text{Maximize } Z = 8x + 10y
\]

---

## 3. Constraints

### a) Heating machine time

- Each regular pane uses 3 minutes of heating.  
- Each tempered pane uses 5 minutes of heating.  
- Heating machine available: 300 minutes per day.

\[
3x + 5y \le 300
\]

### b) Cooling machine time

- Each regular pane uses 5 minutes of cooling.  
- Each tempered pane uses 8 minutes of cooling.  
- Cooling machine available: 300 minutes per day.

\[
5x + 8y \le 300
\]

### c) Nonnegativity

\[
x \ge 0,\quad y \ge 0
\]

---

## 4. Complete LP model

\[
\begin{align}
\text{maximize} \quad & 8x + 10y \\
\text{subject to} \quad
& 3x + 5y \le 300 \quad \text{(heating time)}\\
& 5x + 8y \le 300 \quad \text{(cooling time)}\\
& x \ge 0,\ y \ge 0
\end{align}
\]

---

## 5. Solving by hand (outline)

1. Find intersection of constraints:

   \[
   \begin{cases}
   3x + 5y = 300 \\
   5x + 8y = 300
   \end{cases}
   \Rightarrow x=200,\ y=-60
   \]

   This intersection is infeasible (negative \(y\)), so both machine constraints cannot be tight simultaneously in the feasible region.

2. Candidate corner points (using constraints and axes):

   - Heating vs \(x\)-axis: \(3x \le 300 \Rightarrow x \le 100,\ y=0\): point \((100,0)\).  
   - Cooling vs \(x\)-axis: \(5x \le 300 \Rightarrow x \le 60,\ y=0\): point \((60,0)\).  
     The effective bound on \(x\) is 60 (cooling is tighter).
   - Heating vs \(y\)-axis: \(5y \le 300 \Rightarrow y \le 60,\ x=0\): point \((0,60)\).  
   - Cooling vs \(y\)-axis: \(8y \le 300 \Rightarrow y \le 37.5,\ x=0\): point \((0,37.5)\).  
     The effective bound on \(y\) is 37.5 (cooling is tighter).

   Feasible corners (including origin):

   - \((0,0)\)
   - \((60,0)\)  (cooling binding, heating slack)
   - \((0,37.5)\) (cooling binding, heating slack)

3. Evaluate profit at these points:

   - \((0,0)\): \(Z = 8(0) + 10(0) = 0\).
   - \((60,0)\): \(Z = 8(60) + 10(0) = 480\).
   - \((0,37.5)\): \(Z = 8(0) + 10(37.5) = 375\).

   Best continuous solution: \((x,y) = (60,0)\) with maximum profit \(Z^* = \$480\).

4. If integer-only solutions are required, \((60,0)\) is already integral and optimal.

**Answer:**  
- Produce 60 regular panes and 0 tempered panes per day.  
- Maximum profit: \$480 per day.

## 6. Suitable Python tools to implement and solve this LP

You can model and solve this with several Python libraries:

- **PuLP**: Simple LP/MIP modeling library, works with CBC by default.
- **OR-Tools (linear_solver module)**: Google’s optimization suite; very good for LP/MIP.
- **SciPy (`scipy.optimize.linprog`)**: Basic LP solver, good for continuous LP (no integrality).

### Example with PuLP

```python
import pulp as pl

# 1. Define the problem
prob = pl.LpProblem("Glass_Production", pl.LpMaximize)

# 2. Decision variables
x = pl.LpVariable('regular_panes', lowBound=0)
y = pl.LpVariable('tempered_panes', lowBound=0)

# 3. Objective function
prob += 8 * x + 10 * y, "Total_Profit"

# 4. Constraints
prob += 3 * x + 5 * y <= 300, "Heating_Time"
prob += 5 * x + 8 * y <= 300, "Cooling_Time"

# 5. Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Status:", pl.LpStatus[prob.status])
print("Regular panes (x):", x.varValue)
print("Tempered panes (y):", y.varValue)
print("Maximum profit:", pl.value(prob.objective))
```

This will return \(x = 60\), \(y = 0\), and maximum profit = 480.

[CODER]:
import pulp as pl

def main():
    # Define the problem
    prob = pl.LpProblem("Glass_Production", pl.LpMaximize)

    # Decision variables
    x = pl.LpVariable('regular_panes', lowBound=0)
    y = pl.LpVariable('tempered_panes', lowBound=0)

    # Objective function: Maximize 8x + 10y
    prob += 8 * x + 10 * y, "Total_Profit"

    # Constraints
    prob += 3 * x + 5 * y <= 300, "Heating_Time"
    prob += 5 * x + 8 * y <= 300, "Cooling_Time"

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get the final objective value
    final_obj = pl.value(prob.objective)

    # Print strictly in required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
