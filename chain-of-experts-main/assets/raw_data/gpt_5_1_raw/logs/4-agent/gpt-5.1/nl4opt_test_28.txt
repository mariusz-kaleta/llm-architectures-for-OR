## 1. Decision variables

Let

- \(x_d \ge 0\): number of sled dog trips  
- \(x_t \ge 0\): number of truck trips

## 2. Objective function

Maximize the total amount of fish transported.

- Each sled dog trip carries 100 fish.
- Each truck trip carries 300 fish.

Objective:

\[
\max \; Z = 100 x_d + 300 x_t
\]

## 3. Constraints

### a) Budget constraint

Total cost of all trips cannot exceed the budget \(1000\).

- Sled dog trip cost: 50  
- Truck trip cost: 100  

\[
50 x_d + 100 x_t \le 1000
\]

### b) Relationship between sled dog and truck trips

The number of sled dog trips must be *less than* the number of truck trips.

- In a pure LP with continuous variables, a strict inequality is usually approximated by:

\[
x_d \le x_t
\]

- If you use an ILP with integer variables and want strictness exactly:

\[
x_d + 1 \le x_t
\]

Choose one form depending on your modelling choice.

### c) Non-negativity (and integrality if required)

\[
x_d \ge 0, \quad x_t \ge 0
\]

If modelling trips as integers:

\[
x_d, x_t \in \mathbb{Z}_{\ge 0}
\]

## 6. Suggested Python libraries

For this small LP/ILP, several libraries work well:

- `PuLP` – very convenient for teaching and small/medium models.
- `ortools.linear_solver` (from Google OR-Tools) – robust, supports LP and MIP.
- `Pyomo` – good if you want a more general algebraic modelling language.
- `scipy.optimize.linprog` – suitable for continuous LP only (not integers, not strict inequality).

A typical choice for this task: `PuLP` or OR-Tools.

Example (sketch) using PuLP for the continuous LP:

```python
import pulp as pl

# Problem
prob = pl.LpProblem("Fish_Transport", pl.LpMaximize)

# Variables
x_d = pl.LpVariable('sled_dog_trips', lowBound=0)
x_t = pl.LpVariable('truck_trips', lowBound=0)

# Objective
prob += 100 * x_d + 300 * x_t

# Constraints
prob += 50 * x_d + 100 * x_t <= 1000       # Budget
prob += x_d <= x_t                         # Sled trips ≤ truck trips

# Solve
prob.solve()
print("Status:", pl.LpStatus[prob.status])
print("sled_dog_trips =", x_d.varValue)
print("truck_trips =", x_t.varValue)
print("Max fish =", pl.value(prob.objective))
```

For integer trips, set `cat="Integer"` when creating `LpVariable` and replace `x_d <= x_t` with `x_d + 1 <= x_t`.

import pulp as pl

def main():
    # Create the ILP problem (integer trips, strict inequality via x_d + 1 <= x_t)
    prob = pl.LpProblem("Fish_Transport_ILP", pl.LpMaximize)

    # Decision variables: integer, non-negative
    x_d = pl.LpVariable('sled_dog_trips', lowBound=0, cat="Integer")
    x_t = pl.LpVariable('truck_trips', lowBound=0, cat="Integer")

    # Objective: maximize transported fish
    prob += 100 * x_d + 300 * x_t

    # Constraints
    prob += 50 * x_d + 100 * x_t <= 1000    # Budget constraint
    prob += x_d + 1 <= x_t                  # Strict inequality x_d < x_t via integrality

    # Solve
    prob.solve()

    # Get objective value
    final_obj = pl.value(prob.objective)

    # Required final print
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
