## 1. Decision variables

Let  
- \(x_J \ge 0\): number of times process J is performed  
- \(x_P \ge 0\): number of times process P is performed  

If only whole runs are allowed, add:
- \(x_J, x_P \in \mathbb{Z}\) (integer programming)
Otherwise, treat them as continuous (standard LP).

## 2. Objective function

Each run of:
- J yields 5 units of metal  
- P yields 9 units of metal  

Maximise total metal extracted:
\[
\max Z = 5x_J + 9x_P
\]

## 3. Constraints

### a) Water constraint

Each run of:
- J uses 8 units of water  
- P uses 6 units of water  

Total available water ≤ 1500 units:
\[
8x_J + 6x_P \le 1500
\]

### b) Pollution constraint

Each run of:
- J produces 3 units of pollution  
- P produces 5 units of pollution  

Total allowable pollution ≤ 1350 units:
\[
3x_J + 5x_P \le 1350
\]

### c) Nonnegativity (and integrality if needed)

\[
x_J \ge 0,\quad x_P \ge 0
\]
and optionally
\[
x_J, x_P \in \mathbb{Z}
\]

## 4. Complete LP/IP model

Linear Program (continuous):
\[
\begin{aligned}
\max\quad & 5x_J + 9x_P \\
\text{s.t.}\quad 
& 8x_J + 6x_P \le 1500 \quad \text{(water)} \\
& 3x_J + 5x_P \le 1350 \quad \text{(pollution)} \\
& x_J \ge 0,\; x_P \ge 0
\end{aligned}
\]

Integer Program (if only whole processes are allowed):

\begin{aligned}

import pulp

def main():
    # Create the LP maximization problem
    prob = pulp.LpProblem("Metal_Extraction", pulp.LpMaximize)

    # Decision variables (continuous, non-negative)
    x_J = pulp.LpVariable('x_J', lowBound=0, cat='Continuous')
    x_P = pulp.LpVariable('x_P', lowBound=0, cat='Continuous')

    # Objective function: Maximize 5*x_J + 9*x_P
    prob += 5 * x_J + 9 * x_P

    # Constraints
    prob += 8 * x_J + 6 * x_P <= 1500  # Water constraint
    prob += 3 * x_J + 5 * x_P <= 1350  # Pollution constraint

    # Solve the problem
    prob.solve(pulp.PULP_CBC_CMD(msg=False))

    # Get the optimal objective value
    final_obj = pulp.value(prob.objective)

    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
