Goal: Minimise the total amount of waste produced by the two machines while satisfying heart and brain dosage requirements.

2. Suggested model type  
Model: Linear Programming (LP) with two decision variables.

3. Main decision variables  
Define non‑negative continuous decision variables such as:
- x₁: Number of minutes machine 1 is used.  
- x₂: Number of minutes machine 2 is used.

4. Objective (to guide the Modeller)  
- Total waste per minute from:
  - Machine 1: 0.3 units waste/min.
  - Machine 2: 0.5 units waste/min.
- Objective: Minimise total waste generated by both machines over their operating times: a linear function in x₁ and x₂.

Dosage delivery constraints:
- Heart dosage upper bound:
  - Machine 1 delivers 0.5 units/min to the heart.
  - Machine 2 delivers 0.3 units/min to the heart.

- Brain dosage lower bound:
  - Machine 1 delivers 0.8 units/min to the brain.
  - Machine 2 delivers 1.0 unit/min to the brain.
  - Total brain medicine received from both machines must be **at least** 4 units.

Non-negativity / feasibility:
- Time on each machine cannot be negative:
  - x₁ ≥ 0
  - x₂ ≥ 0

## 1. Decision variables

Let  
- \(x_1 \ge 0\): number of minutes machine 1 is used  
- \(x_2 \ge 0\): number of minutes machine 2 is used  

(Assume fractional minutes are allowed; if not, these would be integer variables.)

---

## 2. Objective function

Waste generated per minute:
- Machine 1: 0.3 units/min  
- Machine 2: 0.5 units/min  

Total waste:
\[
\text{Waste} = 0.3x_1 + 0.5x_2
\]

**Objective:**  
\[
\min\ Z = 0.3x_1 + 0.5x_2
\]

---

## 3. Constraints

### (a) Heart dosage (at most 8 units)

Heart medicine per minute:
- Machine 1: 0.5 units/min  
- Machine 2: 0.3 units/min  

Total heart medicine constraint:
\[
0.5x_1 + 0.3x_2 \le 8
\]

### (b) Brain dosage (at least 4 units)

Brain medicine per minute:
- Machine 1: 0.8 units/min  
- Machine 2: 1.0 units/min  

Total brain medicine constraint:
\[
0.8x_1 + 1.0x_2 \ge 4
\]

### (c) Non‑negativity

\[
x_1 \ge 0, \quad x_2 \ge 0
\]

---

## 4. Full Linear Programming model

\[
\begin{aligned}
\min_{x_1,x_2}\quad & 0.3x_1 + 0.5x_2 \\
\text{s.t.}\quad
& 0.5x_1 + 0.3x_2 \le 8 \quad &\text{(heart)}\\
& 0.8x_1 + 1.0x_2 \ge 4 \quad &\text{(brain)}\\
& x_1 \ge 0,\ x_2 \ge 0 \quad &\text{(non-negativity)}
\end{aligned}
\]

This is a linear program with two continuous variables.

## 5. Suitable Python libraries

Any standard LP solver or interface will handle this easily:

- **PuLP** (`pulp`) – simple, pure Python LP modeling.
- **SciPy** (`scipy.optimize.linprog`) – good for small LPs; part of SciPy.
- **OR-Tools** (`ortools.linear_solver`) – industrial-strength Google library.

For this 2-variable LP, **PuLP** or **SciPy** is more than sufficient.

## 6. Example in PuLP

```python
import pulp as pl

# Problem
prob = pl.LpProblem("Medicine_Machines", pl.LpMinimize)

# Decision variables
x1 = pl.LpVariable('x1', lowBound=0)
x2 = pl.LpVariable('x2', lowBound=0)

# Objective: minimize waste
prob += 0.3 * x1 + 0.5 * x2

# Constraints
prob += 0.5 * x1 + 0.3 * x2 <= 8    # heart
prob += 0.8 * x1 + 1.0 * x2 >= 4    # brain

# Solve
prob.solve(pl.PULP_CBC_CMD(msg=False))

print("Status:", pl.LpStatus[prob.status])
print("x1 (minutes on machine 1):", x1.value())
print("x2 (minutes on machine 2):", x2.value())
print("Minimum total waste:", pl.value(prob.objective))
```

You can swap PuLP for SciPy or OR-Tools with similar constraints and objective definitions.

import pulp as pl

def main():
    # Create the LP problem: minimize waste
    prob = pl.LpProblem("Medicine_Machines", pl.LpMinimize)

    # Decision variables (non-negative)
    x1 = pl.LpVariable('x1', lowBound=0)  # minutes on machine 1
    x2 = pl.LpVariable('x2', lowBound=0)  # minutes on machine 2

    # Objective function: minimize total waste
    prob += 0.3 * x1 + 0.5 * x2

    # Constraints
    prob += 0.5 * x1 + 0.3 * x2 <= 8      # Heart dosage constraint
    prob += 0.8 * x1 + 1.0 * x2 >= 4      # Brain dosage constraint

    # Solve the problem
    prob.solve(pl.PULP_CBC_CMD(msg=False))

    # Get objective value
    obj_value = pl.value(prob.objective)

    # Print final objective in required format
    print(f"FINAL_OBJ: {obj_value}")

if __name__ == "__main__":
    main()
