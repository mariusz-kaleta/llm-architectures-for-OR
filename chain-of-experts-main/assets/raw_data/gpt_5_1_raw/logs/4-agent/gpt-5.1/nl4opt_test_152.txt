## 1. Decision variables

Let  
- \(x \ge 0\): number of cups of drink A in the mix  
- \(y \ge 0\): number of cups of drink B in the mix  

Variables are continuous (fractions of a cup allowed) unless explicitly required to be integers.

---

## 2. Objective: minimise total Vitamin K

- Drink A: 4 units of Vitamin K per cup  
- Drink B: 12 units of Vitamin K per cup  

Total Vitamin K in the mix:
\[
K_{\text{total}} = 4x + 12y
\]

**Objective function**:
\[
\min\; 4x + 12y
\]

## 3. Constraints

### a) Vitamin A requirement (minimum)

- Drink A: 8 units Vitamin A per cup  
- Drink B: 15 units Vitamin A per cup  
- Need at least 150 units:

\]

---

### b) Vitamin D requirement (minimum)

- Drink A: 6 units Vitamin D per cup  
- Drink B: 2 units Vitamin D per cup  
- Need at least 300 units:

\[
6x + 2y \;\ge\; 300
\]

---

### c) Vitamin E requirement (maximum)

- Drink A: 10 units Vitamin E per cup  
- Drink B: 20 units Vitamin E per cup

### d) Non-negativity

## 4. Complete LP formulation

\[
\begin{aligned}
\min\;& 4x + 12y \\
\text{s.t. } 
& 8x + 15y \;\ge\; 150 \quad &(\text{Vitamin A})\\
& 6x + 2y \;\ge\; 300 \quad &(\text{Vitamin D})\\
& 10x + 20y \;\le\; 400 \quad &(\text{Vitamin E})\\
& x \;\ge\; 0,\; y \;\ge\; 0
\end{aligned}
\]

This is a standard linear program.

## 5. Suitable Python libraries

Any LP-capable solver will work. Common choices:

- **PuLP**  
  - Pure Python, easy syntax, interfaces to CBC, GLPK, etc.  
  - Good for teaching and small/medium problems.

- **SciPy (`scipy.optimize.linprog`)**  
  - Good if you’re already using SciPy stack.  
  - Handles linear constraints and objectives directly.

- **OR-Tools (Google OR-Tools Linear Solver)**  
  - Industrial-strength, supports multiple back-end solvers (CBC, GLOP, CLP, etc.).  
  - Good for larger or more complex models.

Example outline with PuLP (just to show mapping, not to solve here):

```python
import pulp as pl

# Problem
prob = pl.LpProblem("VitaminMix", pl.LpMinimize)

# Variables
x = pl.LpVariable('x', lowBound=0)
y = pl.LpVariable('y', lowBound=0)

# Objective
prob += 4*x + 12*y

# Constraints
prob += 8*x + 15*y >= 150   # Vitamin A
prob += 6*x + 2*y >= 300    # Vitamin D
prob += 10*x + 20*y <= 400  # Vitamin E

# Solve
prob.solve()
print(x.value(), y.value())
```

[CODER]:
import pulp as pl

def main():
    # Define the problem
    prob = pl.LpProblem("VitaminMix", pl.LpMinimize)

    # Decision variables (continuous, non-negative)
    x = pl.LpVariable('x', lowBound=0)
    y = pl.LpVariable('y', lowBound=0)

    # Objective function: Minimize total Vitamin K
    prob += 4 * x + 12 * y

    # Constraints
    # Vitamin A: 8x + 15y >= 150
    prob += 8 * x + 15 * y >= 150

    # Vitamin D: 6x + 2y >= 300
    prob += 6 * x + 2 * y >= 300

    # Vitamin E: 10x + 20y <= 400
    prob += 10 * x + 20 * y <= 400

    # Solve the problem
    prob.solve()

    # Get the objective value
    final_obj = pl.value(prob.objective)

    # Print the final objective value in required format
    print(f"FINAL_OBJ: {final_obj}")

if __name__ == "__main__":
    main()
